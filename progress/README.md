# Progress 폴더

이 폴더에 각자의 미니 프로젝트 진행 기록을 올립니다.

## 폴더 구조

```
progress/
  민호/
    ai-news-hub.md
  홍길동/
    이메일-자동화.md
    보고서-템플릿.md
```

## 규칙

1. **자기 이름으로 폴더**를 만드세요
2. 폴더 안에 **프로젝트별 파일**을 만드세요 (파일 이름 = 프로젝트 이름)
3. `docs/templates/progress-template.md`를 복사해서 시작하세요
4. 미니 프로젝트가 끝나면 `docs/case-studies/` 폴더에 케이스 스터디도 올려주세요
