# Progress 폴더

이 폴더에 각자의 미니 프로젝트 진행 기록을 올립니다.

## 폴더 구조

```
progress/
  김민호/
    01-2026-02-23-팀-협업-시스템.md
    02-2026-02-24-회의-비서.md
  박승현/
    01-2026-02-23-GIRIN-마스터-md-만들기.md
```

## 파일명 규칙

`[번호]-[시작일]-[프로젝트이름].md`

- **번호**: 몇 번째 미니 프로젝트인지 (01, 02, 03...)
- **시작일**: 프로젝트를 시작한 날짜
- **프로젝트이름**: 한글 OK

## 규칙

1. **자기 이름으로 폴더**를 만드세요
2. 위 파일명 규칙에 맞춰 파일을 만드세요
3. `docs/templates/progress-template.md`를 참고하세요
4. 미니 프로젝트가 끝나면 `docs/case-studies/` 폴더에 케이스 스터디도 올려주세요
