### 배운 개발 지식

- **git pull** — GitHub에 새로 올라온 것만 내 컴퓨터로 가져오기. 공유 폴더에서 새 파일만 받아오는 것과 같음.
- **git commit vs git push** — commit은 내 컴퓨터에만 저장, push는 GitHub에 업로드. 편지를 쓰는 것(commit)과 우체통에 넣는 것(push)의 차이.
- **Collaborator 권한** — GitHub 저장소에 다른 사람이 파일을 올릴 수 있게 초대하는 것. 공유 폴더에 접근 권한을 주는 것과 같음.
- **gh auth login** — GitHub CLI로 인증하기. 비밀번호 대신 브라우저 로그인으로 안전하게 연결.
