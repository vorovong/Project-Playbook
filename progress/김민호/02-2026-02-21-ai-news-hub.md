### 결과

- **프로젝트 종료 처리**: daily-update.yml의 cron schedule 비활성화 → Gemini API 비용 차단
- 수동 실행(workflow_dispatch)은 유지하여 필요시 1회성 실행 가능

### 배운 개발 지식

- **GitHub Actions schedule 비활성화** — 워크플로우 파일에서 `schedule:` 부분을 주석 처리(`#`)하면 자동 실행이 중단됨. 알람 앱에서 반복 알람을 끄는 것과 같음. 파일 자체를 삭제하지 않아도 되고, 주석만 풀면 언제든 다시 켤 수 있음.
