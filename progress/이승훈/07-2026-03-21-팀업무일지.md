| 내가 한 말 | 결과 |
|---|---|

### 결과

**엑셀 버전 (generate-team-worklog.js):**
- ExcelJS로 팀 업무일지 엑셀 파일 자동 생성
- 스케줄 보드 + 팀원별 업무일지 시트
- 주간 일정 관리, 일별 진행 기록, 팀원별 카테고리
- 색상/서식 자동화, 병합셀, 높이 조정

**구글 시트 버전 (팀업무일지-Apps-Script.js):**
- 구글 시트에 붙여넣기하는 Apps Script 코드
- 4개 탭 자동 생성: 스케줄 보드 + 팀원별 업무일지 + 대표 대시보드 + 사용법
- 팀원(수빈, 승훈, 팀원3)별 카테고리 설정
- 드롭다운 메뉴 자동화
- 주간 업무 계획 + 일간 업무 기록 + 주간 회고 + 팀원 상태 추적

### 배운 개발 지식

- **엑셀 vs 구글 시트 자동화** — 엑셀은 ExcelJS로 파일을 생성하고, 구글 시트는 Apps Script로 시트 안에서 자동화. 둘 다 장단점이 있어서 두 버전을 함께 만듦.
- **Apps Script 탭 구조** — 하나의 구글 시트 안에 여러 탭(시트)을 코드로 자동 생성. 각 탭마다 역할이 다름 (스케줄/업무일지/대시보드).
