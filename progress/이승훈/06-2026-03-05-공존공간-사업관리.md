| 내가 한 말 | 결과 |

### 결과

- 프로젝트 폴더 생성 (`mini-projects/이승훈/공존공간-사업관리/`)
- 기본 구조 설계:
  - Node.js + Express 서버 (포트 3001)
  - SQLite 데이터베이스 (7개 테이블: 사업, PDF, 임차인, 정산, 월세, 통계)
  - 관리자 웹 화면 (index.html, rental.html)
- 핵심 기능 설계: 사업 CRUD, PDF 문서 관리, 임차인 관리, 주간 정산, 월세 수금, 대시보드

### 배운 개발 지식

- **SQLite** — 파일 하나로 동작하는 가벼운 데이터베이스. 별도 서버 설치 없이 바로 사용 가능. 엑셀 파일처럼 하나의 파일에 데이터를 저장하지만, 검색/정렬이 훨씬 빠름.
- **WAL 모드 (Write-Ahead Logging)** — 데이터베이스가 읽기와 쓰기를 동시에 할 수 있게 해주는 설정. 여러 명이 동시에 쓸 때 속도가 빨라짐.
- **Foreign Key** — 테이블 간의 연결고리. 임차인 테이블과 정산 테이블을 연결해서 "이 정산은 어느 임차인 것인지" 자동으로 추적.
