### 배운 개발 지식

- **HWPX 자동 생성** — 한글 파일의 XML 구조를 직접 조작하여 문서를 만드는 방식. 레퍼런스 문서에서 XML 뼈대를 추출하고, 데이터만 교체하면 서식이 그대로 유지됨.
- **lxml 라이브러리** — Python에서 XML을 읽고 수정하는 도구. 한글 파일 내부의 XML을 파싱하여 텍스트를 교체할 때 사용.

- 📐 문서 자동화 전략 (Document Automation) ⚖️
  보고서 생성 방식 선택.
  선택지: ①API로 처음부터 생성(유연하지만 서식 깨짐) ②템플릿 복사+텍스트 교체(서식 보존, 구조 제한)
  판단: 신보재단 양식이 정해져 있어서 서식 보존이 핵심. 템플릿 방식이 적합.
  = 빈칸 채우기 시험처럼, 양식은 그대로 두고 답만 쓰는 게 빠르고 정확함.
