- **Data Entry Form 패턴** — 입력 폼 하나 + 전송 버튼으로 데이터를 쌓고, 요약 시트가 자동 계산하는 구조. 식당 주문서처럼 한 장 쓰고 넘기면 주방에서 알아서 정리하는 것.
- **SUMPRODUCT** — 여러 조건을 동시에 만족하는 셀만 골라 합산하는 수식. "3월이면서 1주차인 행의 카드매출만 합쳐줘" 같은 요청을 처리.
- **Apps Script** — 구글시트에 붙이는 자동화 코드. 전송 버튼 클릭 → 검증 → 저장 → 초기화를 한 번에 처리.
