### 배운 개발 지식

- **프론트매터(Frontmatter)** — 파일 맨 위에 `---`로 감싼 메타데이터 영역. 도서관 책의 분류 카드처럼, 파일의 종류·소유자·상태를 한눈에 파악할 수 있게 해줌.
- **HTML + CSS + JavaScript** — 웹 앱을 만드는 세 가지 기본 재료. HTML은 뼈대(구조), CSS는 옷(디자인), JavaScript는 근육(동작). 이 세 가지로 브라우저에서 돌아가는 앱을 만들 수 있음.
- **Firebase** — 구글이 만든 클라우드 저장소. 서버 없이도 데이터를 저장하고 여러 사람이 함께 쓸 수 있게 해주는 서비스. 창고를 직접 짓지 않고 빌려 쓰는 것과 같음.
