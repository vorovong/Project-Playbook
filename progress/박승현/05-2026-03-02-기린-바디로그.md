### 배운 개발 지식

- **PWA (Progressive Web App)** — 웹사이트를 앱처럼 홈 화면에 설치할 수 있게 만드는 기술. 카페 단골이 즐겨찾기 대신 바로가기 아이콘을 홈에 놓는 것과 같음
- **Service Worker** — 앱과 서버 사이에서 대리인 역할을 하는 프로그램. 오프라인에서도 앱이 동작하게 해줌. 카페에 재고가 떨어지면 대신 창고에서 가져오는 직원 같은 것
- **localStorage** — 브라우저가 내 기기에 데이터를 저장하는 공간. 내 폰 안에만 있는 메모장 같은 것. 다른 사람 폰에서는 안 보임
- **Firestore** — 구글 클라우드 데이터베이스. 여러 기기에서 실시간으로 같은 데이터를 공유할 수 있음. 공유 구글시트 같은 것

- 📐 인풋 미니멀리즘 (Input Minimalism) ⚖️
  매일 기록하는 앱에서 입력이 많으면 3일이면 포기함.
  선택지: ①상세하게 다 기록 (정확↑ 지속↓) ②핵심만 기록 (정확↓ 지속↑) ③핵심 필수 + 상세 선택
  판단: ③ — 지속이 안 되면 정확도는 의미 없음. 매일 30초가 핵심
  = 운동일지를 논문처럼 쓰면 3일이면 그만두는 것 (습관 앱의 법칙)

- 📐 로컬스토리지 vs 클라우드 (Local vs Cloud Storage) ⚖️
  localStorage는 간편하지만 혼자만 볼 수 있음. 길드 공유엔 Firestore 같은 클라우드 필요.
  선택지: ①localStorage로 빠르게 (개인용) ②처음부터 Firebase (느리지만 공유)
  판단: ① 먼저 → 쓸 만하면 ②로 전환. 데이터 없이 구조만 만들면 의미 없음
  = 일단 혼자 써보고 괜찮으면 카톡 단톡방 만드는 것
