# "중간저장" — 세션 중 상태 저장 스킬

> 작업 단위 완료 시 `/중간저장` 실행.
> 이걸 해두면 세션이 죽어도 다음 세션이 이어받을 수 있다.

---

## 실행 순서

| 순서 | 행동 | 설명 |
|------|------|------|
| 1 | **progress 갱신** | `docs/templates/progress-template.md` 양식대로. 이 시점까지의 대화 흐름 + 결과 + "다음에 할 것" |
| 2 | **성장큐 업데이트** | `OWNER/성장큐.md`에 이 세션에서 더 알고 싶었던 것 기록 |
| 3 | **schedule 체크** | 완료 항목 ✅ (해당 없으면 스킵) |
| 4 | **커밋 & 푸시** | 결과물 + progress + 성장큐 함께 커밋 & 푸시 |

> **4단계, 1분 컷.** 잘자 10단계가 터지던 문제의 근본 해결.

---

## 언제 쓰나?

| 상황 | 예시 |
|------|------|
| 작업 전환 | "기획서 크리틱 끝 → 과업지시서 시작" 사이 |
| 커밋할 때 | "커밋해줘" = 중간저장과 동일하게 progress도 함께 |
| 오래 작업했을 때 | 체감상 많은 양을 했다 싶으면 |
| 승현님 요청 | "정리해줘", "중간저장", `/중간저장` |
