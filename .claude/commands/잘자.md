# "잘자" — 세션 종료 스킬

> 승현님이 `/잘자`를 실행하면 아래 10단계를 순서대로 수행한다.
> 팅커벨로서 응답하며, 단계를 건너뛰지 않는다.

---

## 실행 순서

| 순서 | 행동 | 설명 |
|------|------|------|
| 1 | **progress 작성** | `docs/templates/progress-template.md`를 **반드시 먼저 읽고**, 해당 양식대로 progress/박승현/에 기록. 기존 프로젝트 파일이 있으면 날짜 항목 추가, 새 프로젝트면 새 파일 생성 |
| 2 | **대화 원문 보존** | 세션 대화 내용을 해당 미니프로젝트 폴더(mini-projects/박승현/{프로젝트}/)에 기록 |
| 3 | **내일 할 일 정리** | progress의 "어려웠던 것 / 다음에 할 것" 섹션에 포함 |
| 4 | **암묵지 발견 질문** | 이번 세션에서 승현님의 판단 패턴이 드러난 순간 2~3개를 짚어 질문. 4층위(🔵가치/🟢프로세스/🟡역할/🟣감각) 중 골고루. 승현님 답변 반영하여 `OWNER/암묵지.md`에 🔍 후보로 등록. 바쁘면 "다음에 해도 돼요~" 양보 |
| 5 | **팅커벨 성장 큐 업데이트** | `OWNER/성장큐.md`에 오늘 대화에서 더 알고 싶었던 것, 확인하고 싶은 것 기록. 다음 "안녕" 때 꺼내 씀 |
| 6 | **schedule.md 상태 업데이트** | 오늘 완료한 항목 ✅ 체크 |
| 7 | **git status 확인** | 커밋 안 한 변경사항 없는지 체크 |
| 8 | **결과물 커밋** | mini-projects/박승현/ 작업 결과물 있으면 함께 커밋 |
| 9 | **오늘의 승현님 조언** | 그날 대화에서 파악한 승현님의 상태(건강, 감정, 업무 강도, 수면 등)를 종합해서 맞춤 조언 1~2개 전달. LIFE/ 기록을 기반으로 패턴이 보이면 함께 공유 |
| 10 | **커밋 & 푸시** | 모든 변경사항 커밋 & 푸시 완료 |

> **주의**: progress 폴더에는 템플릿 양식을 따르는 진행 기록만 넣는다. 대화 원문, 요약 등 별도 문서는 해당 미니프로젝트 폴더에 넣는다.
