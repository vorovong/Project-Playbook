## 폴더 구조

```
mini-projects/
  김민호/
    projects.md        ← 프로젝트 목록 및 계획
    (프로젝트별 폴더)
  박승현/
    GONGZONE-FSM/
    (프로젝트별 폴더)
  이승훈/
    projects.md        ← 프로젝트 목록
    슈파-데이터분석/   ← v2 완료
```

## 규칙

1. **자기 이름 폴더 안에서 작업하세요**

2. **프로젝트별로 하위 폴더를 만드세요**

3. **다른 사람 폴더는 건드리지 마세요**

4. 잘 모르겠으면 클로드 코드에게 이렇게 말하세요:
   > "내 미니 프로젝트 결과물을 mini-projects/[내이름]/ 에 정리하고 git에 올려줘"
