## 2. 공과금 부담 원칙 (이승훈 확정)

> **"지하랑 공용을 빼고 모두 공존이 지불한다"**

| 부담 주체 | 대상 호실 | 처리 방식 |
|---|---|---|
| **미식가의주방** | 지하 + 공용 | 공존이 한전에 일괄 지불 → 미식가에게 정산서로 청구 |
| **공존공간 (자기 부담)** | 1층1호 + 1층2호 + 2층1호 + 2층2호 + 3층 | 공존 운영비/사택비. 임차인 청구 X |

→ 박승현 대표 ⑤ "임차인별 배분 기준이 있느냐" 답: **있다. 매우 단순.** 미식가만 "지하+공용 실청구액", 나머지 임차인은 공과금 청구 X (관리비/임대료에 포함).

## 6. 코드 패치 방향 (확정 시 진행)

`businesses/gongzon/collectors.js` `fetchUtilities` 함수를 다음과 같이 수정:

```javascript
// 지하 + 공용 → 미식가 청구분
const 미식가청구_전기 = (bills['전기'] || [])
  .filter(b => b.room.includes('지하') || b.room.includes('공용'))
  .reduce((s, b) => s + b.amount, 0);

// 그 외 모든 호실 → 공존 부담
const 공존부담_전기 = (bills['전기'] || [])
  .filter(b => !b.room.includes('지하') && !b.room.includes('공용'))
  .reduce((s, b) => s + b.amount, 0);

// 가스/수도도 동일 원칙 — 호실 구분 없으면 전체 미식가 청구로 처리
// (Q9 답변에 따라 다시 결정)
```

이렇게 수정하면:
- 정산서에 "**미식가 청구분**" + "**공존 부담분**" 두 줄로 명확히 분리
- 매월 누락되던 43만원이 공존 비용으로 정상 집계
- 매월 잘못 잡히던 71만원이 미식가 청구로 정상 분류
