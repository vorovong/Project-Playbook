### 2-2. 얼마에요 ERP (기업용) - 전체 기능

#### A. 경리/회계
- AI 로봇 자동 전표기장 (특허 기술)
- 카드/통장 내역 자동 수집 및 전표 처리
- 신용카드 매입/매출, 국세청 세금계산서 자동 수집
- 전자세금계산서 발행 (일반, 영세율, 면세, 수정 등) + 홈택스 자동 연동
- 고정자산 감가상각 자동 계산
- 어음 및 복수통화 관리
- 예산 편성 및 집행 관리
- 영수증 스캔 전표 처리
- 모바일 결재 및 승인 시스템

#### B. 인사/급여
- 근태 설정: 근태/휴가 유형 커스터마이징
- 근태 입력: 일괄 등록, 엑셀 가져오기
- 급여 자동 산정: 기본급/연장/야간/휴일 수당
- 통상임금/최저급여/주휴수당 자동 계산
- 4대보험 자동 처리 (취득/상실, 세법 변경 시 자동 업데이트)
- 근로계약서 전자 작성/서명
- 급여명세서, 원천징수영수증 발급
- 연차 현황 추적
- 퇴직금 자동 계산
- 인사 총무 양식 자동 생성

#### C. 매입/매출
- 거래처별 잔액 및 미수금 관리
- 판매/구매/미수미지급금 실시간 확인
- 거래명세표 메일 전송

#### D. 재고/유통
- 바코드 기반 재고 입출고 (모바일 스캔)
- 5단계 입출고 프로세스
- 재고 수량 조회, 분실/폐기 관리
- 바코드 라벨 출력
- 전용품목 관리

#### E. 제조/생산 (B1 전용)
- BOM(Bill of Materials) 관리
- 다공정 관리
- 외주공장 관리

#### F. 건설/건축 (C1 전용)
- 현장관리
- 일용직 관리
- 기성금 청구

#### G. 세무
- 부가세, 원천세, 연말정산 신고 무료 지원
- 세무사 무료 상담
- 세무사랑Pro 연동 (간편 부가세 신고)

#### H. 협업 (Works)
- 그룹웨어: 메일, 메신저, 전자결재 통합

## 3. 기술 스택 / 연동

### 3-1. API 제공

아이퀘스트는 공식 API를 제공한다 (https://iquest.co.kr/products/olma_api.do)

| API | 기능 | 비고 |
|-----|------|------|
| **빌택스 API** | 전자세금계산서 정발행/역발행, 국세청 자동 전송 | 사내 시스템 연동 |
| **얼마싸인 API** | 블록체인 기반 전자계약, 대량 전송 | Premium 요금제 필요 |
| **예금주 조회 API** | 은행코드+계좌번호로 실시간 예금주 확인 | 주요 은행/금융기관 |

- 인증: 도입 문의 후 인증 ID/Key 발급 -> 액세스 토큰 방식
- 보안: API 요청/응답 간 암호화

### 4-3. 장점

1. **엑셀과 동일한 입력 방식** - 직원 학습 비용 낮음
2. **클라우드 기반** - 설치 불필요, 어디서든 접속
3. **거래내역 입력 -> 매입매출전표 자동 연동** - 이중입력 불필요
4. **홈택스/은행/카드 자동 수집** - 수작업 최소화
5. **세무사 무료 상담** 제공
6. **부가세/원천세/연말정산 무료 지원**
7. **비용 대비 가성비** 우수 (특히 3~14인 규모)
8. **모바일 앱** 실시간 경영현황 확인

### 4-4. 단점 / 불만

1. **서버 불안정** - PC 접속 장애 빈번, 모바일 로그인 오류
2. **장애 시 공지 부재** - 서버 다운 시 사전/사후 안내 미흡
3. **고객센터 응답 형식적** - 실질적 해결보다 정형 답변
4. **서버 속도** - 모바일 접속 시 느리다는 피드백
5. **소상공인용(얼마)과 기업용(얼마에요) 구분 혼란** - 어떤 제품을 써야 하는지 파악 어려움

## 7. 공존공간 적용 관점 평가

### 적합한 점
1. **"얼마" 소상공인 앱이 가장 적합** - 기업용 ERP는 과도
2. 급여관리 4인 무료 (소규모 사업장에 유리)
3. 4대보험 신고 기능 내장
4. 세금계산서 전자발행 가능
5. 오픈뱅킹 연동으로 계좌 관리 자동화
6. 부가세/원천세 신고 지원

### 부적합/주의할 점
1. POS 연동 없음 (캐시노트가 이 부분은 강점)
2. 서버 안정성 이슈 보고됨
3. 소상공인용 "얼마"와 기업용 "얼마에요" 간 기능 차이 명확히 확인 필요
4. 데이터 내보내기(엑셀 등) 범위 확인 필요
5. API가 있지만 소상공인 앱 수준에서는 활용 어려움

### 결론
> "얼마"는 소상공인이 급여/4대보험/세금계산서/장부를 **하나의 앱에서** 해결할 수 있는 몇 안 되는 올인원 서비스다. 캐시노트(매출분석)나 삼쩜삼(세금환급)이 특정 기능에 특화된 반면, 얼마는 **경영관리 전반**을 커버한다. 다만 각 기능의 깊이는 전문 서비스 대비 얕을 수 있으므로, 핵심 니즈(급여? 장부? 세금?)에 따라 조합 사용도 고려할 것.
