## 3. 기술 스택 / 연동

### API 및 연동

| 항목 | 상태 |

| **공개 API** | 미제공 (외부 개발자용 API 없음) |

- **현대벤디스**: 업무협약
- **인천소상공인연합회**: 업무협약

### 기술적 한계
- 외부 시스템 연동 API 부재 -> 자체 자동화 구축 불가

- 데이터 일괄 Export(CSV, Excel 등) 기능 미확인

## 7. 공존공간 관점 시사점

### 적합성 평가

| 관점 | 평가 | 설명 |
|------|------|------|
| **비용** | ★★★★★ | 완전 무료, 즉시 도입 가능 |
| **알바 근태관리** | ★★★★☆ | GPS/QR/자동 3가지 방식, 1분 단위 정밀 |
| **급여 정산** | ★★★★☆ | 자동 계산, 명세서 발급까지 |
| **법적 문서** | ★★★★★ | 근로계약서, 급여명세서, 3년 보관 |
| **시스템 연동** | ★☆☆☆☆ | API 없음, POS/회계 연동 불가 |
| **확장성** | ★★☆☆☆ | 알바 특화, 정규직/복합 관리 한계 |
| **자동화 연계** | ★☆☆☆☆ | 외부 자동화 파이프라인 연결 불가 |

### 도입 시 고려사항

**장점 (도입 근거)**
1. 비용 제로 - 소상공인에게 최적
2. 근로기준법 준수 자동화 (계약서, 명세서, 보관)
3. 알바몬에서 채용한 직원을 바로 관리 가능
4. 학습 비용 낮음 (직관적 UI)

**한계 (주의사항)**
1. 알바몬 기업회원 필수 (종속성)
2. 외부 시스템 연동 API 없음 -> 자체 자동화 시스템과 데이터 연계 불가
3. 정규직 인력 관리에는 부적합
4. 임차인 정산, 공과금 관리 등 공존공간 고유 업무와 직접 연결 불가
5. 데이터 Export 제한적 -> 별도 대시보드 구축에 활용 어려움

### 결론

보스몬은 **알바생 근태/급여 관리 단독 용도**로는 무료라는 압도적 장점이 있어 즉시 도입 가능하다. 
그러나 공존공간의 **통합 자동화 시스템**(임차인 정산, 공과금, 매출 관리 등)과 연결하려면 
API가 없어 데이터 파이프라인 구축이 불가하다는 근본적 한계가 있다.

**추천 활용 방안:**
- 단기: 보스몬으로 알바 근태/급여 즉시 관리 (비용 절감)
- 중장기: 시프티(API 제공) 또는 자체 구축 시스템으로 전환 검토
