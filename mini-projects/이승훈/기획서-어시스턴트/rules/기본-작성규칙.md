# 기본 작성 규칙

모든 문서 작성 시 항상 적용한다.
출처: 2025 개정 공문서 작성법 길라잡이 (2025. 1. 8.)

---

## 0. 문체 (필수)

### A. AI스러운 표현 금지

기획서는 사람이 쓴 것처럼 읽혀야 함. 아래 패턴을 쓰지 않음.

| 금지 패턴 | 왜 문제인지 | 대신 |
|---|---|---|
| "전략적 결합/시너지/융합" | 포장만 있고 내용 없음 | 구체적 사실로 대체 |
| "~에서 나온", "우연이 아니라" | 드라마틱하게 꾸밈 | 사실 관계만 서술 |
| "로컬 생태계를 함께 만들어온" | 추상적, 사람이 안 쓰는 말 | 실제로 뭘 했는지 쓸 것 |
| "혁신적인/획기적인/차별화된" | 자기 자랑 수식어 | 수치나 사실로 증명 |
| "패러다임/레버리지/임팩트" | 불필요한 외래어 나열 | 쉬운 말로 |
| "~을 통한 ~의 극대화" | AI 특유의 구조 | 짧게 끊어 쓸 것 |

**핵심 원칙:**
- 승훈님이 투자자 앞에서 **말로 설명하듯** 쓸 것
- 포장 대신 **팩트**(숫자, 날짜, 실적)
- 문장이 어색하면 틀린 것임. 소리 내어 읽었을 때 자연스러워야 함

### B. 음슴체 (필수)

모든 문서에서 음슴체를 기본으로 사용한다.

| 사용 | 사용 안 함 |
|---|---|
| ~함 | ~합니다 |
| ~임 | ~입니다 |
| ~것 | ~것입니다 |
| ~필요함 | ~필요합니다 |
| ~완료함 | ~완료하였습니다 |
| ~예정임 | ~예정입니다 |
| ~바람 | ~바랍니다 |

예시:
- "매출 30억 달성함" (O)
- "3단계 로드맵 수립 완료" (O)
- "신규 파트너십 체결 예정임" (O)

## 1. 항목 번호 체계

### 항목 기호 (우선순위/순서가 있을 때)

```
첫째 항목:   1., 2., 3., 4., ...
둘째 항목:     가., 나., 다., 라., ...
셋째 항목:       1), 2), 3), 4), ...
넷째 항목:         가), 나), 다), 라), ...
다섯째 항목:         (1), (2), (3), (4), ...
여섯째 항목:           (가), (나), (다), (라), ...
일곱째 항목:             ①, ②, ③, ④, ...
여덟째 항목:               ㉮, ㉯, ㉰, ㉱, ...
```

### 특수 부호 (대등 병렬일 때)

```
□ → ○ → - → •
```

### 규칙
- 항목 기호와 내용 사이: 1칸 띄움
- 하위 항목: 앞 항목에서 2칸씩 오른쪽
- 항목이 하나뿐이면: 항목 번호 생략
- 대등하게 이어진 문장은 항목을 나누고, 종속적으로 이어진 문장은 나누지 않음
- 첫째 항목 간 한 줄 띄어도 됨

## 2. 날짜·시간·금액 표기법

### 날짜
- 아라비아 숫자, '일' 다음 반드시 마침표: `2025. 3. 10.`
- 요일 포함: `2025. 3. 10.(월)`
- 기간: `2025. 2. 20.~2. 24.` 또는 `2025. 2. 20.-2. 24.`
- 연도만 쓸 때: `2025년` (숫자 뒤에 '년')
- 연도에 쉼표 사용 금지: ~~2,025~~ → `2025`

### 시간
- 24시각제 + 아라비아 숫자: `09:00`, `13:20`
- 10 미만은 0 포함: `09:00` (~~9:00~~)
- 시만 쓸 때: `09시`
- 시와 분 사이 쌍점, 띄움 없음: `14:30`
- 일시 표기: `2025. 2. 28.(금) 14:00`
- 마감 표기: `서류 마감: 18:00` (~~마감 시한~~, ~~밤 12시까지~~)

### 금액
- 일반: `금500,000원(금오십만원)`
- 계약: `일금500,000원정(일금오십만원정)`
- '금' 바로 뒤에 숫자 (띄움 없음)
- 글자와 숫자 금액이 다르면 → 글자 금액 인정 (단, 글자가 맞춤법 틀리면 숫자 인정)

## 3. 표현법 6원칙

### (1) 사실성
- 나타내고자 하는 바와 진술된 표현이 사실적으로 일치
- 사동 표현 유의 (향상시키다 → 향상하다, 발송시키다 → 보내다)
- 중첩어 제거: 2월달→2월, 기간 동안→기간에, 새로 나온 신상품→신상품
- 다중 표현 주의: 안전선 밖으로 물러나다 → 안전선 안으로 물러나다

### (2) 용이성
- 읽기 쉽고 알기 쉬운 용어 사용
- 외래어보다 우리말: 노미네이트→후보 지명, 로하스→친환경 살이, 블랙아웃→대정전
- 문장은 짧게 끊어서 항목별로 표현
- 전문 용어는 풀어쓰고 괄호 안에 원어: `연구 개발(R&D)`

### (3) 명확성
- 뜻이 불분명한 말 피하기
- 의무·금지·재량·예외 분명하게
- 문장 구성 요소 생략하지 않기
- 기한 명확히: `2025. 2. 28. 18:00까지 서류 제출 마감` (날짜만 쓰면 몇 시까지인지 불명확)

### (4) 비고압성
- 부정적 표현보다 긍정적 표현: 입산 금지 → 입산이 제한되어 있습니다
- 행동 제약보다 정보 안내: 들어가지 마시오 → 이곳은 금연 구역입니다
- 감정적 표현 배제
- 상대방 중심으로 표현

### (5) 비표지성
- 특정 지역·학교·성·상품 등을 이유 없이 부각하지 않기
- 우선순위 있으면 우선순위 적용, 없으면 가나다순
- 성별 표기: 수가 많은 쪽 먼저 — `참가자 60(여 35, 남 25)`

### (6) 수요자 중심
- 주어를 수요자(고객, 독자)로 놓기
- 교부→받다/수령, 접수→신청/제출, 공지사항→알림/안내
- 예: "여권을 교부합니다" → "여권을 받으실 수 있습니다"

## 4. 띄어쓰기 핵심

- 기관명·단체명·국가명: 붙여 씀 (한국공공언어진흥원, 행정안전부장관)
- '자'와 '-부': 앞말과 띄움 (`2025. 2. 28. 자` / `2025. 2. 28.부`)
- ~시행령/시행규칙: 모법과 띄움 (`국어기본법 시행령`)
- 조·항·호·목 번호: 서로 붙여 씀 (`제1조제1항제1호`)
- 안, 밖, 전, 후, 이상, 이하, 미만, 이내: 앞말과 띄움 (`1년 이상`, `학교 안`)
- 중, 내, 외: 앞말과 띄움 (`기일 내`, `관계전문가 중`)

## 5. 문장 부호 핵심

- **마침표**: 연월일 아라비아 숫자 뒤에 찍음 (`2025. 3. 10.`)
- **쌍점**: 표제 뒤 항목 나열 시 (`장소: 한국공공언어진흥원`), 시·분 구별 (`14:30`), 쌍점 앞에 빈칸 없음
- **가운뎃점(·)**: 짝이루는 어구 사이 (`조사·분석`), 공통 성분 줄임 (`상·중·하위권`)
- **겹낫표(『』)**: 책·신문 이름 (`『삼국유사』`)
- **홑낫표(「」)**: 법률·규정·작품명 (`「건축법」`, `「헌법」`)
- **소괄호**: 보충 설명 (`커피(coffee)`, `2025. 3. 10.(월)`)
- **대괄호**: 괄호 안에 괄호 필요시 바깥 (`[홍길동(안전감사실)]`), 한자어 대응 (`떨어짐[추락]`)

## 6. 구별해야 할 말

| 혼동 쌍 | 구별 |
|---|---|
| 참고 / 참조 | 참고: 한 개 제시. 참조: 두 개 이상 비교·대조 |
| 주최 / 주관 | 주최: 기획. 주관: 맡아서 시행 |
| 소계 / 합계 / 총계 | 소계: 작은 단위. 합계: 동일 과목. 총계: 과목별 합계의 합 |
| 고시 / 공고 | 고시: 계속 효력, 법적 구속력. 공고: 일시적, 구속력 없음 |
| 기획 / 계획 | 기획: 무엇을 할 것인가. 계획: 어떻게 할 것인가 |
| 년 / 년도 | 년: 반복·연속 없음. 년도: 반복·연속·지속 있음 |
| 이상 / 초과 | 이상: 그 수 포함. 초과: 그 수 미포함 |
| 이하 / 미만 | 이하: 그 수 포함. 미만: 그 수 미포함 |
| 이외 / 외 | 이외: 대상 포함하여 밖 (괴산 이외 3곳). 외: 대상 빼고 밖 (괴산 외 2곳) |
| 전 / 이전 | 전: 그 날 미포함 (2월 10일 전 = ~2월 9일). 이전: 포함 (~2월 10일) |
| 아래 / 다음 | 아래: 동일 공간에 하위 내용. 다음: 이질 공간까지 이어짐 |
| 퍼센트 / 퍼센트포인트 | %는 비율 비교. %p는 % 단위끼리 비교 |
| 제척 / 기피 / 회피 | 제척: 자동 배제. 기피: 제3자 청구. 회피: 본인 신청 |
