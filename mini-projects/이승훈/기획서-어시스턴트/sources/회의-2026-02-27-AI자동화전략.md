## 한줄 요약
소상공인 맞춤형 자동화 솔루션 사업화 + AI 회의 비서를 단순 요약 → 내부 업무 시스템 연동으로 고도화.

## 핵심 결정
- 소상공인 자동화 사업: 무거운 ERP 대신 가벼운 맞춤 자동화 도구 제공
- AI 회의 비서 진화: 회의록 → 액션 아이템 자동 분류 → AI 수행 가능 과업 즉시 실행
- 구글 Gemini API 기반 시스템 (시간당 약 700원)
- 노션 연동: GitHub 접근성 낮은 팀원을 위해 최종 결과물은 노션으로
- 피지컬 AI: 제조업 숙련공 동작을 비전 AI로 촬영 → 데이터셋 구축

## 기획서 활용 포인트
- "AI를 직접 만들어 쓰는 팀" = 기술 내재화 역량 증명
- 회의 → 실행까지 AI 파이프라인 = 운영 효율성 어필
- 피지컬 AI 데이터셋 = 장기 비전의 구체적 실행 근거
