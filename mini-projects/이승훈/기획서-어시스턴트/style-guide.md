# 글쓰기 DNA — 이승훈

## 1. 어미
- 음슴체 (~함/~임/~완료/~예정)
- "합니다"체는 소상공인 대상 안내문에서만 사용

## 2. 구조
- 서술보다 **표로 정리**
- 한 문단에 하나의 메시지
- 반복/군더더기 삭제 — 같은 말 두 번 하지 않음

## 3. 데이터 vs 스토리
- **항목별로 맞는 방식을 쓴다**
- 숫자가 있는 항목 → 숫자로 말함 (Before/After 비교)
- 숫자가 없는 항목 → 스토리로 설명 (맥락으로 설득)

## 4. 표현 원칙
- 정확한 표현 (논의사항 ≠ 결정사항, 제안 ≠ 추진)
- 맥락에 맞는 톤 (중기부가 한 말 ≠ 현장이 건의한 말)
- 불필요한 섹션은 과감히 삭제

## 5. 금지
- AI스러운 표현 ("획기적인", "혁신적인", "다양한")
- 같은 내용 반복
- 불확실한 숫자 (추측 금지)
- 장황한 배경 설명

## 6. 숫자 표현
- 금액: 1억 2,000만 원 (쉼표 + 만 원 단위)
- 비교: 14억 → 29억 (+107%)
- 날짜: 2026. 3. 11.
- 시간: 24시각제 09:00
