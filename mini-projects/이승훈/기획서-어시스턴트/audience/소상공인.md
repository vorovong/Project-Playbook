# 청중 프로필: 소상공인 (슈파 사용자)

## 누구인가
- 슈파를 사용해서 식자재를 공동구매하는 가게 사장님들
- 바쁘다. 복잡한 건 안 읽는다.
- "이거 쓰면 나한테 뭐가 좋은데?"가 전부
- "모든 상인은 창작자다" — 보호 대상이 아닌 동등한 파트너로

## 중시하는 것 (공통)

| 순위 | 항목 | 설명 |
|---|---|---|
| 1 | 가격 | 얼마나 싸지는가, 절감 효과 |
| 2 | 편리함 | 주문이 쉬운가, 배송은 어떤가 |
| 3 | 품질 | 식자재 품질이 괜찮은가 |
| 4 | 신뢰 | 다른 사장님들도 쓰고 있는가 |

---

## 연령별 차이

### 20~30대 젊은 사장님

| 항목 | 특징 |
|---|---|
| 관심 포인트 | 있어 보이는 언어, 테크, 포장이 중요 |
| 톤 | 세련되고 트렌디한 느낌 |
| 설득 방식 | 브랜딩, 데이터 시각화, 앱/테크 강조 |
| 채널 | 앱, SNS, 카톡 자연스러움 |

### 50~60대 사장님

| 항목 | 특징 |
|---|---|
| 관심 포인트 | "나한테 뭐 해줄 건데?" — 직접적 이익 |
| 톤 | 매우 쉽고 직접적 |
| 설득 방식 | 가격 절감 금액, 실제 사례, 간단한 비교 |
| 채널 | 전화, 대면, 카톡 (단순한 형태) |

---

## 문서 형식
- 서비스 소개서 (1~2페이지)
- 카카오톡으로 공유 가능한 간결한 형태
- 가격 비교표, 후기/사례
- 연령대에 따라 같은 내용도 다른 버전으로 만들 수 있음
