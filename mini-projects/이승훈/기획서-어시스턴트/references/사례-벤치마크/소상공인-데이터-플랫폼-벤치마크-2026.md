## 2. 캐시노트 (KCD/한국신용데이터) — 데이터 사업 모델 심층 분석

### 2-1. 3대 수익 축

| 수익원 | 설명 | 비고 |
|---|---|---|
| **금융상품 중개** | 매출/계좌 데이터 기반 소상공인 맞춤 대출 추천 → 금융사 중개 수수료 | 29개 금융사, 33개 금융상품 연결 |
| **B2B 데이터 판매 (크레딧브리지)** | 사업자 동의하에 운영 데이터(매출 추이, 단골 수, 단골 매출 비율 등)를 금융사에 전달 | 개인사업자 신용평가용 |
| **멤버십 구독 (캐시노트 플러스)** | 월 33,000원 유료 구독 — 사업 필수 패키지 | 2023년 출시 |

### 2-2. 데이터 판매 구조

- **데이터 수집**: 캐시노트 앱을 통해 사업장 매출, 카드결제, 계좌 흐름, 단골 비율 등 수집
- **데이터 규모**: 연간 522조 원 규모 거래정보 (170만 사업장)
- **데이터 구매자**: 은행, 카드사, 보험사 등 금융기관
- **활용 방식**: 개인사업자 신용평가 시 기존 개인 CB 점수 + 캐시노트 사업 데이터 결합
- **제품명**: 캐시노트 크레딧 브리지 (Cashnote Credit Bridge)
- **핵심 차별점**: 기존 금융권이 가지지 못한 "사업장 실제 운영 데이터"를 제공

### 2-3. 장기 전략

- **소호은행 설립 추진**: '한국소호은행 컨소시엄' 결성, 소상공인 특화은행 인가 추진 중
- **대안신용평가**: 개인신용 중심 평가 → 사업역량 기반 평가 모델로 전환
- **월간 흑자 달성**: 2024년 첫 월간 흑자 → 지속가능 성장 변곡점

## 9. 슈파 관점 시사점

### 캐시노트 모델의 핵심 교훈

1. **무료 앱으로 사업장 확보 → 데이터 축적 → 금융사에 데이터 판매**: 이것이 KCD의 핵심 플라이휠
2. **사업자 동의 기반**: 개인정보 이슈를 "사업자가 스스로 금융사에 데이터를 제공해 더 좋은 대출 조건을 받는" 구조로 해결
3. **크레딧브리지**: 기존 금융권이 갖지 못한 "사업장 실제 운영 데이터"를 B2B로 판매 — 이것이 데이터 사업의 핵심
4. **소호은행 설립 추진**: 궁극적으로 데이터를 가진 자가 직접 금융을 하겠다는 전략

### 슈파가 참고할 수 있는 데이터 사업 방향

| KCD 모델 | 슈파 적용 가능성 |
|---|---|
| 사업장 매출 데이터 → 금융사 신용평가 | 식자재 구매 데이터 → 식품 유통사/금융사 제공 |
| 170만 사업장 규모의 경제 | 로컬 특화 (수원 행궁동 등) → 상권 데이터 |
| 금융중개 수수료 | 식자재 공동구매 + 금융상품 연결 |
| 유료 멤버십 | AI 경영 자동화 프리미엄 |
