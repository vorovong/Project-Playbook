## 1. 한국 AC(액셀러레이터) 등록 현황

### 등록 요건

| 항목 | 요건 |
|------|------|
| **법적 근거** | 벤처투자촉진에관한법률 (구 중소기업창업지원법) |
| **법인 형태** | 상법상 회사 |
| **자본금** | 1억원 이상 |
| **전문인력** | 상근 2인 이상 (기술사, 변호사, 공인회계사, 변리사, 박사(이공/경상) 등 자격요건 충족자) |
| **등록 방법** | 벤처투자 종합정보시스템(VICS) 온라인 신청 |
| **처리 기간** | 접수일로부터 30일 이내 적부 통보 |
| **주된 업무** | 초기창업자 선발/전문보육, 초기창업자 투자, 개인투자조합/벤처투자조합 결성 및 집행 |

### 등록 현황 (추정)

- 2020년: 300개 돌파 (이후 일부 반납하여 약 290개 활동)
- 2023년 5월: 약 439개 등록
- 2023년 7월: 15곳 등록 말소 (자격 반납/취소)
- 2025~2026년: 정확한 수치는 [창업진흥원 등록현황 페이지](https://www.kised.or.kr/prePubDetail/index.es?mid=a10103010000&prePubId=9)에서 확인 필요

### 트렌드

- 등록 수 지속 증가 추세이나, 실적 미달로 자격 반납하는 AC도 증가
- "양적 팽창 vs 질적 관리" 이슈 부각 — 오히려 **차별화된 AC에 기회**

### 신도시 활용 포인트

- 자본금 1억원, 전문인력 2인이면 등록 가능 — **진입 장벽이 높지 않음**
- 등록 AC 439개+ 중 "로컬 특화 AC"는 소수 — **로컬 전문 AC로 차별화 가능**
- 등록 후 TIPS 운영사 선정, 정부 사업 주관기관 참여 등 연계 가능

## 3. 벤처스튜디오 vs 전통 AC 모델 비교

| 비교 항목 | 전통 AC (액셀러레이터) | 벤처스튜디오 |
|-----------|----------------------|-------------|
| **핵심 활동** | 기존 스타트업 선발 → 단기 집중 프로그램 | 직접 아이디어 발굴 → 회사 설립 → 운영 |
| **프로그램 기간** | 보통 3~6개월 배치 | 지속적 (설립~스케일업까지) |
| **지분 취득** | 5~15% | 30~80% |
| **초기 투자** | 소규모 (시드) | 중규모 ($500K~$2M+) |
| **관여 수준** | 멘토링/네트워킹 중심 | 운영팀 직접 참여 (제품개발, 마케팅, 채용 등) |
| **포트폴리오 수** | 배치당 다수 (10~30개) | 소수 집중 (연 2~5개) |
| **마무리** | 데모데이 → 후속 투자 유치 | 독립 운영 가능 시점까지 동행 |

### 성공률 비교 (핵심 수치)

| 지표 | 전통 스타트업 | 벤처스튜디오 스타트업 |
|------|-------------|---------------------|
| **전반적 성공률** | 기준 | +30% 높음 |
| **시드 라운드 진입률** | — | 84% |
| **시리즈A 도달률** | 42% | 72% |
| **제로→시리즈A 소요 기간** | 56개월 | 25.2개월 (절반 이하) |
| **평균 IRR (내부수익률)** | 21.3% | 53% |

- 출처: [Bundl](https://www.bundl.com/articles/why-venture-studio-startups-have-higher-long-term-success-rates), [VC Lab](https://govclab.com/2025/03/17/venture-studio-versus-accelerator-funds/), [SeedReady](https://www.seedready.org/resources/incubators-accelerators-venture-studios/)

### 벤처스튜디오가 로컬에 적합한 이유

- 소수 집중 모델 → 지역 내 유망 기업에 깊은 관여 가능
- 운영팀 직접 참여 → 창업 경험 부족한 로컬 창업자에게 실행력 보완
- 높은 지분 → 장기적 동반 성장 인센티브
- MIT 논문: "이전 창업 경험이 벤처스튜디오 성공의 핵심 조건" — 지역 사업 경험이 있는 신도시 팀에 유리

### 신도시 활용 포인트

- **"AC 등록 + 벤처스튜디오 운영 모델" 하이브리드** 전략 가능
- AC 등록으로 정부 사업 접근권 확보 + 실제 운영은 벤처스튜디오 방식 (깊은 관여, 높은 지분)
- 기획서에서 "전통 AC 대비 성공률 30% 높음, 시리즈A 72% 도달" 수치 인용 가능
- IRR 53% vs 21.3% — 투자자/출자자 설득에 강력한 데이터

```
    ↓
투자조합 결성 → 민간 자금 확보 → 로컬벤처 펀드 운용
```

## 종합: 신도시 기획서 핵심 활용 포인트 요약

### 시장 기회

1. 벤처투자 시장 12조+ 규모이나 **로컬/임팩트 분야는 블루오션**
2. 소셜벤처 투자 **1년 만에 9배 성장** (282억→2,671억)
3. 등록 AC 400개+ 중 **로컬 전문 AC는 극소수** → 차별화 포지션

### 모델 우위

4. 벤처스튜디오 성공률 **전통 대비 30% 높음**, 시리즈A 도달률 **72% vs 42%**
5. IRR **53% vs 21.3%** — 투자 수익률도 압도적
6. "AC 등록 + 벤처스튜디오 운영" 하이브리드로 **정부 지원 + 깊은 관여** 양쪽 확보

### 정책 연계

7. AC 등록 → TIPS 운영사 → 발굴 기업에 **최대 15억원 R&D 연계**
8. 로컬크리에이터 사업 → 선정 후 **최대 9억원 추가 지원** 연계 가능
9. 예비/초기/도약 창업패키지 주관기관 참여로 **지속적 정부 사업 수주**

### 벤치마킹 대상

10. **소풍벤처스/소풍커넥트**: 임팩트+로컬 투자 모델의 직접적 레퍼런스
11. **로컬스티치**: 공간 기반 크리에이터 커뮤니티 → 사업 창출 모델
12. **Parallel18**: 지역 기반이지만 글로벌 연결하는 혁신 허브 모델
