## 1. 공통 운영 체계

### 1-1. 파일 저장 규칙

| 저장소 | 용도 |
|--------|------|
| **나스(NAS)** | 전체 파일 아카이빙 |
| **구글 드라이브** | 최종 파일 (24, 25년 지원사업 위주) |

**파일명 규칙**

- 구조: `[공존공간] 문서명_사업명_초안/버전/최종`
- 예시: `[공존공간] 사업계획서_글로컬상권창출팀_초안`

**프로젝트 코드 형식**: `[유형]-[연도]-[일련번호]_[사업명]`

| 유형 코드 | 의미 |
|-----------|------|
| GOV | 정부/지자체 공모·지원 |
| CONS | 용역/컨설팅 |
| PRJ | 자체 프로젝트/행사 |
| OPS | 운영/상시 업무 |
| HR | 인사/채용/내부규정 |

**인덱스 필수 컬럼**: 프로젝트코드, 프로젝트명, 기간, Drive 폴더 링크, NAS 폴더 링크, 최종 산출물 Top3 링크, 비고

### 1-2. 수발신 공문 관리

#### 발신 공문 작성

1. 발신공문 작성
2. [구글 공유드라이브] → [수발신공문] → [수발신 공문접수/엑셀] → [발신탭] 작성하여 시행번호 부여
3. 파일명: `시행번호_공문제목` (예: `2025-016_공문제목`)
4. hwp 파일 → [발신공문파일] 드라이브 저장 / pdf 파일 → 수신처에 발송

#### 수신 공문 접수

1. 수신공문 파일 저장
2. [수발신 공문접수/엑셀] → [수신탭] 작성
3. 파일명: `접수날짜_(발신처) 공문제목` (예: `250401_(수원도시재단) 파일명`)

### 1-3. 수의계약 기준

| 구분 | 조건 |
|------|------|
| **1인 견적** | 추정가격 2천만원 이하 (여성/장애인 기업: 5천만원) |
| **2인 이상 견적** | 추정가격 5천만원 이하 (용역/물품/기타) |

참고: 나스 [coexistence] → [송수남 인수인계] → [4. 기타 참고자료] → 수원시 2022년 계약실무 (39페이지)

### 1-4. 회의록 작성 규칙

**필수 포함 항목**: 일시, 장소, 회의명, 참석자, 회의 내용, 회의 사진

**참석자 서명지**: 회의 날짜·시간 명시, 성명·소속·서명 기재

**사진 촬영 주의사항**:
- 참석자 전원이 나오도록 촬영
- 다과/식사 제공 시 — 다과 사진 + 전원 착석 사진 촬영
- 식사 장소에서 **술이 보이면 안 됨** → 식사 나오기 전 전원 착석 시 촬영
- 현수막/배너가 잘 보이게 촬영
- 주요 인사(시장, 구청장, 회장 등) 참석 시 반드시 함께 촬영

**물품 구매 증빙 촬영**: 제품·수량이 잘 보이게, 영수증과 날짜·품목·수량 일치

**파일 정리**:
- 나스 경로: `/coexistence/----- 사단법인 행궁동/# 사단법인 행궁동행 회의록`
- 폴더명: `날짜_회의명` (예: `250703_나혜석 생가터 회의록`)
- 포함: 회의자료, 회의록, 회의 사진
- **회의 후 바로 정리할 것**

### 1-5. 세부 사업계획서/결과보고서 작성

- 각 사업 진행 시 세부 사업계획서 + 결과보고서 작성 (최종 결과보고서 작성을 쉽게 하기 위함)
- 지출 관련 — 품의서 또는 내부기안 작성하여 근거 문서 생성
- 품의서 대체 가능: 내부기안(세부 사업계획서/결과보고서)을 근거로 지출 승인
- **내부기안 번호 발급**: 구글시트에서 관리
- **저장**: 나스 [coexistence] → [-----2025 내부기안] / 파일명: `내부기안번호_제목`

#### 경기도청 확인 사항 (임영섭 주무관)

**회원 수 기준**:

- 회원명부 중 5~10명 **전화 확인** 실시

**비영리법인 공유자산**: 운영 가능. 정관에 수익 목적·항목 명시 필요. 수익금은 설립 목적·공공 사업에 재투자.

### 6-7. Q&A (회원 응대용)

**Q. 사단법인 행궁동행은 어떤 곳인가요?**
행궁동의 지속적 활동 기반을 만들고, 지역 주민·상인과 공익 활동을 제도적으로 운영하기 위한 사단법인입니다.

**Q. 어떤 일을 하나요?**
상권 활성화, 공동체 활동(주민·상인 모임), 문화 발전, 직접사업(수월래/소식지/토론), 전략사업(주차장/치맥거리/마을마이스)

**Q. 누가 회원이 될 수 있나요?**
행궁동 거주자/상인, 발전 활동 단체·개인, 공동체에 관심 있는 분

**Q. 회비는 어디에 쓰이나요?**
사업 진행, 소식지 제작, 행사 간식비, 홍보 인쇄비 등 공익 목적 사업에만 사용. 연말 보고서·총회 통해 투명 공개.

**Q. 수월래가 뭐예요?**
수원화성 성곽길을 절기마다 한복/흰 티셔츠 입고 청사초롱 들고 걷는 야간 문화 의례. 회원 중심 운영, 추후 일반 시민으로 확대 예정.
