# 슈파(SUPA) MVP — 소상공인 반복 업무 자동화 리서치

## 소상공인 반복 잡무 목록

소상공인(음식점/카페/소매점)이 **장사 외에** 매일/매주 반복하는 업무를 8개 영역으로 분류했다.

---

### 1. 리뷰 답변 관리

| 항목 | 내용 |
|------|------|
| **설명** | 배달앱(배민, 요기요, 쿠팡이츠), 네이버 플레이스, 카카오맵, 구글맵 등에 올라오는 고객 리뷰에 답변 작성. 부정 리뷰 대응, 감사 답변, 별점 관리 등 |
| **빈도** | 매일 (배달 업종은 하루 10~50건 이상) |
| **시간 소요** | 하루 30분~2시간 |
| **고통 지수** | ★★★★★ (부정 리뷰 스트레스 + 시간 소모 + 무시하면 매출 직결) |
| **AI 해결 가능성** | **높음** — 리뷰 톤 분석 후 맞춤 답변 초안 자동 생성. 부정/긍정 분류, 핵심 키워드 추출도 가능 |
| **기술 난이도** | ★★☆☆☆ (LLM API + 플랫폼별 리뷰 크롤링/연동) |
| **참고 서비스** | **딜리봇** — 배달앱 리뷰 자동 답변 서비스. "하루 2시간 → 10분" 표방. 배민/요기요 등 멀티플랫폼 통합 관리 |

---

### 2. SNS/마케팅 콘텐츠 제작

| 항목 | 내용 |
|------|------|
| **설명** | 인스타그램, 네이버 블로그, 카카오채널 등에 올릴 홍보 콘텐츠(사진+글) 제작. 신메뉴 소개, 이벤트 공지, 일상 포스팅, 스토리/릴스 제작 |
| **빈도** | 주 2~5회 (꾸준히 해야 효과) |
| **시간 소요** | 건당 30분~1시간, 주 3~5시간 |
| **고통 지수** | ★★★★☆ (뭘 올려야 할지 모르겠음 + 촬영/편집 부담 + 효과 불확실) |
| **AI 해결 가능성** | **높음** — 텍스트 카피, 해시태그 자동 생성. 이미지 템플릿 자동 적용. 게시 일정 자동화 |
| **기술 난이도** | ★★★☆☆ (LLM + 이미지 생성/편집 + SNS API 연동) |
| **참고 서비스** | **스마트포스트** — 멀티 SNS 예약 포스팅, 웹 이미지 편집기 제공 / **소셜비즈(NHN)** — 인스타 DM 자동응답, 마케팅 자동화 / **Mirra** — 한글 최적화 AI 콘텐츠 생성, 캐러셀/숏폼 자동 제작 / **해외: Jasper AI, Canva AI, Later** |

---

### 3. 메뉴/가격표 관리

| 항목 | 내용 |
|------|------|
| **설명** | 메뉴판 디자인 변경, 배달앱 메뉴 업데이트, 네이버 플레이스 정보 수정, 가격 인상 시 전 채널 동시 반영, 계절 메뉴 교체 |
| **빈도** | 월 1~4회 (가격 변동, 시즌 메뉴 등) |
| **시간 소요** | 건당 1~3시간 (플랫폼별로 각각 수정해야 하므로) |
| **고통 지수** | ★★★☆☆ (귀찮지만 빈도 낮음. 단, 배달앱 3~4개 동시 관리 시 높아짐) |
| **AI 해결 가능성** | **중간** — 메뉴판 디자인 템플릿 자동 생성은 가능. 멀티 플랫폼 동시 업데이트는 API 제한 있음 |
| **기술 난이도** | ★★★☆☆ (디자인 템플릿 엔진 + 배달앱 API 연동은 비공식) |
| **참고 서비스** | **해외: eatOS** — AI 동적 가격 조정 / **국내: 배달앱 자체 메뉴 관리** (통합 솔루션은 부재) |

### 4. 세금/장부/매출 정리

| 항목 | 내용 |
|------|------|
| **설명** | 일일 매출 정리, 카드/현금/배달앱별 정산 확인, 경비 영수증 정리, 부가세/종소세 신고 준비, 인건비 계산 |
| **빈도** | 매일(매출 확인) + 매월(정산) + 분기/연 2회(세금 신고) |
| **시간 소요** | 일 10~20분(매출 확인) + 월 2~4시간(정산) + 세금 신고 기간 5~10시간 |
| **고통 지수** | ★★★★★ (모르면 돈 잃고, 틀리면 가산세. 세무사 비용도 부담) |
| **AI 해결 가능성** | **높음** — 매출 자동 집계, 경비 자동 분류, 세금 시뮬레이션. 이미 성숙한 시장 |
| **기술 난이도** | ★★★★☆ (금융 데이터 연동, 세법 로직, 보안 요구 높음) |
| **참고 서비스** | **캐시노트** — 160만 사업장 사용, 카카오톡 기반 매출/비용 자동 관리 / **쎔(SSEM)** — AI 자동 장부 + 세금 신고 (평점 4.8) / **머니핀** — AI 회계 진단, 자동 계정과목 분류 / **이지샵, 비즈넵, 모바일택스** |

### 5. 직원 관리/스케줄

| 항목 | 내용 |
|------|------|
| **설명** | 알바/직원 근무표 작성, 대타 조정, 출퇴근 기록, 급여 계산, 4대보험 처리, 근로계약서 관리. 2026년부터 전자적 근로시간 관리 의무 확대 |
| **빈도** | 주 1회(스케줄 작성) + 매일(출퇴근 관리) + 월 1회(급여) |
| **시간 소요** | 주 1~3시간(스케줄) + 월 2~4시간(급여/보험) |
| **고통 지수** | ★★★★☆ (알바 구인난 + 갑작스러운 결근 대응 + 노무 법규 복잡) |
| **AI 해결 가능성** | **중~높음** — 스케줄 자동 최적화, 출퇴근 자동 기록, 급여 자동 계산은 가능. 인간관계 조율은 불가 |
| **기술 난이도** | ★★★☆☆ (스케줄 알고리즘 + 노무 법규 DB + 급여 연동) |
| **참고 서비스** | **시프티(Shiftee)** — 근무표 자동 생성, 멀티스케줄 / **오늘근무** — 교대근무 달력, 급여 계산 / **해외: 7Shifts** — 레스토랑 특화 직원 관리 |

### 6. 고객 응대 (예약, 문의)

| 항목 | 내용 |
|------|------|
| **설명** | 전화/카톡/DM으로 오는 예약, 영업시간 문의, 주차 안내, 메뉴 질문, 단체 문의 등. 같은 질문이 하루에도 수십 번 반복 |
| **빈도** | 매일 수시 (피크타임에 집중) |
| **시간 소요** | 하루 30분~2시간 (전화 응대 포함 시 더 많음) |
| **고통 지수** | ★★★★★ (요리/서빙 중에 전화 받아야 함. 같은 질문 반복. 놓치면 매출 손실) |
| **AI 해결 가능성** | **높음** — FAQ 챗봇, 전화 AI 응대, 예약 자동화. 반복 질문의 80%는 자동 처리 가능 |
| **기술 난이도** | ★★★☆☆ (챗봇: 쉬움 / 전화 AI: 중간 / 예약 시스템 연동: 중간) |
| **참고 서비스** | **네이버 예약** — 무료, 노쇼 방지, 결제 연동 / **테이블링** — 대기/예약 관리 토탈 솔루션 / **토스 포스** — 예약+결제 통합 / **해외: OpenTable, Resy** |

### 7. 식자재/재고 관리

## MVP 기능 후보 순위

**평가 기준:** ① 당장 실행 가능성 (기술 난이도 + 개발 기간) ② 사장님 체감 가치 (시간 절약 + 스트레스 해소 + 매출 영향) ③ 경쟁 강도 (기존 서비스 유무)

| 순위 | 기능 | 실행 가능성 | 체감 가치 | 경쟁 강도 | 추천 이유 |
|:----:|------|:----------:|:---------:|:---------:|-----------|
| **1** | **리뷰 답변 AI 도우미** | ★★★★★ | ★★★★★ | 중 (딜리봇 존재, 하지만 배달앱 중심) | LLM만으로 빠르게 MVP 가능. 매일 체감하는 고통 해결. 네이버/카카오/구글맵까지 통합하면 차별화 |
| **2** | **정부 지원사업 맞춤 알림** | ★★★★★ | ★★★★☆ | 낮음 (민간 통합 서비스 거의 없음) | 공공 데이터 + LLM 요약으로 빠르게 구현. 돈 되는 정보라 체감 가치 높음. 블루오션 |
| **3** | **고객 문의 자동응답 (FAQ 챗봇)** | ★★★★☆ | ★★★★★ | 중 (네이버 톡톡 등 있지만 설정 어려움) | "영업시간, 주차, 예약" 같은 반복 질문 자동 처리. 사장님이 직접 느끼는 즉각적 편리함 |
| **4** | **SNS 콘텐츠 자동 생성** | ★★★★☆ | ★★★★☆ | 높음 (Canva, Mirra 등 다수) | 경쟁 많지만, "소상공인 특화" + "한국어 최적화"로 차별화 가능. 메뉴 사진 올리면 인스타 포스팅 완성 |
| **5** | **매출/경비 자동 요약** | ★★★☆☆ | ★★★★★ | 높음 (캐시노트, 쎔 등 성숙 시장) | 체감 가치 최상이지만 경쟁 치열. 단독 MVP보단 부가 기능으로 적합 |
| **6** | **메뉴판/가격표 자동 생성** | ★★★★☆ | ★★★☆☆ | 낮음 | 빈도 낮아 체감 가치 제한적. 하지만 기술적으로 쉽고, SNS 콘텐츠 기능과 시너지 |
| **7** | **직원 스케줄 최적화** | ★★★☆☆ | ★★★★☆ | 중 (시프티, 7Shifts) | 직원 있는 매장에만 해당. MVP 초기 타겟을 1인/소규모로 잡으면 후순위 |
| **8** | **식자재 발주 예측** | ★★☆☆☆ | ★★★★☆ | 중 (미리, 박스히어로) | 가치 높지만 POS 연동 등 기술 허들. MVP보단 성장기 기능 |

## 결론 및 추천

### 슈파 MVP 핵심 전략: "사장님이 매일 30분 아끼는 경험"

소상공인이 가장 고통스러워하는 업무는 **리뷰 관리**, **고객 문의 응대**, **세금/장부 정리** 순이다. 이 중 기술적 실행 가능성과 시장 빈틈을 고려하면:

### MVP v1 추천 조합 (2~3개월 내 구현 가능)

**1순위: 리뷰 답변 AI 도우미**
- 왜: 매일 체감 + LLM으로 빠르게 구현 + 배달앱 외 플랫폼(네이버/카카오/구글) 통합이 차별점
- 어떻게: 사장님이 리뷰 스크린샷만 찍으면 → AI가 답변 초안 생성 → 원클릭 복사
- 경쟁 대비: 딜리봇은 배달앱 중심. 슈파는 "모든 리뷰 채널 통합"

**2순위: 정부 지원사업 맞춤 알림**
- 왜: 블루오션 + 공공 데이터 활용 + "돈 되는 정보"라 체감 가치 즉각적
- 어떻게: 업종/매출규모/지역 입력 → 맞춤 지원사업 추천 + 마감 알림
- 경쟁 대비: 정부 포털(중소벤처24)은 UX가 어렵고 능동적 알림 없음

**3순위: FAQ 자동응답 (고객 문의)**
- 왜: 반복 질문 자동 처리로 즉각적 시간 절약
- 어떻게: 매장 정보 입력 → 카카오톡 채널/네이버 톡톡에 AI 챗봇 자동 설정
- 경쟁 대비: 기존 챗봇은 설정이 어려움. 슈파는 "5분 만에 설정 완료"

### MVP 전략 요약

```
Phase 1 (MVP): 리뷰 AI + 지원사업 알림
  → "시간도 아끼고 돈도 벌게 해주는 앱"으로 포지셔닝
  → 월 무료~9,900원 프리미엄 모델

Phase 2 (성장): + FAQ 챗봇 + SNS 콘텐츠 생성
  → "마케팅까지 해주는 슈퍼파트너"

Phase 3 (확장): + 매출 요약 연동 + 직원 관리 + 식자재 발주
  → "사장님의 모든 잡무를 맡는 AI 비서"
```

### 핵심 차별점 후보

1. **올인원 통합** — 리뷰+마케팅+지원사업을 한 앱에서 (현재는 각각 별도 앱)
2. **소상공인 전용 한국어 AI** — 범용 AI가 아닌 "사장님 말투"로 소통
3. **5분 온보딩** — 사업자등록번호 입력만으로 업종/지역/규모 자동 파악
4. **무료 시작 → 가치 체감 후 결제** — 리뷰 답변 월 30건 무료 같은 프리미엄 모델
