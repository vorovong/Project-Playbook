# 슈파(SUPA) MVP 경쟁사/참고 서비스 상세 분석

> 한국 소상공인 대상 서비스 7개를 핵심 기능, 가격, 강점/약점, 슈파 시사점 중심으로 분석

---

## 1. 딜리봇 (Dilibot) — 배달앱 리뷰 AI 자동 답변

**공식 사이트:** https://review.dilibot.com/

### 핵심 기능
- 배달의민족, 요기요, 쿠팡이츠 리뷰를 **한 곳에서 통합 관리**
- AI가 리뷰의 감정/내용을 분석하여 **맞춤형 답변 자동 생성**
- 새 리뷰 등록 시 **실시간 알림**
- 하루 2시간 걸리던 리뷰 관리를 10분으로 단축 (공식 주장)

### 가격/수익 모델
- 무료 체험 기간 제공 (기간 미공개)
- 이후 월 구독 모델로 추정 (구체적 가격 비공개 — 사이트 가입 후 확인 필요)
- 경쟁 서비스 AI 리뷰넷도 유사한 프리미엄 구독 모델 운영

### 타겟 고객
- 배달앱 운영 음식점 사장님 (1인 운영자 중심)
- 리뷰 관리에 시간을 많이 쓰는 다매장 운영자

### 강점
- 배달 3사(배민/요기요/쿠팡이츠) 통합 관리 — 앱 하나로 끝남
- AI 감정 분석 기반 답변으로 기계적 답변 느낌 감소
- 실시간 동기화로 리뷰 누락 방지

### 약점/사용자 불만
- 가격 정보를 공개하지 않아 진입 장벽 존재
- AI 답변 품질이 천편일률적이라는 소비자 측 인식 (클리앙 등 커뮤니티에서 "AI 답변 티가 난다" 의견)
- 배달앱 외 플랫폼(네이버, 카카오맵 등) 리뷰는 미지원
- 경쟁자 다수 (AI 리뷰넷, 리뷰닥터, 장부대장 등)

### 슈파가 배울 점 / 피할 점
- **배울 점:** "하루 2시간 → 10분" 같은 구체적 시간 절약 메시지가 효과적
- **배울 점:** 멀티 플랫폼 통합은 핵심 가치 — 슈파도 여러 채널을 한 곳에서
- **피할 점:** 가격 비공개는 소상공인에게 불신을 줌. 투명한 가격 공개 필수
- **피할 점:** AI 답변이 "뻔하다"는 인식 → 개인화/톤 커스터마이징 중요

## 2. 캐시노트 (CashNote) — 매출/비용 자동 관리

**공식 사이트:** https://cashnote.kr/
**운영사:** 한국신용데이터(KCD)

### 핵심 기능
- 카드 매출/입금 **자동 수집 및 대시보드**
- 매출 추이, 요일별/시간대별 분석
- 상권 비교 분석
- 고객 방문 분석
- 미수금 진단
- 금융상품(대출 등) 추천

### 가격/수익 모델
- **기본 서비스: 완전 무료** — 이것이 200만 사업장 확보의 핵심
- **캐시노트 플러스 멤버십: 월 16,500원** (정가 33,000원, 오픈 할인 적용 중)
  - 경영 상세 분석 6종, 마켓 할인 쿠폰, 전용 콘텐츠, 법무 상담 등 14개 서비스
- **수익 3축:**
  1. 금융상품 중개 수수료 (대출 추천 → 금융기관 수수료)
  2. 데이터 사업 (522조원 규모 거래 데이터 기반)
  3. 유료 멤버십 구독

### 타겟 고객
- 전 업종 소상공인 (특히 카드 결제 비중 높은 오프라인 매장)
- 200만+ 사업장 이용 중 (2025년 10월 기준)

### 강점
- **무료 → 대규모 사용자 확보 → 데이터 축적 → 수익화** 플라이휠 완성
- 2025년 12월 **월간 영업이익 흑자** 달성 (설립 9년 8개월 만)
- 사장님들의 "국민앱" 포지셔닝
- 카드사/은행 등과의 제휴 네트워크

### 약점/사용자 불만
- 무료 서비스만으로는 깊이 있는 분석 부족
- 플러스 멤버십 전환율이 과제 (200만 중 유료 전환 비율 미공개)
- 매출 분석은 있지만 **"그래서 뭘 해야 하는지"** 실행 가이드 부족
- 배달앱 데이터 연동 한계

### 슈파가 배울 점 / 피할 점
- **배울 점(핵심):** 무료로 시작 → 사용자 확보 → 데이터 기반 수익화 모델
- **배울 점:** "디지털 점장"이라는 포지셔닝 — 도구가 아닌 역할로 브랜딩
- **배울 점:** 매출 2배 성장의 비결은 구독 모델 도입 타이밍
- **피할 점:** 데이터 보여주기만 하고 액션 연결이 약한 점 → 슈파는 인사이트+실행까지
- **참고:** 캐시노트와 겹치지 않는 영역(리뷰, SNS, 마케팅)에서 차별화 가능

## 3. 쎔 (SSEM) — AI 자동 장부 + 세금 신고

**공식 사이트:** https://ssem.kr/

### 핵심 기능
- 매입/매출 자료 **자동 수집** (카드사, 홈택스, 온라인 쇼핑몰)
- 부가가치세, 종합소득세 **알고리즘 자동 계산**
- 업종/연령별 **절세 혜택 자동 적용**
- 온라인 쇼핑몰(쿠팡, 11번가, G마켓 등) 매출 자동 수집
- "3분이면 세금 신고 완료" (공식 주장)
- 안심신고 서비스: 계산 오류 시 전문가 배정

### 가격/수익 모델
- **세금 계산 자체는 무료** — 부가세액 계산 무료 확인 가능
- **신고 건당 33,000원** (부가세 신고 기준)
- **안심신고 플러스: 1,000원~** (오류 보상 보험 성격)
- 세무사 비용(월 10~20만원) 대비 압도적 가격 경쟁력

### 타겟 고객
- 개인사업자 (특히 세무사 비용이 부담되는 영세 사업자)
- 누적 100만 명 가입
- 카카오뱅크, 현대카드 등 금융사 제휴

### 강점
- **건당 과금 모델** — 월 구독 부담 없이 필요할 때만 결제
- 세무사 대비 1/3~1/6 가격
- "3분 완료"라는 극단적 시간 절약 메시지
- 세법 개정 시 자동 반영

### 약점/사용자 불만
- 복잡한 세무 상황(다중 사업장, 법인 등)에서는 한계
- AI 계산 오류 가능성에 대한 불안감
- 세금 신고 시즌(1월, 5월, 7월)에만 사용 → 일상적 접점 부족
- 장부 관리 기능은 캐시노트 대비 약함

### 슈파가 배울 점 / 피할 점
- **배울 점:** 건당 과금은 소상공인 심리적 부담을 줄임
- **배울 점:** 전문가 서비스(세무사)를 대체하는 가격 포지셔닝이 명확
- **배울 점:** "안심신고"처럼 AI 결과에 대한 보증 장치 → 신뢰 확보
- **피할 점:** 시즌성 서비스는 일상적 리텐션이 약함 → 슈파는 매일 쓰는 도구여야

## 4. 스마트포스트 (SmartPost) — 멀티 SNS 예약 포스팅

**공식 사이트:** https://www.smartpost.kr/

### 핵심 기능
- **멀티채널 예약 포스팅**: 인스타그램, 페이스북, 카카오스토리, 네이버 밴드, 네이버 블로그, 티스토리, 다음 블로그, 구글 블로그
- 웹 이미지 편집기 내장
- 콘텐츠 기획/작성/검수 협업 기능
- 한국 SNS 특화 (카카오스토리, 네이버 밴드 등 지원)

### 가격/수익 모델
- **채널당 1개 계정 무료** 제공
- 유료 이용권 존재 (구체적 가격 비공개 — 사이트 내 확인 필요)
- SaaS 구독 모델

### 타겟 고객
- 소상공인, 스타트업, 1인 마케터
- 적은 예산으로 SNS 마케팅을 직접 하려는 사업자

### 강점
- **한국 SNS 생태계 특화** — 해외 툴(Buffer, Hootsuite)이 지원 못하는 네이버 밴드, 카카오스토리 지원
- 무료 시작 가능
- 포스팅부터 협업까지 올인원

### 약점/사용자 불만
- **인지도 매우 낮음** — 소상공인 사이에서 거의 알려지지 않음
- 소상공인은 SNS 예약 포스팅보다 **"무슨 콘텐츠를 올려야 하는지"**가 더 큰 고민
- 도구만 제공하고 콘텐츠 전략/아이디어는 부재
- 업데이트 빈도, 사용자 커뮤니티 활성도 불명확
- 가격 정보 불투명

### 슈파가 배울 점 / 피할 점
- **배울 점:** 한국 SNS 특화는 필수 — 네이버 블로그, 카카오 채널 등 지원해야
- **핵심 교훈:** 소상공인에게 "포스팅 도구"만으로는 부족. **"무엇을 올릴지"**(콘텐츠 생성)가 핵심 가치
- **피할 점:** 스마트포스트의 낮은 인지도는 소상공인 눈높이 마케팅 부재 때문
- **슈파 기회:** AI로 콘텐츠 아이디어 생성 + 자동 포스팅을 결합하면 스마트포스트를 완전히 대체 가능

## 5. 테이블링 (Tabling) — 대기/예약 관리

**공식 사이트:** https://www.tabling.co.kr/ (고객용) / https://b2b.tabling.co.kr/ (사업자용)

### 핵심 기능
- **원격 줄서기**: 고객이 매장 방문 없이 앱으로 대기 등록
- **예약 관리**: 날짜/시간/인원별 예약
- **실시간 대기 현황** 안내
- **모바일 메뉴 주문/결제** 기능
- 2016년 서비스 시작 — 한국 웨이팅 앱의 선구자

### 가격/수익 모델
- **초기:** 기기 설치비(44만원)만 내면 이용료 무료 → 이 전략으로 빠르게 확산
- **현재:** 월 99,000원 구독료로 전환 (유료화)
- MAU 115만 명 (2024년 12월 기준, 전년 대비 3배 성장)

### 타겟 고객
- 웨이팅이 발생하는 인기 음식점/카페
- 예약 관리가 필요한 레스토랑

### 강점
- 한국 웨이팅 앱 시장 1위 포지션
- 소비자 앱 MAU 115만 → 양면 네트워크 효과
- "줄 안 서도 되는" 고객 경험 혁신

### 약점/사용자 불만
- **유료화 전환에 대한 강한 반발** — "무료로 시작해서 의존하게 만든 뒤 유료 전환"
- 월 99,000원이 영세 음식점에게는 부담
- "울며 겨자 먹기로 이용" (아주경제 보도)
- 경쟁사 캐치테이블과의 경쟁 심화
- 웨이팅 없는 매장에는 무용지물 — 범용성 부족

### 슈파가 배울 점 / 피할 점
- **배울 점:** "무료 → 의존 → 유료 전환"은 효과적이지만 타이밍과 방법이 중요
- **배울 점:** 양면 네트워크(소비자+사장님) 효과는 강력한 해자(moat)
- **피할 점(중요):** 갑작스러운 유료화는 사장님 신뢰를 깨뜨림. 가격 로드맵을 처음부터 투명하게
- **피할 점:** 한 가지 기능(웨이팅)에만 특화하면 범용성 부족 → 슈파는 올인원 방향이 맞음

## 6. 네이버 스마트플레이스 — 무료 매장 관리 도구

**공식 사이트:** https://new-m.pay.naver.com/smartplace/

### 핵심 기능
- **매장 정보 등록/관리**: 운영시간, 메뉴, 사진, 편의시설 등
- **예약/주문 관리**: 네이버 예약 시스템 통합
- **리뷰 관리**: 방문자 리뷰 확인/답글
- **통계 대시보드**: 검색 노출, 방문자 통계
- **네이버웍스 연동**: 실시간 알림, 직원 소통
- **네이버 톡톡**: 고객 채팅 상담

### 가격/수익 모델
- **모든 기능 완전 무료**
- 네이버는 광고(네이버 플레이스 광고, 검색 광고)로 수익화
- 스마트플레이스 자체는 사장님을 네이버 생태계에 묶어두는 인프라

### 타겟 고객
- 사실상 **모든 오프라인 사업장** (음식점, 카페, 미용실, 병원 등)
- 한국에서 사업하면 거의 필수적으로 등록

### 사장님들이 실제로 쓰는 것 vs 안 쓰는 것
| 실제로 쓰는 기능 | 잘 안 쓰는 기능 |
|---|---|
| 매장 정보 등록 (필수) | 통계 대시보드 (보는 사람 소수) |
| 리뷰 확인 | 네이버 톡톡 (관리 부담) |
| 사진 등록 | 네이버웍스 연동 (존재 자체를 모름) |
| 네이버 예약 (일부 업종) | 상세 분석 도구 |

### 강점
- **완전 무료 + 네이버 검색 노출** — 이 조합을 이길 수 없음
- 한국 소비자의 검색 행동과 직결 (네이버 지도 → 플레이스)
- 별도 앱 설치 없이 네이버 안에서 완결

### 약점/사용자 불만
- **등록 후 광고 전화 폭탄** — 네이버 직원 사칭 사기 만연
- 등록했다고 상위 노출되는 것이 아님 → 광고 집행 없이는 묻힘
- 관리 UI가 직관적이지 않음 (초보 사장님 혼란)
- 등록 후 방치하는 경우 많음 → "효과 없다"는 인식
- 리뷰 관리 도구가 단순함 (AI 분석/자동 답변 없음)

### 슈파가 배울 점 / 피할 점
- **배울 점:** 네이버 스마트플레이스는 경쟁 대상이 아니라 **연동 대상**
- **핵심 기회:** 스마트플레이스의 약점(리뷰 자동 답변 없음, 통계 활용 어려움)을 슈파가 보완
- **배울 점:** 사장님들은 "등록은 하는데 관리는 안 함" → 자동화가 핵심 니즈
- **피할 점:** 네이버와 직접 경쟁하는 구도는 피해야 함. 보완재 포지셔닝

## 7. 토스 포스 / 토스플레이스 — 결제 + 매장관리 통합

**공식 사이트:** https://tossplace.com/

### 핵심 기능
- **토스 포스 (무료 POS 소프트웨어)**: 메뉴 등록, 주문/결제, 매출 조회
- **토스 프론트/터미널**: 결제 단말기 하드웨어
- **AI 자동 상품 등록**: 메뉴판 촬영 → AI가 자동으로 POS에 등록 (2026년 신규)
- **매출 장부**: 실시간 매출 조회
- **사장님 스토리**: 매장 운영 팁 콘텐츠

### 가격/수익 모델
- **토스 포스 소프트웨어: 완전 무료** (기존 태블릿/PC에 설치)
- **토스 프론트2 단말기**: 약 20~30만원대 (하드웨어 비용)
- **수익원**: 카드 결제 수수료 (VAN 수수료 수준)
- 소프트웨어 무료 → 단말기 + 결제 수수료로 수익화

### 타겟 고객 및 성장
- 2030 젊은 사장님 중심 (가맹점주 44%가 20~30대)
- **2025년 누적 가맹점 16만 곳** (2024년 대비 4.5배 성장)
- 음식점 48%, 카페 20%, 뷰티숍 19%

### 강점
- **토스 브랜드 파워** — 2030 세대 신뢰도 높음
- POS 소프트웨어 무료 → 초기 비용 0원
- UX가 극도로 직관적 — 알바생도 교육 없이 사용
- 결제 데이터 기반 매장 분석으로 확장 가능
- AI 기능 적극 도입 (메뉴 자동 등록 등)

### 약점/사용자 불만
- 아직 전통적 POS(포스뱅크 등) 대비 기능이 부족한 영역 존재
- 기존 VAN사/POS 업체와의 계약 해지 필요
- 결제 이외 매장 운영(직원관리, 재고 등) 기능은 아직 초기 단계
- 대형 프랜차이즈보다 개인 매장 중심

### 슈파와 겹치는 부분
| 영역 | 토스플레이스 | 슈파(예상) |
|---|---|---|
| 매출 분석 | O (결제 데이터 기반) | O (캐시노트 연동 가능) |
| 리뷰 관리 | X | O (핵심 기능) |
| SNS/마케팅 | X (사장님 스토리 콘텐츠만) | O (핵심 기능) |
| 결제 | O (핵심) | X |
| AI 활용 | O (메뉴 등록) | O (콘텐츠 생성, 리뷰 답변) |

### 슈파가 배울 점 / 피할 점
- **배울 점:** "무료 소프트웨어"로 진입 → 생태계 확장 전략은 교과서적
- **배울 점:** 2030 사장님을 타겟한 UX/브랜딩 — 슈파도 같은 세대 공략 가능
- **배울 점:** AI 기능을 실용적으로 도입 (메뉴 자동 등록처럼 즉각 체감 가능한 것)
- **피할 점:** 결제 영역은 토스플레이스와 경쟁하지 말 것 — 보완 관계가 이상적
- **핵심 포지셔닝:** 토스플레이스가 "결제+매출"이라면, 슈파는 "마케팅+고객관리"

## 슈파 MVP를 위한 핵심 시사점

### 1. 포지셔닝 전략
- 캐시노트(매출), 토스플레이스(결제)가 커버하지 못하는 **"마케팅+고객소통" 영역**이 블루오션
- 네이버 스마트플레이스는 경쟁이 아닌 **연동 대상**
- 딜리봇/스마트포스트의 기능을 AI로 통합하면 차별화 가능

### 2. 가격 전략
- 캐시노트/토스플레이스의 성공 공식: **무료로 시작 → 사용자 확보 → 프리미엄 전환**
- 쎔의 건당 과금도 소상공인 친화적 모델
- 가격은 반드시 **투명하게 공개** (딜리봇의 실패 요인)
- 테이블링의 갑작스러운 유료화 반발을 교훈으로

### 3. 핵심 차별화 포인트
- 스마트포스트는 "도구만 줌" → 슈파는 **"무엇을 할지까지 AI가 알려줌"**
- 딜리봇은 "배달앱만" → 슈파는 **"전 채널 통합"**
- 캐시노트는 "데이터만 보여줌" → 슈파는 **"데이터 → 액션까지"**

### 4. 반드시 피해야 할 실수
- 가격 비공개 (딜리봇)
- 갑작스러운 유료화 (테이블링)
- 도구만 제공하고 콘텐츠/전략은 부재 (스마트포스트)
- 네이버/토스와 직접 경쟁하는 구도

### 5. 연동/파트너십 우선순위
1. **네이버 스마트플레이스** — 리뷰/예약 데이터 연동 (최우선)
2. **캐시노트** — 매출 데이터 기반 마케팅 인사이트
3. **토스플레이스** — 결제 데이터 연동
4. **배달 3사** — 리뷰 통합 관리
