## 2. 정부 지원사업 알림 — 기술 조사

**결론: 기술적으로 당장 만들 수 있고, 블루오션이다.**

| 항목 | 핵심 |
|------|------|
| **데이터** | 기업마당 + 중소벤처24 API로 공고 수집 가능 (일일 1,000~10만건) |
| **매칭** | MVP는 규칙 기반(업종/지역/매출) → 고도화 시 LLM 구조화 |
| **LLM 비용** | 공고 1건 요약 2~5원, 월 3,000~7,500원 |
| **알림** | 카카오 알림톡 최적 (열람율 90%+, 건당 8~13원) |
| **월 운영비** | 약 15~18만원 (서버+LLM+알림톡+DB) |
| **경쟁** | 정부 포털은 UX 불편, 민간 통합 서비스 거의 없음 |

## 3. 경쟁사 상세 분석

### 핵심 매트릭스

| 서비스 | 핵심 영역 | 가격 | 사용자 수 | 슈파 관계 |
|---|---|---|---|---|
| **딜리봇** | 배달 리뷰 AI 답변 | 유료(미공개) | 미공개 | 기능 흡수 가능 |
| **캐시노트** | 매출/비용 관리 | 무료+월16,500원 | **200만+** | 연동 파트너 |
| **쎔(SSEM)** | 세금 신고 | 건당 33,000원 | 100만+ | 간접 경쟁 |
| **스마트포스트** | SNS 예약 포스팅 | 무료+유료 | 매우 적음 | 직접 경쟁/대체 |
| **테이블링** | 대기/예약 관리 | 월 99,000원 | MAU 115만 | 비경쟁 |
| **네이버 스마트플레이스** | 매장 정보/검색 | 무료 | 사실상 전체 | **연동 필수** |
| **토스플레이스** | 결제+POS | 무료(SW) | 16만 가맹점 | 보완 관계 |

### 슈파 포지셔닝

```
캐시노트 = 매출/데이터    → "숫자를 보여준다"
토스플레이스 = 결제/POS    → "돈을 받는다"
슈파 = 마케팅+고객소통     → "고객을 만든다" ← 여기가 비어있다
```

### 반드시 피해야 할 실수
- 가격 비공개 (딜리봇)
- 갑작스러운 유료화 (테이블링)
- 도구만 제공, 콘텐츠/전략 부재 (스마트포스트)
- 네이버/토스와 직접 경쟁하는 구도

## 4. 소상공인 실제 니즈/고충

### 핵심 숫자

| 지표 | 수치 |
|------|------|
| 2024년 연간 폐업자 | **100만 8,282명** (역사상 최초 100만 돌파) |
| 3년 생존율 | 52.3% |
| 디지털 도구 활용 소상공인 | **15.4%** |
| IT 서비스 월 지불 의향 (대다수) | **무료~1만원** |
| 배달앱 수수료 | 매출의 **16~29%** |

### 가장 아픈 곳 순위

1. 내수 부진 → 매출 감소 (77.4%)
2. 고정비 부담 — 이자(48.7%), 인건비(38.1%), 재료비(36.7%)
3. 배달앱 수수료 (매출의 16~29%)
4. 세무/장부 스트레스
5. 마케팅/홍보를 혼자 다 해야 함 (17.1%)
6. 정부 지원 정보 접근 어려움

### 도구 설계 필수 조건

1. **월 1만원 이하 or 무료 시작** — 70%+가 무료 아니면 안 씀
2. **3분 안에 쓸 수 있어야 함** — 배울 시간 없음
3. **즉각적 가치 체감** — "이만큼 절약됐다" 명확히
4. **모바일 우선** — PC 앞에 앉을 시간 없음
5. **50대도 쓸 수 있는 UX** — 큰 글씨, 단순 흐름
6. **"돈을 아끼게 해주는 도구" > "돈을 벌게 해주는 도구"** — 절약 프레이밍이 더 설득력

## 5. 추가 기능 후보 (기존 8개 외)

### Phase 2 유력 후보 (5개)

| 기능 | 체감 가치 | 이유 |
|------|:---:|------|
| **AI 전화 응대** | 최상 | 1인 매장의 1위 고통. "부재중 AI 콜백"으로 시작 가능. 경쟁 막 시작 |
| **구인/채용 AI** | 높음 | "사람 못 구한다" 만년 고충. AI 공고 작성 + 근로계약서 자동 생성 |
| **법률/노무 AI 상담** | 높음 | 상담 1건 10~30만원 절감. 소상공인 특화 서비스 부재 |
| **계약/문서 관리** | 중~높음 | "계약서 사진 → AI 요약 + 만료일 등록" 한 기능으로 강력 |
| **상호 홍보/크로스 마케팅** | 중 | 기술 쉬움 + 네트워크 효과로 슈파 확산 기여 |

### 기존에 완전히 빠져있던 신규 영역

- **법률/행정** (임대차, 노무, 계약서) — 소상공인 만성 고민
- **사장님 개인 생활** (건강, 워라밸, 은퇴) — 감성적 차별화
- **B2B 협업** (공동구매, 크로스 마케팅, 업체 매칭) — 플랫폼 전략 핵심

### 빼야 할 것
- AI CCTV, 매장 음악, 공동 배달, 금융 직접 진출

## 6. 해외 올인원 도구 성공 사례

### 성공한 Hook(진입점)의 공통점

| 조건 | 설명 | 사례 |
|---|---|---|
| **매일 사용** | 일회성 아닌 매일 열어보는 도구 | Square(결제), Homebase(스케줄) |
| **즉시 가치 체감** | 써보자마자 "좋다" | Durable(30초 웹사이트) |
| **무료/극저비용** | 지갑 저항 제거 | Homebase(무료), Wix(무료) |
| **기존 도구 교체 불필요** | 위에 얹을 수 있음 | Womply(기존 POS 위에) |
| **데이터가 쌓임** | 다음 기능의 연료 | Square(결제→대출) |

### 올인원 확장 패턴 (모든 성공 사례 공통)

```
[1단계] Hook 기능 (무료/저비용, 매일 사용)
    ↓ 데이터 축적
[2단계] 데이터 활용 (분석, 인사이트)
    ↓ 신뢰 확보
[3단계] 돈 관련 기능 (결제, 급여, 대출)
    ↓ 금전적 락인
[4단계] 마케팅/성장 (고객 확보, 리뷰, 광고)
    ↓ 성장 의존
[5단계] AI 자동화 (위 모든 기능을 AI가 자동 운영)
```

### 핵심 교훈 6가지

1. **매일 사용하는 기능이 hook** — 결제(Square), 스케줄(Homebase), POS(Toast)
2. **"기존 도구 위에 얹는" 전략이 초기에 유리** — "다 바꾸세요"는 안 통함
3. **자동화가 핵심 가치** — "해주는 도구"보다 "알아서 해주는 도구"
4. **데이터 플라이휠 설계 필수** — hook → 데이터 → 다음 기능의 연료
5. **AI 전화 응대, 리뷰 자동 관리, 마케팅 카피 자동 생성** — 2024~2026 핵심 트렌드
6. **한국은 결제/POS가 이미 포화** — 한국 고유 pain point에서 hook을 찾아야 함

## 종합 전략 제안

### 슈파 MVP 최종 추천 조합

**Phase 1 (MVP):**
| 기능 | 역할 |
|------|------|
| **리뷰 답변 AI** (네이버 플레이스) | Hook — 매일 체감, 고통 직결 |
| **정부 지원사업 맞춤 알림** | 차별화 — 블루오션, "돈 되는 정보" |
| **SNS 콘텐츠 생성** | 보조 — "뭘 올릴지" 해결, AI로 빠르게 구현 |

**Phase 2 (성장):**
- FAQ 자동응답 (고객 문의)
- AI 전화 응대 (부재중 콜백부터)
- 구인/채용 AI
- 법률/노무 AI 상담

**Phase 3 (플랫폼):**
- 다른 플랫폼 리뷰 확장 (배민, 쿠팡이츠)
- 매출 데이터 연동 (캐시노트 연동)
- B2B 협업 (사장님 네트워크)
- 식자재 발주 예측

### 가격 전략

```
무료: 리뷰 답변 월 30건 + 지원사업 알림 + SNS 콘텐츠 월 10건
프리미엄 (월 9,900원): 무제한 + 자동 모드 + 우선 알림
```

### 핵심 차별화

1. **올인원 통합** — 리뷰+마케팅+지원사업을 한 앱에서 (현재 각각 별도)
2. **소상공인 전용 AI** — 범용 AI가 아닌 "사장님 언어"로 소통
3. **5분 온보딩** — 사업자등록번호 입력만으로 업종/지역/규모 자동 파악
4. **무료 시작 → 가치 체감 후 결제**
