# 한국 소상공인(자영업자) 실제 니즈와 고충 리서치

> 설문조사, 커뮤니티, 뉴스 기반 종합 정리 (2025~2026년 기준)

---

## 1. 최근 설문/통계 — 소상공인이 가장 힘들어하는 것

### 소상공인연합회 2026년 신년 경영 실태조사

- 대상: 전국 소상공인 1,073명 (음식·숙박, 도·소매, 개인서비스업)
- 조사 기간: 2025.12.30 ~ 2026.1.6

**경영 부진의 원인 (복수응답)**

| 순위 | 항목 | 비율 |
|------|------|------|
| 1 | 내수부진(경기 침체·고물가) → 소비 감소 | 77.4% |
| 2 | 금리 인상·부채 증가 → 금융비용 부담 | 33.4% |
| 3 | 원부자재·재료비 상승 | 28.3% |
| 4 | 인건비 부담·인력 확보 어려움 | 26.4% |

**2026년 가장 부담이 큰 비용**

| 순위 | 항목 | 비율 |
|------|------|------|
| 1 | 금융비용(이자) | 48.7% |
| 2 | 인건비 | 38.1% |
| 3 | 원부자재비 | 36.7% |
| 4 | 임대료 | 33.5% |

**경영 위협 요인**

- 내수 부진: 77.7%
- 환율 상승 → 소비자물가 상승: 36.7%
- 최저임금 인상: 31.9%

**원하는 정책 지원**

- 금융 지원: 71.9%
- 세제 지원: 39.0%
- 마케팅·판로 지원: 22.9%
- 업종별 과잉·중복 규제 개선: 19.0%

### 폐업 현황 (2024~2025)

- **2024년 연간 폐업자: 100만 8,282명** (역사상 최초 100만 돌파)
- 3년 생존율: 52.3% (100곳 중 48곳이 3년 내 폐업)
- 1년 내 폐업률: 22%
- 5년 생존율: 40.2%
- 폐업률 높은 업종: 소매업(16%), 음식점업(15%)
- 자영업자 43.6%가 향후 3년 안에 폐업을 고려 중

### 경영상 가장 큰 어려움 (별도 조사)

| 순위 | 항목 | 비율 |
|------|------|------|
| 1 | 인건비·임대료 등 고정비 부담 | 43.4% |
| 2 | 경쟁 심화 | 25.4% |
| 3 | 마케팅·홍보 어려움 | 17.1% |

**폐업 고려 이유**

- 영업실적 지속 악화: 28.2%
- 경기회복 전망 불투명: 18.1%
- 자금 사정 악화·대출 상환 부담: 18.1%
- 임차료·인건비 상승: 11.9%
- 원재료 가격 상승: 11.9%

## 2. 커뮤니티 고충 — 자영업자들이 매일 토로하는 것

### 주요 커뮤니티

- **아프니까 사장이다** (네이버 카페): 회원 168만명, 하루 600개+ 게시글
- **자영업자 갤러리** (디시인사이드)
- **맘카페** (동네 상권 관련)

### 반복되는 고충 패턴

**1) 배달앱 수수료 지옥**
- 점주 95%가 수수료 부담 호소
- 매출의 16~29%가 배달앱 수수료로 빠져나감 (서울시 조사)
- 주문 1만5천원 기준, 수수료 약 5,190원 (34.6%)
- 주문 3만원 기준, 수수료 약 6,810원 (22.7%)
- 배달비 + 결제비 + 광고비 + 할인 분담 합치면 체감 부담 더 큼
- 상위 노출 경쟁으로 광고비만 매출의 20.8% 지출하는 매장도 존재
- "매출은 느는데 남는 게 없다"가 대표 하소연

**2) 리뷰·별점 테러**
- 부당한 별점, 악성 리뷰에 대한 대응 수단 부재
- 리뷰 비공개 설정하면 주문 감소 → 어쩔 수 없이 노출
- "손님이 왕이 아니라 리뷰가 왕"이라는 자조

**3) 인건비 부담 + 구인난**
- 최저임금 인상으로 인건비 부담 가중
- 알바 노쇼, 갑자기 안 나오기, 근무 태도 문제
- 사장이 직접 1인 다역 → 번아웃
- "사람 쓰면 남는 게 없고, 안 쓰면 몸이 남지 않는다"

**4) 임대료 vs 매출 역전**
- 매출은 줄어드는데 임대료는 그대로 or 인상
- 권리금 날릴까봐 폐업도 못 하는 상황
- "월세 내고 인건비 내면 내 월급이 마이너스"

**5) 세무·회계의 어려움**
- 부가세, 종합소득세 신고 매번 스트레스
- 엑셀 장부 정리 중 수식 꼬임·누락 빈발
- 세무사 비용 부담 (월 10~15만원)
- 어떤 비용이 경비 처리 되는지 모름
- 복식부기 자체를 어려워하는 초보 사장님 다수

**6) 정부 지원 정보 접근 어려움**
- 지원 사업이 많은데 "뭐가 있는지 모른다"
- 신청 서류·절차가 복잡해서 포기
- 마감 기한을 놓치는 경우 빈번

## 3. 디지털 전환 현황 — 사용률과 장벽

### 핵심 수치 (소상공인시장진흥공단·중소기업연구원)

| 지표 | 수치 |
|------|------|
| 디지털 기술 활용 소상공인 | **15.4%** |
| 디지털 전환 필요성 인식 | 29.7% |
| 정부지원 필요성 높게 인식 | 20.5% |
| 디지털 기술 수용성 | 5점 만점 2.98점 |

### 소상공인 디지털 수준 분류

| 단계 | 비율 | 설명 |
|------|------|------|
| 전통 방식 고수 | ~70% | 디지털 인식 자체가 부족 |
| 전환 인식 있으나 부족 | ~25% | 필요성은 알지만 실행 못함 |
| 디지털 전환 완료/가능 | ~5% | 스스로 디지털화 가능 |

### 디지털 전환이 필요한 분야

| 분야 | 비율 |
|------|------|
| 사업체 운영관리 | 34.9% |
| 제품/서비스 판매 | 33.4% |
| 고객관리 | 15.4% |
| 원재료 주문·납품 | 9.0% |

### 디지털 전환 준비 수준 (5점 만점)

| 항목 | 점수 |
|------|------|
| 인프라 (장비 등) | 3.12 |
| 지식 | 2.53 |
| 인력 | 2.38 |
| 자금 | 2.21 |

### 소상공인이 실제 사용하는 디지털 도구

- **배달앱**: 배달의민족, 요기요, 쿠팡이츠 (음식점 필수)
- **POS/키오스크**: 카드 결제·주문 처리
- **네이버 스마트플레이스**: 매장 등록·리뷰 관리
- **카카오톡 채널**: 고객 소통·예약
- **인스타그램**: 매장 홍보 (특히 젊은 사장님)
- **엑셀/수기장부**: 매출·비용 정리 (대다수)
- **세무 프로그램**: 삼쩜삼, 홈택스 등 (일부)

### 안 쓰는 이유

1. **배울 시간이 없다** — 하루 12시간+ 영업, 학습할 여유 없음
2. **비용 부담** — 월 몇 만원도 아까움
3. **뭘 써야 하는지 모른다** — 선택지가 많은데 비교할 능력/시간 없음
4. **나이대가 있어서 어렵다** — 디지털 리터러시 자체가 낮은 경우
5. **투자 대비 효과 불확실** — "돈 써봤자 달라지는 게 뭔지 모르겠다"

## 4. 연령대별 차이 — 2030 사장님 vs 4050 사장님

### 디지털 활용 패턴

| 항목 | 2030 사장님 | 4050 사장님 |
|------|-------------|-------------|
| 주요 SNS | 인스타그램, 틱톡 | 네이버 밴드, 카카오스토리 |
| 마케팅 채널 | 인스타 릴스, 숏폼 영상 | 블로그 체험단, 네이버 플레이스 |
| AI 도구 | 뤼튼, ChatGPT (적극 활용) | 에이닷, 네이버 클로바 (제한적) |
| 정보 습득 | 유튜브, 인스타 | 네이버 카페, 지인 추천 |
| 배달앱 대응 | 자체 앱/SNS 주문 시도 | 배달앱 의존도 높음 |
| 디지털 도구 태도 | "쓸 만한 거 없나?" 적극 탐색 | "어려워서 못 쓰겠다" 회피 |

### 코로나 이후 디지털 소비 증가율 (2019→2021)

| 연령대 | 온라인 이용 증가율 |
|--------|-------------------|
| 20대 | +45% |
| 30대 | +63% |
| 40대 | +84% |
| 50대 | +110% |

- 주목할 점: 40~50대의 **증가폭**이 더 큼 (낮은 베이스에서 출발)
- 배달앱 이용: 20대 비중 41%→30% 감소, 40대 15%→24% 증가
- 즉, 4050도 디지털에 적응 중이나 **속도와 깊이에 차이**

### 서울시 중장년 조사 결과 (40~64세 4,400명)

- 40대: 직업 역량·일자리 업그레이드 위한 교육·훈련 수요 높음
- 50대: 변화하는 사회에 적응하기 위한 **디지털 격차 해소**가 주요 과제

## 5. 코로나 이후 변화 — 배달/온라인 의존도와 달라진 업무 패턴

### 배달 시장 폭발적 성장

- 음식 배달 거래액: 20조원 돌파 (전년 대비 43.5% 증가)
- 코로나 이후 비대면 소비 패러다임으로 빠르게 전환

### 매출은 늘었지만 이익은?

- 배달서비스 도입 → **매출 증가** BUT **영업비용도 동반 증가**
- 영업이익·영업이익률: 유의미한 변화 없거나 **오히려 감소**
- 배달 수수료·광고료가 영세 외식업체에 과도한 부담

### 배달 도입이 높은 업체 특성

- 수도권 위치
- 프랜차이즈
- 외국식 음식점
- 여성 고객 비중 높음
- 테이블 수 많은 매장

### 달라진 업무 패턴

**코로나 전:**
- 매장 운영 = 조리 + 서빙 + 계산
- 마케팅 = 전단지, 현수막, 입소문

**코로나 후:**
- 매장 운영 = 조리 + 서빙 + 계산 + **배달 포장 + 배달앱 관리 + 리뷰 대응**
- 마케팅 = 배달앱 광고 + SNS + 네이버 플레이스 + 리뷰 관리
- 추가 업무: 배달앱 3~4개 동시 관리, 메뉴 사진 촬영, 이벤트 기획
- **업무량 2~3배 증가, 사장의 멀티태스킹 요구 급증**

### 핵심 딜레마

> "배달 안 하면 매출이 줄고, 하면 남는 게 없다"

## 6. 소상공인이 IT 도구에 쓰는 돈 — 지불 의향

### 현재 지출 현황

직접적인 대규모 설문은 부족하나, 간접 데이터를 종합하면:

| 항목 | 월 비용 |
|------|---------|
| 배달앱 수수료 (사실상 필수) | 매출의 16~29% |
| POS 시스템 | 3~10만원 |
| 세무사 기장료 | 10~15만원 |
| 네이버 스마트플레이스 광고 | 5~30만원 |
| 배달앱 광고비 | 10~50만원+ |
| 카카오톡 채널 (유료 기능) | 1~5만원 |

### IT 서비스 지불 의향 추정

- 한국 소비자 평균 디지털 구독비: 월 약 3~6만원 (3.4개 서비스)
- 소상공인은 **비용 민감도가 극히 높음** — 경영 여건 악화로 추가 지출 여력 적음
- 디지털 전환 준비 중 **자금 항목이 5점 만점 2.21점**으로 가장 낮음

### 업계 분석 종합

**소상공인의 IT 서비스 지불 의향 구간 추정:**

| 구간 | 월 비용 | 대상 | 조건 |
|------|---------|------|------|
| 무료~1만원 | 0~9,900원 | 대다수 (70%+) | "공짜가 아니면 안 쓴다" / 무료 체험 후 전환 |
| 1~3만원 | 1~2.9만원 | 일부 (20%) | 즉각적 효과 체감 필수 (매출 증가, 시간 절약) |
| 3~5만원 | 3~4.9만원 | 소수 (5~8%) | 여러 기능 통합 솔루션에 한해 |
| 5만원 이상 | 5만원+ | 극소수 (2~3%) | 매출 규모 큰 사업장, 확실한 ROI |

**핵심 인사이트:**
- 월 **1~2만원**이 대다수 소상공인의 심리적 허들
- **무료 → 유료 전환**이 가장 효과적 (프리미엄 모델)
- "돈을 아끼게 해주는 도구"가 "돈을 벌게 해주는 도구"보다 설득력 높음
- 기존에 쓰는 돈(세무사비, 배달앱 수수료)을 **대체하거나 줄여주는** 프레이밍 필요

## 7. 종합 인사이트 — 소상공인 도구를 만든다면

### 가장 아픈 곳 (Pain Point 순위)

1. **내수 부진으로 매출 자체가 감소** — 도구로 해결 어려움, 매출 증대 지원이 핵심
2. **고정비(인건비·임대료·이자) 부담** — 비용 절감 도구에 대한 수요
3. **배달앱 수수료 과다** — 수수료 최적화, 자체 주문 채널 니즈
4. **세무·회계·장부 관리 스트레스** — 자동화 도구 수요 높음
5. **마케팅·홍보를 혼자 다 해야 함** — SNS 관리, 리뷰 대응 도구
6. **정부 지원 정보 접근 어려움** — 맞춤형 지원 사업 알림

### 도구 설계 시 고려할 점

1. **가격**: 월 1만원 이하 or 무료 시작 필수
2. **학습 비용 최소화**: 배울 시간 없는 사장님, 3분 안에 쓸 수 있어야 함
3. **즉각적 가치 체감**: "이걸 쓰니까 이만큼 절약됐다/벌었다" 명확히
4. **모바일 우선**: PC 앞에 앉을 시간 없음, 스마트폰으로 다 해야 함
5. **연령대 고려**: 50대도 쓸 수 있는 UX (큰 글씨, 단순 흐름)
6. **기존 도구와 연동**: 배달앱, POS, 홈택스 데이터 활용
