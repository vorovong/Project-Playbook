# 컨설팅 보고서 작성 규칙

서울신용보증재단 경영개선 컨설팅 보고서 양식에 맞춘 작성 기준.

---

## 기본 톤

- **실용적, 구체적** — 추상적 표현 금지 ("혁신적인 마케팅 전략" → "네이버 플레이스 키워드 최적화")
- **현장 언어** — 소상공인이 읽어도 이해할 수 있는 말
- **근거 기반** — 숫자/데이터가 있으면 반드시 포함. 없으면 추측하지 않는다

---

## 항목별 작성 기준

### 1. 사전조사/현황분석

**목적:** 이 업체가 지금 어떤 상태인지 객관적으로 보여주기

작성 포인트:
- 매출분석 시스템(POS 등) 현황
- 해당 업종 특성 및 최신 트렌드
- POS 자료 기반 매출 분석 (있으면)
- 경영환경분석 (내부환경 + 외부환경)
- 판매채널 현황 (온라인/오프라인)
- 상권 및 인구통계 특성

**주의:**
- 전사본에서 나온 팩트만 쓴다. 추측으로 채우지 않는다
- "-" 불릿으로 나열. 한 항목 1~2줄

### 2. 컨설팅목적/추진과제

**목적:** 이 컨설팅에서 뭘 해결하려는지

작성 포인트:
- 매출 활성화 방향 (온라인/오프라인)
- 고객 유입/유지 전략
- 메뉴/상품 개선
- 브랜딩/마케팅 방향

**주의:**
- 3~5개 과제로 정리. 너무 많으면 핵심만
- 각 과제가 사전조사에서 발견한 문제와 연결되어야 함

### 3. 수행내역

**목적:** 실제로 뭘 했는지

작성 포인트:
- 벤치마킹 (어디를, 왜)
- 키워드/SNS 분석
- 시스템 활용 (네이버, 배민 등)
- 교육/컨설팅 내용

**주의:**
- 구체적으로. "마케팅 전략 수립" → "인스타그램 릴스 콘텐츠 3개 제작 + 해시태그 전략 수립"
- 실제로 한 것만 쓴다

### 4. 컨설팅결과/(기대)효과

**목적:** 이 컨설팅으로 뭐가 달라졌는지/달라질 건지

작성 포인트:
- 완료된 것 (시스템 구축, 최적화 등)
- 기대 효과 (방문객 증가, 매출 증대 등)
- 후속 조치 필요한 것

**주의:**
- 결과는 팩트, 기대효과는 "기대" 표시 명확히
- 숫자가 있으면 반드시 포함

### 5. 활용자료/제공자료

**목적:** 컨설팅에 사용하고 제공한 자료 목록

작성 포인트:
- 무료 체험이벤트 분기별 실행 계획
- 재무제표 자료 (매출, 손익, 당좌분석 등)
- 소상공인 지원사업 안내
- 기타 제공 자료

### 6. 현장 사진 (2페이지)

- 1~3일차, 각 일차별 내부(필수)/외부
- 타임스탬프 앱 사용 필수
- 사진은 HWPX에 직접 삽입하지 않음 (수기 부착 또는 별도 처리)

## 표현 금지 목록

| 쓰지 않는 표현 | 대신 쓰는 표현 |
|---|---|
| 혁신적인 | (구체적 내용으로 대체) |
| 차별화된 | (구체적 차이점 서술) |
| 시너지 | 함께 했을 때의 효과 |
| 극대화 | 늘리기, 높이기 |
| 최적화된 솔루션 | (구체적 방법 서술) |
| 전략적 접근 | (구체적 접근법 서술) |
