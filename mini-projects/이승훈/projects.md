## 완료된 프로젝트

### 1. 슈파 데이터분석 (v2)

**분야:** 식자재 공동구매 데이터 분석
**상태:** v2 완료
**파일:** `슈파-데이터분석/index.html`

#### 주요 기능
- 엑셀형 스프레드시트 입력 (직접 데이터 입력 가능)
- 파트너별 구매 리포트 자동 생성
- 공급처별 총구매량 집계
- 협상 제안서 자동생성
- 엑셀 파일 가져오기 (SheetJS)
- 비밀번호 잠금 기능
- 발표 모드
- localStorage 데이터 저장 (v1→v2 자동 마이그레이션 지원)

#### 향후 계획
- 파트너가 직접 데이터를 입력하는 방식 검토 중
- 복잡도가 올라가면 풀 프로젝트로 전환 예정
