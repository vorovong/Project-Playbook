## 2라운드 개선안 요약

1라운드에서 미애(브랜드)와 영희(데이터)가 공통으로 짚은 핵심 문제 3가지:
- **Service + Vision이 같은 말을 두 번 한다**
- **Impact 시나리오가 브랜드 페이지의 결을 깬다**
- **두 번째 인용이 Origin 본문과 중복이며, 사람의 목소리가 없다**

이 문서는 "그래서 구체적으로 어떻게?"를 다룬다.

---

## 1. 전체 섹션 순서 재배치안

### 현재 구조 (v2-장황)

```
Hero -> Reality -> 인용1 -> Origin -> Service -> Vision -> Impact -> 인용2 -> Principle -> CTA
```

10개 섹션. 중반에 Service/Vision/Impact가 늘어지고, 후반부로 갈수록 영업 페이지 톤으로 기운다.

### 제안 구조 (v3)

```
Hero -> Reality -> 인용1 -> Origin -> What We Do(통합) -> 인용2(사람 목소리) -> Principle -> CTA
```

8개 섹션. 핵심만 남긴다.

**변경 포인트:**
- Service + Vision = "What We Do" 하나로 통합
- Impact = 완전 삭제. 대체 섹션 없음 (이유는 아래)
- 인용2 = 사람의 목소리로 교체
- Principle = 위치 유지하되 항목 3개로 축소 (4개 -> 3개, "동네에서 확장합니다"는 CTA로 흡수)

**왜 Impact를 대체 없이 삭제하는가:**

1라운드에서 "수치 시뮬레이션 대신 체감형 문장으로 바꾸자"고 제안했었다. 하지만 다시 생각하면, "가게 하나가 달라지는 모습"을 감성적으로 풀어도 결국 같은 역할(설득)을 한다. 브랜드 페이지에서 설득 섹션은 하나면 충분하고, 그 역할은 이미 "What We Do" 통합 섹션이 한다.

절감 수치가 필요한 맥락(IR, 사업계획서, 서비스 랜딩페이지)은 별도 페이지에서 다루면 된다. 브랜드 페이지의 임무는 "이 브랜드가 뭘 믿는가"를 느끼게 하는 것이지, "얼마를 아낄 수 있는가"를 증명하는 게 아니다.

## 2. Service + Vision 통합: "What We Do" 섹션 구성안

### 설계 원칙

- 4개 서비스를 카드로 나열하지 않는다. 하나의 이야기로 흘러가게 한다.
- "따로따로 보이지만 하나의 흐름"이라고 설명하는 대신, 읽는 사람이 리듬으로 느끼게 한다.
- 프랜차이즈 본사 비유는 유지한다 -- 이것은 이 페이지에서 가장 강력한 포지셔닝 문장이다.

### 텍스트 방향

```
WHAT WE DO

사장님이 장사만 하셨는데,
가게 운영 전체가 돌아갑니다.

---

식자재를 같이 사서 단가를 낮추고,
거래 내역은 자동으로 정리되고,
손님 리뷰엔 사장님 말투 그대로 답변이 달리고,
가게 소식은 알아서 SNS에 올라갑니다.

기술을 배우실 필요 없습니다.
사장님이 하시던 대로 하시면, 뒤에서 슈파가 연결합니다.

---

큰 프랜차이즈에는 본사가 해주는 일들이 있습니다.
슈파는 동네 사장님한테 그 역할을 합니다.
```

### 구조 해설

- **첫 두 줄:** 결론 먼저. "장사만 하셨는데 전체가 돌아간다"는 것이 슈파의 약속이다.
- **중간 네 줄:** 공동구매/거래분석/리뷰/SNS를 카드가 아니라 연쇄 문장으로 배치. "~하고, ~되고, ~달리고, ~올라갑니다" 리듬이 "하나의 흐름"을 설명 없이 체감하게 만든다.
- **"기술을 배우실 필요 없습니다":** 현재 Vision 섹션의 핵심 메시지. 버리면 안 된다. 사장님들의 가장 큰 심리적 허들이 "또 뭐 배워야 하나?"이기 때문이다.
- **마지막 두 줄:** 프랜차이즈 본사 비유. 이 문장 하나가 슈파의 포지셔닝 전체를 담는다. 독립된 호흡으로 분리해서 무게감을 준다.

### v2에서 삭제되는 것들

| 삭제 항목 | 이유 |
|---|---|
| 서비스 카드 4개 (아이콘 + 설명 박스) | 나열형은 "기능 소개"가 되고, 브랜드 페이지의 결과 맞지 않는다 |
| "따로따로 보이지만 하나의 흐름입니다" | 설명하지 않아도 리듬으로 느끼게 만들었다 |
| "사장님이 식자재를 주문하면, 그 데이터가..." | What We Do 중간 네 줄이 같은 내용을 더 간결하게 전달한다 |
| "발주, 마케팅, 데이터 관리, 교육" | 프랜차이즈 본사 비유에서 구체 항목 나열은 과잉. "그 역할"로 충분하다 |
| 서비스 배지 (NOW / SOON) | 브랜드 페이지에서 출시 상태 표기는 영업 페이지 문법이다 |

## 3. Impact 섹션 삭제 후 처리

### 삭제 대상 전체

- Impact 섹션 라벨, 타이틀, 본문
- scenario 박스 (월 매출 2,700만 원 시뮬레이션)
- "리뷰 답변에 매일 30분, SNS에 매일 20분" 문장
- "식자재비 절감은 바로 순이익이 됩니다" 문장

### 영희 지적 사항과의 연동

영희가 짚은 문제들이 Impact 삭제로 자연 해소된다:

| 영희 지적 | 해소 방법 |
|---|---|
| "순이익 200만 원" vs 푸터 "221만 원" 불일치 | 시나리오 삭제로 해당 수치 자체가 사라짐 |
| 시간 절약 수치(30분, 20분, 1시간)에 출처 없음 | Impact 삭제 + Reality 섹션에서도 구체 시간 수치 제거 (아래 참고) |
| "5% 절감" 근거 미제시 | 시나리오 삭제로 해당 주장 자체가 사라짐 |

### Reality 섹션 연동 수정

Impact가 사라지면, Reality 섹션의 problem-list도 정비해야 한다. 영희의 "출처 없는 시간 수치" 지적을 반영한다.

**현재 problem-list (5개):**
```
1. 매출의 절반 가까이가 재료비로 나갑니다
2. 남는 건 월 200만 원 남짓
3. 문 닫으신 사장님 절반이 "재료비 때문"이라고 하셨습니다
4. 가격 비교에 하루 1시간, 장부 정리에 매일 30분
5. 리뷰가 쌓여도 답변 쓸 시간이 없습니다
```

**제안 problem-list (3개):**
```
1. 매출의 40%가 재료비로 나갑니다
2. 남는 건 월 220만 원 남짓
3. 폐업 사장님 46%가 "재료비 부담"을 꼽으셨습니다
```

변경 근거:
- 항목 5개 -> 3개: 1라운드에서 "감정 곡선상 3개로 충분하다"고 판단한 것 유지
- "절반 가까이" -> "40%": 영희 지적 반영. 숫자 자체가 충분히 충격적이다
- "200만 원" -> "220만 원": 영희 지적 반영. 푸터 출처(221만 원)와 통일
- "절반이 재료비 때문" -> "46%가 재료비 부담을 꼽으셨습니다": 영희 지적 반영. 복수응답임을 감안한 표현
- 4번(시간 수치), 5번(리뷰 시간) 삭제: 출처 없는 수치. "What We Do" 섹션에서 "리뷰엔 답변이 달리고"로 자연스럽게 해결됨을 보여주므로, Reality에서 별도로 시간을 언급할 필요 없다

### 삭제된 4번, 5번의 감정적 역할을 어디서 보충하는가

4번("가격 비교에 하루 1시간")과 5번("리뷰 쓸 시간이 없다")은 "사장님의 하루가 잡무로 채워진다"는 감정을 전달하는 항목이었다. 이 감정은 Reality 섹션 본문이 이미 담당하고 있다:

```
가게 문 열기 전에 식자재 주문.
가게 문 닫고 나서 리뷰 답변, SNS 올리기, 장부 정리.

사장님이 진짜 잘하시는 건 요리이고 손님 맞이인데,
정작 그 시간은 줄고 잡무 시간만 늘어갑니다.
```

이 본문이 "잡무의 무게"를 느끼게 하고, problem-list 3개가 "구조적 숫자"로 뒷받침한다. 역할 분담이 명확해진다.

## 4. 두 번째 인용 블록: 사람의 목소리

### 현재 문제

```
"사장님한테 필요한 건 '최신 기술'이 아니라 '당장 도움 되는 것'입니다."
```

Origin 본문 마지막 문장과 거의 동일. 강조가 아니라 실수처럼 보인다. 그리고 이 인용은 "누구의 말"인지가 없다. 브랜드 자체의 메시지를 인용 부호로 감싼 것일 뿐이다.

### 위치 변경

v3 구조에서 인용2는 "What We Do" 다음, "Principle" 앞에 온다.

```
... What We Do -> [인용2] -> Principle -> CTA
```

이 위치에서 인용2의 역할: "슈파가 하는 일"을 들은 직후, 실제로 그 일을 겪은 사람의 한마디. 감정적 전환점이자 신뢰 장치.

### 3가지 선택지

**선택지 A: 실제 사장님 목소리 (최우선 권장)**

```
"처음엔 또 뭐 팔려고 하나 했는데,
진짜 옆에서 같이 하더라고요."
-- 행궁동 OO식당
```

왜 최우선인가:
- 브랜드 페이지에서 가장 강력한 신뢰 장치는 제3자의 목소리다
- "또 뭐 팔려고 하나 했는데"가 방문자의 방어심리를 대변해준다. 읽는 사람이 "나도 그렇게 생각했는데"라고 느끼는 순간, 그 다음 문장("진짜 옆에서 같이 하더라고요")의 무게가 달라진다
- 공존공간이 2년간 142회 현장 경험을 했으므로, 실제 사장님 한마디를 수집하는 것은 어렵지 않다

주의: 실제 사장님의 동의를 받은 실명/가게명이어야 한다. 허구의 인용은 브랜드 신뢰를 즉사시킨다.

**선택지 B: 대표(이승훈) 목소리**

```
"기술을 만드는 게 아니라
사장님 옆자리를 채우는 겁니다."
-- 공존공간 이승훈
```

왜 차선인가:
- 대표의 얼굴과 이름이 나오면 "이 브랜드 뒤에 사람이 있다"는 인식이 생긴다
- 하지만 대표의 말은 결국 브랜드의 말이다. 제3자의 신뢰감에는 못 미친다
- 대표 사진을 함께 넣으면 효과가 올라가지만, 현재 페이지가 텍스트 중심이므로 톤이 갑자기 바뀔 수 있다

**선택지 C: 팀 목소리 (복수)**

```
"사장님이 '고마워요' 하실 때,
이 일을 왜 하는지 알게 됩니다."
-- 슈파팀
```

왜 3순위인가:
- 진정성은 있지만, "팀"이라는 주체가 추상적이다
- 개인의 이름이 없으면 인용의 무게가 가볍다

### 최종 권고

**선택지 A를 쓸 수 있으면 A.** 실제 사장님 인터뷰에서 한마디를 뽑아오는 것이 최선이다. 당장 수집이 어려우면 **선택지 B**로 시작하고, 사장님 목소리가 확보되면 교체한다.

## 5. Principle 섹션 조정

### 현재 4개 -> 3개로 축소

| 현재 | 제안 |
|---|---|
| 01 현장에서 시작합니다 | 유지. "142번의 현장 경험"은 이 페이지에서 유일한 자체 데이터 포인트. |
| 02 먼저 결과를 보여드립니다 | 유지. Impact 섹션을 삭제했으므로, "먼저 결과를 보여드린다"는 약속이 여기서 한 번은 나와야 한다. |
| 03 같이 만들어갑니다 | 유지. "완성된 제품이 아니라 같이 만들어가는 서비스" -- 브랜드 정체성의 핵심. |
| 04 동네에서 확장합니다 | 삭제. CTA 섹션의 부연으로 흡수. |

### 04 항목의 CTA 흡수 방안

현재 CTA:
```
가게 이야기, 편하게 들려주세요.
거기서부터 시작하겠습니다.
```

제안 CTA:
```
CONTACT

수원에서 시작합니다.
사장님 가게에서 시작합니다.

가게 이야기, 편하게 들려주세요.
```

"수원에서 시작합니다"가 04의 "동네에서 확장합니다"를 흡수하면서, CTA의 구체성을 높인다. "사장님 가게에서 시작합니다"는 Origin의 "주방 옆에서 시작했습니다"와 수미쌍관 구조를 만든다.

## 6. 영희 지적 추가 반영: 인라인 출처

Reality 섹션의 problem-list 3개 항목에 인라인 출처를 넣는다.

```html
<li>매출의 <strong>40%</strong>가 재료비로 나갑니다
  <span class="stat-source">농림축산식품부, 2024</span></li>
<li>남는 건 월 <strong>220만 원</strong> 남짓
  <span class="stat-source">식품외식경제, 2024</span></li>
<li>폐업 사장님 <strong>46%</strong>가 "재료비 부담"을 꼽으셨습니다
  <span class="stat-source">중소기업중앙회, 2025</span></li>
```

`.stat-source` 스타일: `font-size: 11px; color: #444; margin-left: 8px;`

숫자 옆에 출처가 바로 보이면, "이 브랜드는 숫자를 대충 쓰지 않는다"는 인식이 생긴다. 푸터 출처는 유지하되, 핵심 3개는 이중 표기한다.

## 7. v3 전체 섹션 맵 (최종)

| 순서 | 라벨 | 역할 | 감정 곡선 |
|---|---|---|---|
| 1 | Hero | 첫인상. "사장님 옆에서 같이 일합니다" | 호기심 |
| 2 | Reality | 문제 공감. 숫자 3개로 구조적 어려움 제시 | 공감, 약간의 무거움 |
| 3 | 인용1 | "같이 하면 어떨까?" -- 전환점 | 전환, 희망 |
| 4 | Origin | 공존공간에서 시작한 이야기. 현장에서 출발 | 신뢰, 진정성 |
| 5 | What We Do | 슈파가 하는 일을 하나의 흐름으로. 프랜차이즈 본사 비유 | 이해, "아 이런 거구나" |
| 6 | 인용2 | 실제 사장님(또는 대표)의 한마디 | 감정적 확인, 신뢰 |
| 7 | Principle | 일하는 방식 3가지. 142회 현장 경험 | 신뢰 강화 |
| 8 | CTA | "수원에서 시작합니다. 가게 이야기 들려주세요." | 행동 유도 (부드럽게) |

감정 곡선이 "호기심 -> 공감 -> 전환 -> 신뢰 -> 이해 -> 확인 -> 강화 -> 행동"으로 한 번도 끊기지 않고 흐른다. 현재 v2의 "공감 -> 전환 -> 신뢰 -> 설명 -> 설명 -> 설득 -> 중복 -> 강화 -> 행동"에서 중간의 반복과 톤 전환이 제거된 구조다.

## 부록: 강점/약점/제안 요약

### 강점 3개

1. **Reality-Origin 흐름은 건드릴 필요 없다.** 1라운드에서 "이 페이지에서 가장 좋은 부분"이라 했고, 2라운드에서도 동의한다. 숫자 정확도만 영희 기준으로 맞추면 완성이다.
2. **프랜차이즈 본사 비유가 슈파의 포지셔닝을 한 문장으로 끝낸다.** "큰 프랜차이즈에는 본사가 해주는 일들이 있습니다. 슈파는 동네 사장님한테 그 역할을 합니다." 이 두 줄은 이 페이지의 킬링 카피다.
3. **CTA의 톤이 브랜드 페이지에 정확히 맞는다.** "가게 이야기, 편하게 들려주세요." 영업이 아니라 대화를 제안하는 톤. 여기에 "수원에서 시작합니다"를 더하면 구체성과 진정성이 동시에 올라간다.

### 약점 3개

1. **Service/Vision/Impact 3개 섹션이 같은 말을 세 번 한다.** 서비스 카드 -> 흐름 설명 -> 수치 시뮬레이션. 감정의 농도가 희석되는 주범이다.
2. **인용 블록 2개 모두 브랜드 자체의 메시지다.** 사람의 얼굴이 없다. 인용은 페이지에서 가장 눈에 띄는 요소인데, 그 공간을 활용하지 못하고 있다.
3. **숫자 표현이 "대충 비슷하게"로 일관된다.** "절반 가까이"(40.4%), "200만 원"(221만 원), "절반"(46%). 한두 개면 수사적 선택이지만 세 개가 겹치면 신뢰 문제가 된다.

### 개선 제안 3개 (실행 순서)

1. **Service + Vision을 "What We Do"로 통합하고, Impact를 삭제한다.** 10개 섹션 -> 8개 섹션. 중반부 반복 제거. 페이지 길이가 30% 이상 줄어들면서 감정 곡선이 끊기지 않는다.
2. **두 번째 인용을 실제 사장님 목소리로 교체한다.** 불가능하면 대표 이승훈의 한마디로 대체. "사람의 얼굴"이 이 페이지에 하나는 있어야 한다.
3. **Reality problem-list를 3개로 줄이고, 숫자를 원본 기준으로 통일하며, 인라인 출처를 넣는다.** 영희의 데이터 정확성 지적을 한 번에 해소한다.
