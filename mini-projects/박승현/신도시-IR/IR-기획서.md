## 2. Problem — 숫자로 보는 위기

### 핵심 메시지
> **대한민국에서 가장 거대하지만, 가장 비효율적인 시장.**

### The 95% Majority

| 지표 | 수치 |
|------|------|
| 국내 소상공인 사업체 수 | 약 790만 7천 개 (전체의 95.2%) |
| 소상공인 종사자 수 | 약 1,090만 명 (전체의 45.9%) |

> 소상공인은 국가 경제의 모세혈관이자 고용의 핵심이나,
> 가장 낙후된 경영 환경에 방치되어 있다.

### 구조적 문제

| 문제 | 현실 |
|------|------|
| **경영 비효율** | 세무, 마케팅, 재고, 고객관리를 사장님이 직접 → 본업에 집중 못 함 |
| **고비용 구조** | 개별 점포 단위의 구매·마케팅·채용 = 구조적으로 '고비용 저효율' |
| **기존 지원의 한계** | 교육·컨설팅(간접 지원) → "배웠는데 할 시간이 없다" |
| **디지털 격차** | AI 기술은 발전하지만, 소상공인 현장에 맞는 도구는 거의 없음 |
| **생존 위기** | 인건비 급등 + AI 가속화 속에서, 데이터 없는 소상공인은 '소멸' 위기 |

## 3. Solution — 슈파(SUPA): 사장님의 슈퍼 파트너

### 핵심 메시지
> **"뭉치면 싸지고, 쌓이면 보인다."**
> AI로 경영을 도와주고, 유저가 모이면 비용까지 낮춰준다.
> 사장님이 내는 돈은 거의 없다. AI가 운영하니까.

### 한 문장 정의
**슈파 = 소상공인의 AI 경영 파트너. 경영 효율화(거의 무료) + 공동구매·물류 비용 절감.**

### (주)신도시의 정의
> 도시의 상업 활동을 관장하는 운영체제(OS)를 공급하는 **로컬 테크(Local Tech)** 기업.
> 핵심 솔루션 '슈파'로 파편화된 로컬 비즈니스를 디지털로 혁신한다.

### 왜 지금 가능한가
- AI 운영 비용 ≒ 0 → 수수료를 극도로 낮게 받아도 사업이 돌아감
- 기존 SaaS는 구독료가 수익 → 무료로 못 함
- 슈파는 구독료가 아니라 **데이터와 네트워크 효과**가 수익 → 무료/소액이 전략

## 4. Product — 서비스 구조

### 4-1. 플라이휠 (핵심 도식)

```
AI로 운영비 ≒ 0   →   수수료 최소화
       ↓
사장님 부담 없이 가입   →   유저 확보
       ↓
유저가 모이면   →   공동구매·물류 원가 절감
       ↓
경영 지원 + 실제 비용 절감   →   더 많은 사장님 유입
       ↓
데이터 축적   →   진짜 자산
       ↓
       🔄 계속 돈다
```

### 4-2. Phase 1 — 행궁동·수원 장악 (~27년 2분기)

**"Traffic First, Monetize Later." — 선 트래픽 확보, 후 고도화**

**① Cost Innovation — 비용의 파괴적 혁신 [Entry Strategy]**

초기 유저 확보를 위한 핵심 가치(Killer Value) = '즉각적인 비용 절감'

| 기능 | 설명 |
|------|------|
| **슈파 Buying (공동 구매)** | 1,000개 가맹점의 구매력 통합 → 식자재·소모품·배달용기를 프랜차이즈 본사 공급가 수준으로 인하 |
| **Shared Service (공유 인프라)** | 세무 기장, 법률 자문, 방역, 렌탈 등 공통 고정비를 'N분의 1' 구조로 최소화 |

- 유통 도메인 파트너사와 연결 (직접 물류 안 함 = **Asset Light**)
- 5개소만 묶어도 유통 업체에게 매력적 → 50개소, 100개소면 협상력 ↑

> 🟡 **[미결정 #5]** 유통 파트너 후보 — 구체적 업체명 or 유형 (식자재 유통사? 배달 플랫폼? 물류 3PL?)

**② AI 경영 효율화 (거의 무료)**

| 기능 | 설명 |
|------|------|
| 세무·장부 자동 정리 | AI가 매출·비용 자동 분류, 세무 신고 보조 |
| SNS 콘텐츠 자동 생성 | 가게 사진 → 인스타·네이버 포스트 자동 생성 |
| 매출·원가 분석 리포트 | 월별·주별 매출 트렌드, 원가율 알림 |
| 인력 매칭 | 단기 아르바이트·전문 인력 연결 |
| 고객·예약 관리 | 단골 관리, 예약 자동 응대 |
| 맞춤 자동화 | 가게별 고유 문제 진단 → 커스텀 해결 (현장 지식 기반) |

> 🟡 **[미결정 #3]** 슈파 MVP 기능 범위 — 위 기능 중 9월까지 정확히 뭘 만드는지 확정 필요

**③ Data Insight — 데이터 기반 통찰 [Value-Add Strategy]**

비용 절감으로 확보된 유저들의 데이터를 분석하여 '정답지' 제공.

| 기능 | 설명 |
|------|------|
| **Peer Benchmarking** | "감으로 장사하지 마세요." 내 가게 매출·원가율·인건비를 동일 상권 상위 10%와 비교 |
| **Predictive AI** | 1,000개소+ 데이터 축적 시, 날씨·행사·유동인구 결합 → "내일의 적정 발주량"과 "필요 인력 수" 예측 |

**④ 완전 무료 전략 + 프리미엄 쿠폰**

> **Phase 1에서는 구독료 0원. 경영자동화 전 기능을 무료로 개방한다.**
> 프리미엄 기능(Predictive AI, Peer Benchmarking, 맞춤 자동화)도 **홍보 쿠폰**으로 무료 제공.
> → 유저가 프리미엄에 익숙해진 상태로 Phase 2 진입 → 구독 전환율 극대화.

| 기존 플랫폼 | 수수료 | 슈파 |
|------------|--------|------|
| 네이버 스마트스토어 | 5~8% | **구매몰 수수료 3%** |
| 쿠팡 마켓플레이스 | 10~15% | (기존 대비 50~80% 절감) |
| 배달의민족 | 6.8% + 광고비 (실질 15~20%) | |
| SPC 등 B2B 유통 | 15~25% | |

- 슈파 구매몰 수수료 3% 중 PG 결제 수수료 약 1.5% → 슈파 실질 마진 약 1.5%
- AI 운영비 ≒ 0이므로 1.5% 마진으로도 운영 가능

**KPI**: 행궁동 100개소 → 수원 1,000개소
**수익**: 구매몰 수수료 3% + B2G 용역 (캐시카우). 구독료 없음

### 4-3. Phase 2 — 국내 확장 + 수익화 (💰 27년~)

**세 가지 수익 엔진이 동시에 켜지는 단계**

**① 구독 서비스 시작 — 월 4,900원**

Phase 1에서 프리미엄 쿠폰으로 무료 체험한 유저 → 자연스러운 유료 전환.

| 구분 | 내용 |
|------|------|
| **무료** (유지) | 기본 경영자동화 (세무 자동정리, 매출분석, 기본 리포트, SNS 기본 생성) |
| **프리미엄** (월 4,900원) | Predictive AI, 맞춤 자동화, 마케팅 고급 분석, Peer Benchmarking, 우선 지원 |
| **광고** (별도) | 유통사/공급사 → 소상공인 대상 타겟 광고 |

- 가격 근거: 월 5,000원 미만 = 한국인 구독 심리적 최소 저항 구간 ("커피 한 잔 값")
- 네이버 플러스 멤버십(4,900원)과 동일 가격대 → 익숙함
- **전환율 목표: 30%**

> **전환율 30% 근거**
>
> | 근거 | 설명 |
> |------|------|
> | 가격 | 월 4,900원 — 5,000원 미만 최소 저항 구간 |
> | 체험 선행 | Phase 1에서 프리미엄을 이미 무료로 써봄 → "없으면 불편" |
> | 가치 체감 | Predictive AI = 직접적 매출 영향 (발주량 예측, 인력 최적화) |
> | 명분 | 구독료의 일부가 상권 기금으로 → "내 돈이 우리 동네를 지킨다" |

> **검증 계획**: 행궁동 100개소 실증 기간(26.9~12월)에 프리미엄 쿠폰 사용률·잔존율 실측.
> Phase 2 전환 시 목표 미달이면 가격/기능 조정 후 재검증.

**② 상권 기금 선순환 — 구독이 동네를 지킨다**

```
소상공인이 월 4,900원 구독
       ↓
일부를 공존 기금에 출연
       ↓
기금이 로컬 기획 프로젝트 지원
(상권 활성화, 젠트리피케이션 방지, 재기 지원)
       ↓
상권이 좋아지면 → 소상공인 매출 ↑
       ↓
더 많은 소상공인이 슈파 가입 + 구독 유지
       ↓
       🔄 선순환
```

> **기존 플랫폼과의 차이**: 배민·쿠팡은 수수료를 가져가기만 한다.
> 슈파는 돈이 지역에서 도는 구조를 만드는 플랫폼이다.

**③ 데이터 판매 — 진짜 자산이 되는 단계**

- 1,000개소의 실데이터(Real Data)가 모여야 유의미한 AI 학습과 패턴 분석 가능
- **CSP와 함께 데이터 판매** → 실제 돈 버는 구조

| 데이터 구매자 | 활용 | 예상 가격대 |
|---|---|---|
| 금융사 | 소상공인 신용평가, 맞춤 대출 | 연 3,000만~1억원 |
| 유통사 | 상권·소비 트렌드, 수요 예측 | 연 2,000~8,000만원 |
| 보험사 | 업종별 리스크 분석 | 연 1,000~3,000만원 |
| 지자체 | 상권 정책 근거 데이터 | 연 500~2,000만원 |
| 프랜차이즈 본사 | 벤치마킹, 신규 출점 | 리포트당 30~100만원 |

> **데이터 가치의 핵심**: 매출 데이터는 카드사에서도 얻을 수 있지만, **원가·인건비·재고·소비자 행동** 데이터는 사업장 내부에서만 나온다. 이 "내부 운영 데이터"가 프리미엄.

> **레퍼런스**: 캐시노트(KCD)는 170만 사업장 기반 연결 매출 1,428억원. 사업장당 약 8만원/년 수준.

**수익**: 구독 4,900원 + 구매몰 수수료 3% + 데이터 판매 (CSP 협업) + 광고
**목표**: 전국 확장 + 데이터 커버리지

### 4-4. Phase 3 — 피지컬 AI (🤖 라스트마일)

**현실에서 데이터를 모으고, 피지컬 AI를 구현하는 도메인**

Phase 1~2에서 쌓은 것:
- 소상공인 현장 접점 (설치·운영 채널)
- 업종별 운영 데이터 (학습 데이터)
- 물류·배송 네트워크 (인프라)

→ 피지컬 AI 접목: 배달·물류 로봇, 매장 자동화 기기, 무인 주문·결제, 재고 자동 관리

**핵심**: 로봇을 만드는 건 다른 회사도 하지만, 소상공인 현장에 놓을 수 있는 **도메인(현장 채널 + 데이터 + 현장 지식)**을 가진 곳은 우리뿐.

> **Phase 2 ~ Phase 3 = 라스트마일**
> 소프트웨어(데이터) → 하드웨어(피지컬 AI) / 디지털 세계 → 현실 세계

### 4-5. Ecosystem Fund (공존 기금)

- 슈파를 통해 절감된 비용의 일부 또는 플랫폼 수익의 일부를 적립
- 상권 보호, 젠트리피케이션 방지, 실패한 소상공인의 재기를 돕는 **지역 상생 기금**
- 플랫폼의 지속 가능성(Sustainability) 확보

## 5. Business Model — 수익 구조

| **Tech Growth** | 피지컬 AI (RaaS, 하드웨어) | Phase 3 |

### 왜 무료/소액이 가능한가

```
기존 SaaS                              슈파
```

| 우위 | 설명 |
|------|------|
