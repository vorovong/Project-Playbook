## 이 청중은 누구인가

- 정부/지자체 사업 공모의 평가위원
- 하루에 제안서 5~10개를 읽음
- 시간이 부족하고, 평가 기준표에 맞춰 점수를 매김

## 이 청중이 중시하는 것

| 우선순위 | 항목 | 승현님 무기 |
|---|---|---|
| 1 | **정량적 성과** — 숫자로 증명 | 글로컬상권 2년 실적 (소비액 +107%, 협업 +710%) |
| 2 | **사업 필요성** — 왜 지금 이것을 해야 하는가 | 에센스: "콘텐츠의 총합이 상권이다" + 현장 경험 |
| 3 | **실행 가능성** — 정말 할 수 있는 팀인가 | 4명 팀이지만 155억 규모 운영 경험 |
| 4 | **예산 합리성** — 돈을 잘 쓸 것인가 | 비예산 자생적 협업 사례 (롯데, 오뚜기) |
| 5 | **차별성** — 다른 곳과 뭐가 다른가 | 현장 지식 = 진입 장벽 |

## 프레이밍 규칙

**강조할 것:**
- "양보다 질" — 방문객 수보다 1인당 소비액
- "자생 구조" — 정부 사업이 끝나도 남는 것
- "전국 최초/유일" — 정책적 차별화

**피할 것:**
- 추상적 비전만으로 채운 배경
- "혁신", "획기적" 같은 빈 수식어
- 예산 항목에 "기타" 남발

## 문서 구조 권장

```
1. 사업 개요 (한 페이지 요약)
2. 배경 및 필요성 (에센스 + 데이터)
3. 목표 (정량 지표 필수)
4. 사업 내용 (프로그램 + 타임라인)
5. 추진 체계 (조직도 + 역할)
6. 예산 계획 (항목별 상세)
7. 기대 효과 (Before/After 예측)
8. 기관 역량 (실적 테이블)
```
