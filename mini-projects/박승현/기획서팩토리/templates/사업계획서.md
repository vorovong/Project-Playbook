> 이 템플릿은 "뼈대"다. 실물 기획서가 들어올 때마다 살을 붙인다.

## 작성 전 확인 (AI 필수)

1. **청중은 누구인가?** → `DNA/audience/` 해당 파일 로드
2. **어떤 사업인가?** → `기획/` 해당 파일에서 에센스 확인
3. **이전에 같은 청중에게 뭘 줬는가?** → `기획서-관리.md` 확인
4. **style-guide.md** 로드

## 목차 구조

> 실물 #001과 비교 검증 완료. 공모사업 양식에 따라 섹션 순서/유무가 달라질 수 있음.

```
제1장. 사업 개요
   - 사업명
   - 신청 기관: (주)공존공간
   - 사업 기간
   - 사업비 (총액 / 자부담) — ※ 별도 예산서가 있으면 여기는 총액만
   - 한 줄 핵심 (에센스)
   - [선택] 마스터플랜 지도 — 물리적 범위 + PLAYER 목록 (#001 패턴)

제1장. 사업 추진 배경 및 필요성
   - 역사 서사: 과거 → 쇠퇴 → 기존 노력 → 이중성 → 제안 (#001 패턴)
   - 에센스 (왜 이것을 하는가 — 1~2문장)
   - 현황 데이터 (Before 숫자)
   - 문제 인식 — 단순한 문제가 아닌 "이중성" 프레이밍
   - 사업 필요성 (외부 레퍼런스/이론 활용: 창조도시론, 15분도시 등)
   - ※ 장황하게 쓰지 않는다. 에센스 + 숫자 + 서사

제2장. 환경 분석 및 전략 수립 ← (#001에서 발견, 기존 템플릿에 없었음)
   - 물리적 현황 (면적, 유동인구, 비교 데이터)
   - 기존 지원사업 현황 (투입 예산, 기간)
   - 통계 데이터 테이블 (인구변화, 사업체수, 유동인구 등)
   - 종합분석 → 전략 도출

제2장. 목표
   - 정량 목표 (숫자로 측정 가능한 것)
     | 지표 | 현재 | 목표 | 측정 방법 |
   - 정성 목표 (구조 변화, 문화 변화)

제3장. 세부 사업 내용
   - 프레임 태깅: 모든 세부사업에 하나의 키워드 체계 부여 (#001: 직주락 W/L/P)
   - 세부사업별 구조:
     | 사업명 | 프레임 태그 | 대상 | 내용 | 기대 효과 |
   - [선택] 로컬 크리에이터 / 협력 네트워크 현황
   - 추진 일정 (월별 타임라인)
     | 월 | 주요 활동 | 산출물 |

제3장 또는 별도. 추진 체계
   - PLAYER 목록 (이해관계자) (#001 패턴)
   - 조직도 / 역할 분담
     | 역할 | 담당 | 책임 범위 |
   - 협력 기관 (해당 시)
   - 거버넌스 (의사결정 구조)

[별도 서류 가능] 예산 계획
   - 항목별 예산 내역
     | 항목 | 산출 근거 | 금액 | 비고 |
   - 자부담 / 보조금 비율
   - ※ "기타" 항목 최소화. 산출 근거 구체적으로

제4장. 기대 효과 / 비전
   - Before/After 예측
     | 지표 | Before (현재) | After (사업 후) |
   - 자생 구조 (사업 종료 후에도 남는 것)
   - 확산 가능성 (타 지역 적용)
   - [선택] 비전형 마무리 (#001: "IF 정조 NOW")

[기관 역량 — 별도 섹션 또는 부록]
   - 주요 실적 테이블
     | 시기 | 사업명 | 규모 | 성과 |
   - 핵심 인력
   - 유사 사업 경험
   - 사업 영역 다이어그램 (#001: 3축 + 6대 역량)
```

## 섹션별 가이드

### "배경 및 필요성"에서 자주 하는 실수

```
❌ 2페이지에 걸쳐 일반적인 시장 동향을 장황하게 서술
✅ 에센스 한 줄 + "우리가 2년간 현장에서 확인한 것" + Before 숫자
```

### "배경"을 역사 서사로 풀기 (#001 실물 패턴)

```
❌ "행궁동은 수원의 중심 관광지로서 활성화가 필요한 지역이다"
✅ "1796년 정조의 신도시 → 인구 37,000→9,000 쇠퇴 → 100억 도시재생 투입
   → 그런데도 '관광도시인데 살기 힘든 동네'라는 이중성"
```

### "환경 분석"에서 데이터 활용법 (#001 실물 패턴)

```
❌ "유동인구가 증가하는 추세임"
✅ 통계 테이블 4종 (인구변화/사업체수/유동인구/거점공간) +
   "인구는 줄고 사업체와 유동인구는 늘었다 → 이것이 이중성의 증거"
   → 여러 데이터가 하나의 결론을 향하도록 배치
```

### "기대 효과"에서 자주 하는 실수

```
❌ "지역 경제 활성화에 기여할 것으로 기대됨"
✅ "1인당 소비액 13,810원 → 18,000원 (+30%) 목표. 산출근거: 2년간 +48% 달성 추세"
```

### "추진 체계"에서 승현님 차별점

```
❌ 일반적 조직도
✅ "4명 팀이지만 AI 자동화로 10명 수준의 생산성" + 구체적 AI 활용 영역 명시
```
