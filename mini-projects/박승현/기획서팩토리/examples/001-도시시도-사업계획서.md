## 승현님 스타일 패턴 발견

### 1. "역사 서사" 구조 — 배경을 스토리로 풀기
```
조선시대 정조의 신도시 → 쇠퇴 (인구 37,000 → 9,000) → 재생 노력 → 현재의 이중성 → 미래 제안
```
- 단순 "현황" 나열이 아니라 **시간축 서사**로 풀어냄
- "관광도시 행궁동의 이중성" — 문제를 단순화하지 않고 복잡성을 보여줌
- "행궁동 인플레이션" — 승현님만의 용어로 현상을 명명

### 2. 데이터 활용 — 4종 통계 테이블
| 데이터 | 출처 | 용도 |
|---|---|---|
| 인구변화 (2000~2021) | 통계지리정보서비스 | 거주인구 감소 증명 |
| 사업체 수 변화 (2010~2021) | 통계청 전국사업체조사 | 상권 성장 증명 |
| 유동인구 변화 (분기별) | — | 활성화 증명 |
| 거점공간 현황 (6개소) | — | 인프라 증명 |

→ **"인구는 줄고, 사업체와 유동인구는 늘었다"** = 데이터로 이중성 증명

### 3. 지도/다이어그램 활용
- 마스터플랜 지도 (p.1): 물리적 범위 + PLAYER + 범례
- 팔달산 문화권 다이어그램 (p.7): 주거-로컬크리에이터-스몰브랜드-역사문화관광 4축
- 비짓행궁 서비스 구조도 (p.10)
- 도시재생 현황도 2종 (p.10)

### 4. 글쓰기 톤
- **형식**: ◯ (대항목) → Ÿ (중항목) → - (소항목) 3단계
- **핵심 문장 박스 처리**: "행궁동 일대 골목상권을 활성화하기 위해서..."
- **볼드 강조**: 핵심 개념만 선별적으로 (`'창조 계급'`, `도시의 미래 인재들이...`)
- **혼합 문체**: 서술형(~함/~됨) + 설명형(~있음) — 순수 음슴체보다 약간 격식
- **외부 레퍼런스**: 리처드 플로리다 <창조 도시론>, 파리 15분 도시, 포틀랜드

## 템플릿/DNA 업데이트 시사점

1. **"환경 분석" 섹션 추가 필요** — 실물에서는 독립 챕터로 존재
2. **역사 서사 패턴 문서화** — 배경을 스토리로 풀되, 데이터로 뒷받침
3. **"이중성" 프레이밍** — 문제를 단순화하지 않는 것이 승현님 특징
4. **직주락 태깅** — 사업을 W/L/P로 분류하는 프레임 반복 사용
5. **PLAYER 목록** — 이해관계자를 명시적으로 나열
6. **예산/추진체계 별도 여부 확인 필요** — 이 문서에 없는 건 별도 서류인지, 생략인지
