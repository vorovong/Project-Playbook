## 왜 이 책자가 좋은가 — 승현님이 "구성이 좋다"고 한 이유 분석

### 1. 같은 내용인데 완전히 다른 프레이밍

| 요소 | 사업계획서 (#001) | 책자 (#002) |
|---|---|---|
| 문체 | 음슴체 (~함, ~됨) | 합니다체 (부드러운 서술) |
| 청중 | 행정 평가위원 | 시민, 관광객, 파트너 |
| 구조 | 제1장~제4장 (체계적 나열) | Chapter + now + 인터뷰 (서사 흐름) |
| 데이터 | 통계 테이블 4종 | 없음 (감성으로 대체) |
| 핵심 전달 방식 | 직접 서술 | **Q&A 인터뷰** (공간 기획자 J 페르소나) |
| 직주락 표기 | (직), (락) 괄호 | **#직 #주 #락** 해시태그 |
| 사진 | 설명용 작은 사진 | **풀스프레드 감성 사진** |

→ **이것이 기획서팩토리가 배워야 할 것**: 같은 사업 → 청중에 따라 완전히 다른 문서

### 2. 시간축 서사의 "대중 버전"

```
사업계획서: 조선시대 정조 → 쇠퇴 (인구 37,000→9,000) → 재생 노력 → 이중성 → 제안
책자:       INTRO 질문 → Ch.1 정조의 꿈 → Ch.2 카페거리 탄생 → now 가능성 → now 이중성 → 인터뷰 → 프로젝트 → 마무리 철학
```

**같은 뼈대**, 다른 살. 책자는 데이터 대신 **질문과 사진**으로 독자를 끌어들임.

### 3. "공간 기획자 J" 페르소나 인터뷰 — 가장 영리한 장치

- "J"는 승현님의 철학을 담은 가상 페르소나
- 사업 비전을 직접 설명하지 않고 Q&A 형식으로 자연스럽게 전달
- 4개 질문으로 로컬 정의 → 갈증 → 방법론 → 미래상 순서
- **핵심 인용:**
  - "행복은 멀리 있지 않다... 가까운 곳에서 만드는 일상이야말로 행복의 지름길"
  - "로컬은 일상에서 즐길 수 있는 것을 찾아가며 행복과 감사함을 느끼고 나누는 일종의 '태도'"
  - "동네 중심의 생활이 하나의 문화가 될 수 있음을 보여주는 것. 제가 그리는 수원입니다."

→ MISSION.md의 "일상의 반경이 행복이고 로컬이다"와 정확히 같은 맥락

## 책자 만들 때 참고할 구성 공식

```
1. 표지 (브랜드 컬러 + 캘리그래피 제목)
2. INTRO (질문으로 시작)
3. 과거 이야기 (역사 서사, 1~2 챕터)
4. 현재 이야기 (가능성 + 한계의 이중성)
5. [선택] 페르소나 인터뷰 (Q&A 형식으로 비전 전달)
6. 프로젝트/콘셉트 소개 (핵심 한 줄 + 설명)
7. 브랜드/프로그램별 소개 (해시태그 + 한 줄 + 설명)
8. 풀스프레드 사진 (호흡)
9. 마무리 에세이 (철학/비전)
10. 뒤표지 (포인트 컬러 + 크레딧)
```

**핵심 원칙:**
- 한 스프레드에 하나의 메시지만
- 풀스프레드 사진으로 호흡 조절 (최소 2회)
- 데이터보다 질문과 감성으로
- 페르소나 인터뷰 = 비전을 자연스럽게 전달하는 최고의 장치
