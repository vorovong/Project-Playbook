## 2. 사단법인 행궁동행

### 2-1. 개요

| 항목 | 내용 |
|------|------|
| 성격 | 행궁동 지역 상권 및 주민 주체 사단법인 |
| 목적 | 지역 자율 운영 구조 — 스스로 결정하고 책임지는 공동체 |
| 기린의 역할 | 설립 주도, 방향성 결정, 대외 대표 |
| 현안 | 행정과의 갈등 대응 중 (2026-03-08 기준) |

### 2-2. 업무 영역

| 업무 | AI 역할 | 사람 역할 |
|------|---------|----------|
| 정관/규약 문서 정리 | 기존 문서 기반 정리 및 초안 수정 | 기린 — 내용 확인 및 최종 승인 |
| 사업계획서 초안 | 연간 사업 방향 기반 초안 작성 | 기린 — 이사회/총회 전 검토 |
| 공문서 작성 | 행정 기관 제출용 공문 초안 | 기린 — 최종 승인 후 발송 |
| 회의록 정리 | 이사회/총회 메모 기반 회의록 정리 | 기린 — 검토 후 공유 |
| 지원사업 탐색 | 사단법인 대상 공모사업 리스트 정리 | 기린 — 지원 여부 결정 |
| 회원/이해관계자 소통 | 공지 문구, 안내 문자 초안 작성 | 기린 — 확인 후 발송 |

## 3. (주)신도시

### 3-1. 개요

| 항목 | 내용 |
|------|------|
| 성격 | 공존공간 스핀오프 — 인력 발굴·육성·투자 |
| 에센스 | 현장에서 깨달은 것이 사업 구조를 바꿨다 |
| 기린의 역할 | 사업 전략 총괄 |
| 상태 | 스핀오프 준비 중 (미설립) |

### 3-2. 사업 구조

```
(주)신도시
├── 로컬벤처스튜디오 (AC) — 인력 발굴·육성·투자
└── 슈파(SUPA) — 소상공인 맞춤 AI 자동화
```

→ 상세: `팅커벨/STRATEGY/기획/신도시/신도시.md`, `기획/신도시/슈파.md`

### 3-3. 업무 영역

| 업무 | AI 역할 | 사람 역할 |
|------|---------|----------|
| IR 기획서 작성/관리 | 초안, 데이터 분석, 프론티어 비교 | 기린 — 방향 확정, 투자자 미팅 |
| 슈파 MVP 기획 | 리서치, 기능맵, 벤치마킹 | 기린 — 방향 확정 |
| 벤처스튜디오 설계 | AC 등록 요건 조사, 구조 설계 | 기린 — 의사결정 |
| 지원사업 탐색 | 창업·AI·문화 분야 공모사업 리스트 | 기린 — 지원 여부 결정 |
| 양조장 운영 | 교육/홍보/제품 문구 초안 | 기린 — 확인 후 사용 |
| 정산/행정 보조 | 정산 서류 초안 | 정직원 — 검증, 기린 — 승인 |

## 4. 에스컬레이션 규칙

### 사이드 프로젝트에서 AI가 단독 처리 불가 (초안 후 반드시 승인 요청)

| 항목 | 이유 | 질문 예시 |
|------|------|----------|
| 사단법인 공식 문서 발송 | 법인 명의 공식 문서 | "공문 초안입니다. 발송해도 될까요?" |
| 이사회/총회 안건 확정 | 법인 운영 의사결정 | "안건 초안입니다. 확인 후 공유 방향 알려주세요" |
| 교육 프로그램 공식 오픈 | 대외 발신, 참가자 모집 | "프로그램 안내문 초안입니다. 오픈해도 될까요?" |
| 계약/협약 관련 문서 | 법적 구속력 | "계약서 초안입니다. 검토 후 승인해 주세요" |

### AI가 자율 처리 가능한 것

| 항목 | 조건 |
|------|------|
| 회의록 초안 정리 | 사실 기반 정리만 |
| 지원사업 탐색 | 정보 수집 및 목록 정리만 |
| 홍보 문구/소개 문구 초안 | 기린 확인 전까지 초안 상태 유지 |
| 행정 서류 체크리스트 | 기존 공모 요강 기반 |
