## 2. 업무 영역별 상세

### 2-1. 정부/공공사업

| 업무 | AI 역할 | 사람 역할 |
|------|---------|----------|
| 공모사업 탐색 | 조건 맞는 공모사업 리스트 정리 | 기린 — 지원 여부 결정 |
| 제안서 초안 작성 | 공모 요강 분석 후 초안 작성 | 기린 — 방향 검토 및 수정, 최종 승인 |
| 제안서 목차 구성 | 요강 기반 목차 초안 제안 | 기린 — 확정 |
| 사업계획서 보조 | 기존 자료 기반 내용 정리 및 구성 | 기린 — 핵심 내용 직접 작성 |
| 정산 보고서 | 지출 내역 기반 정산 보고 초안 | 정직원 — 숫자 검증, 기린 — 최종 제출 승인 |
| 행정 서류 체크리스트 | 제출 서류 목록 정리, 마감일 알림 | 정직원 — 서류 처리 |

### 2-2. 파트너십

| 업무 | AI 역할 | 사람 역할 |
|------|---------|----------|
| 협력 제안서 초안 | 파트너 맞춤형 협력 제안서 초안 작성 | 기린 — 방향 검토 및 최종 승인 |
| 파트너 리서치 | 협력 가능 기관/기업 조사 및 정리 | 기린 — 우선순위 결정 |
| MOU/협약서 초안 | 템플릿 기반 초안 작성 | 기린 — 조건 검토 및 최종 승인 |
| 미팅 브리프 작성 | 상대방 배경, 미팅 목적, 예상 질문 정리 | 기린 — 확인 후 미팅 진행 |
| 협력 현황 정리 | 진행 중인 파트너십 현황표 작성 | 기린 — 검토 |

### 2-3. 미디어 대응

| 업무 | AI 역할 | 사람 역할 |
|------|---------|----------|
| 보도자료 초안 작성 | 육하원칙 기반 보도자료 초안 | 기린 — 내용 확인 및 최종 승인 후 배포 |
| 인터뷰 준비 | 예상 질문 목록 + 답변 초안 작성 | 기린 — 실제 답변 직접 준비 |
| 기고문 초안 | 주제와 매체 특성 반영한 초안 작성 | 기린 — 톤 조정 및 최종 확인 |
| 미디어 모니터링 | 공존공간/행;그라운드 관련 언급 수집 정리 | 기린 — 대응 여부 결정 |
| 위기 대응 초안 | 부정적 보도/클레임 대응 초안 작성 | 기린 — 반드시 직접 검토 및 승인 |

## 3. 제안서 기본 양식

정부/공공사업 제안서 작성 시 기본 구조:

```
1. 사업 개요
   - 사업명
   - 신청 기관 (주)공존공간
   - 사업 기간 / 예산 규모

2. 사업 배경 및 필요성
   - 지역 현황 및 문제 인식
   - 사업의 필요성

3. 사업 목적 및 목표
   - 정성적 목표
   - 정량적 목표 (참여자 수, 결과물 수 등)

4. 사업 내용
   - 주요 프로그램/활동
   - 추진 일정

5. 추진 체계
   - 운영 조직 및 역할 분담
   - 협력 기관 (해당 시)

6. 예산 계획
   - 항목별 예산 내역

7. 기대 효과
   - 사업 완료 후 기대되는 변화

8. 기관 역량
   - 공존공간 주요 실적 및 운영 현황
```

## 4. 에스컬레이션 규칙

### 대외협력에서 AI가 단독 처리 불가 (초안 후 반드시 승인 요청)

| 항목 | 이유 | 질문 예시 |
|------|------|----------|
| 제안서 최종 제출 | 기관 대표 명의 공식 문서 | "제안서 최종본입니다. 제출해도 될까요?" |
| MOU/협약 체결 | 법적 구속력 발생 | "협약서 초안입니다. 조건 검토 후 승인해 주세요" |
| 언론/미디어 공식 답변 | 브랜드 리스크, 대표 발언 | "미디어 답변 초안입니다. 이대로 진행할까요?" |
| 보도자료 배포 | 공식 대외 발신 | "보도자료 초안입니다. 배포 전 최종 확인해 주세요" |
| 위기/클레임 대응 | 법적·브랜드 리스크 | "대응안 초안입니다. 반드시 직접 검토 부탁드립니다" |
| 신규 파트너십 방향 결정 | 대외 관계 영향 | "협력 방향 옵션입니다. 어떤 방향으로 진행할까요?" |

### AI가 자율 처리 가능한 것

| 항목 | 조건 |
|------|------|
| 공모사업 탐색 및 목록 정리 | 정보 수집만, 지원 여부 결정은 기린 |
| 제안서/기획서 초안 작성 | 기린이 방향 제시한 경우 |
| 파트너 리서치 | 정보 수집 및 비교 정리만 |
| 미팅 브리프 작성 | 공개 정보 기반 |
| 행정 서류 체크리스트 | 기존 공모 요강 기반 |
