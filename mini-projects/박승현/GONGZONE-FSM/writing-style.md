# Writing Style — 문체 가이드

> 상위 문서: `main.md`
> 공문서 기준: 2025 개정 공문서 작성법 길라잡이 (한국공공언어진흥원, 2025. 1. 8.)

## 1. 상황별 톤 한눈에 보기

| 상황 | 톤 | 어미 예시 |
|------|-----|----------|

## 2. 공문서 작성 구조

### 2-1. 기본 양식 구조

```
[기관명]
수신  수신처
제목  문서 제목

1. (본문 내용)

붙임  첨부물명 1부.  끝.
```

### 2-2. 제목 작성 규칙

| 원칙 | 설명 | 예시 |
|------|------|------|
| 1안건 1기안 | 하나의 제목에 두 가지 안건 금지 | ❌ "행사 계획 및 대상자 공고" → ✅ 제목/붙임으로 분리 |
| 내용 포괄 | 본문 전체를 포괄하는 제목 | ❌ "활동비 운영" → ✅ "활동비 지원" |
| 명확성 | 기관명 등 구체적으로 표기 | ❌ "우리 기관 행사" → ✅ "(주)공존공간 행사" |

### 2-3. 항목 기호 순서

```
1., 2., 3.       ← 첫째 항목
  가., 나., 다.   ← 둘째 항목
    1), 2), 3)   ← 셋째 항목
      가), 나)   ← 넷째 항목
```

- 항목 기호와 내용 사이: 1칸 띄움
- 하위 항목: 상위 항목에서 2칸 오른쪽
- 항목이 하나뿐일 때: 항목 기호 생략

### 2-4. 붙임 표기

```
붙임  예산 내역서 1부.  끝.

붙임  1. 사업계획서 1부.
      2. 예산 내역서 1부.  끝.
```

- 붙임 다음에 1칸(한글 2타) 여백
- 여러 항목은 묶어서 표기하지 말 것 (이력서, 자기소개서 각 1부 ❌)

### 2-5. '끝.' 표기

- 본문이 끝났을 때: 마지막 줄 끝에 2칸 띄우고 **끝.**
- 붙임이 있을 경우: 붙임 마지막 줄 끝에 2칸 띄우고 **끝.**

## 3. 표기법

### 3-1. 글자

| 원칙 | 올바른 예 | 틀린 예 |
|------|----------|---------|
| 한글 표기 원칙 | 연구 개발(R&D) | R&D |
| 영문은 괄호 안에 | 업무 협약(MOU) | MOU |
| 숫자 표기 | 1층 | 1F |
| 기관명·국가명 붙여 쓰기 | 공존공간 | 공존 공간 |

### 3-2. 날짜

| 원칙 | 올바른 예 | 틀린 예 |
|------|----------|---------|
| 아라비아 숫자 + 마침표 | 2025. 2. 23. | 2025.2.23 / '25. 2. 23. |
| 기간 표기 | 2025. 2. 20.~2. 28. | 2025. 2. 20~28. |
| 날짜+요일 | 2025. 2. 23.(일) | 2025. 2. 23(일). |
| 연도만 쓸 때 | 2025년 | 2025. |

### 3-3. 시간

| 원칙 | 올바른 예 | 틀린 예 |
|------|----------|---------|
| 24시각제 | 09:00 / 13:00 | 오전 9시 / 오후 1시 |
| 10 미만은 0 넣기 | 09:00 | 9:00 |
| 분 없이 시만 | 09시 | 09:00시 |
| 시간 동안 | 09:00-12:00 | 09:00-12:00분까지 |
| 마감 표현 | 서류 마감: 18:00 | 서류 마감 시간: 밤 12시 |

### 3-4. 금액

| 구분 | 표기 방식 | 예시 |
|------|----------|------|
| 일반 문서 | 금OO원(금한글표기) | 금500,000원(금오십만원) |
| 계약서 | 일금OO원정(일금한글표기원정) | 일금500,000원정(일금오십만원정) |

- **금-원 (○)** / 일금-원정 (○)
- 일금-원 (✕) / 금-원정 (✕)

## 4. 표현법 5원칙

### 4-1. 사실성 — 있는 그대로 정확하게

- 육하원칙(누가/언제/어디서/무엇을/어떻게/왜)에 따라 작성
- 중첩어 피하기

| 틀린 표현 | 올바른 표현 |
|----------|-----------|
| 2월달 | 2월 |
| 기간 동안 | 기간에 |
| 새로 나온 신상품 | 신상품 |
| 미리 예측하다 | 예측하다 |
| 새로 신설하다 | 신설하다 |
| 피해를 입다 | 피해를 보다 |

### 4-2. 용이성 — 읽기 쉽게

- 외래어 대신 순화된 표현 사용

| 틀린 표현 | 올바른 표현 |
|----------|-----------|
| MOU 체결 | 업무 협약(MOU) 체결 |
| 스크린도어 설치 | 안전문 설치 |
| 옴부즈맨 | 민원 도우미 |
| 노미네이트 | 후보 지명 |
| 마일리지 | 이용 실적 |

### 4-3. 명확성 — 뜻이 분명하게

- 마감 표기는 시각까지 명시

| 틀린 표현 | 올바른 표현 |
|----------|-----------|
| 2025. 2. 28. 서류 접수 마감 | 2025. 2. 28. 18:00까지 서류 제출 마감 |
| 참석: 시장 및 시장 이하 15명 | 참석: 시장 포함 16명 |

### 4-4. 비고압성 — 위압감 없이

- 행동을 제약하는 표현 대신 정보를 안내하는 방식으로

| 틀린 표현 | 올바른 표현 |
|----------|-----------|
| 관계자 외 절대 출입 엄금 | 관계자만 드나들 수 있습니다 |
| 기한 내 필히 납부 바람 | 마감일 안에 내주시기 바랍니다 |
| 금연! | 이곳은 금연 구역입니다 |
| 입산 금지 | 입산이 제한되어 있습니다 |

### 4-5. 비표지성 — 편견 없이

- 우선순위 없으면 가나다순
- 특정 브랜드·학교·지역 이유 없이 부각 금지

| 틀린 표현 | 올바른 표현 |
|----------|-----------|
| 불우 이웃 돕기 | 이웃 돕기 |
| 참가자: 60명(남자 25, 여자 35) | 참가자: 60(여 35, 남 25) |

## 5. 헷갈리기 쉬운 용어 구별

| 용어 | 의미 | 구별 포인트 |
|------|------|-----------|
| 참고 | 살펴서 생각함 | 한 개 제시할 때 |
| 참조 | 비교하고 대조하여 봄 | 두 개 이상 제시할 때 |
| 주최 | 어떤 일을 기획함 | 기획에 초점 |
| 주관 | 어떤 일을 맡아서 시행함 | 시행에 초점 |
| 고시 | 계속적 효력이 있는 사항 알림 | 법적 구속력 있음 |
| 공고 | 일시적 사항 알림 | 법적 구속력 없음 |
| 기획 | 무엇을 할 것인가를 생각함 | 방향 설정 |
| 계획 | 어떻게 할 것인가를 구상함 | 구체적 실행안 |
| 이상 | 기준 수 포함, 그보다 큰 수 | 10 이상 → 10 포함 |
| 초과 | 기준 수 미포함, 그보다 큰 수 | 10 초과 → 11부터 |
| 이전 | 기준일 포함 | 2월 10일 이전 → 10일 포함 |
| 전 | 기준일 미포함 | 2월 10일 전 → 9일까지 |
| 이외 | 대상 포함하여 밖 | 행궁동 이외 3곳 (행궁동 포함) |
| 외 | 대상 포함하지 않고 밖 | 행궁동 외 2곳 (행궁동 제외) |
| 아래 | 관련 내용이 동일 공간에 위치 | 같은 문서 내 |
| 다음 | 관련 내용이 이어지는 공간에 위치 | 다음 페이지 이후 |

## 6. 공존공간 공문서 자주 쓰는 표현 모음

### 제안서·기획서 마무리 표현

```
위와 같이 사업을 추진하고자 하오니 승인하여 주시기 바랍니다.
위와 같이 보고드리오니 검토하여 주시기 바랍니다.
아래와 같이 계획을 수립하였음을 알려드립니다.
```

### 붙임 문구 예시

```
붙임  1. 사업계획서 1부.
      2. 예산 내역서 1부.  끝.

붙임  협약서 초안 1부.  끝.
```

### 관련 근거 표기

```
1. 관련: OO사업단-123(2025. 2. 23.)「○○ 사업 공모 요강」

1. 관련 근거
   가. OO기관-12(2025. 1. 10.)「○○ 지침」
   나. OO기관-35(2025. 2. 1.)「○○ 안전수칙」
```

## 7. AI가 공문서 작성 시 체크리스트

공문서 초안 작성 후 아래 항목을 점검할 것.

- [ ] 제목이 내용 전체를 포괄하는가?
- [ ] 날짜 표기가 `2025. 2. 23.` 형식인가?
- [ ] 시간 표기가 `09:00` 형식인가?
- [ ] 금액 표기가 `금OOO원(금한글)` 형식인가?
- [ ] 외래어에 괄호로 한글 설명을 병기했는가?
- [ ] 중첩어, 비고압적 표현, 모호한 표현은 없는가?
- [ ] 붙임이 있으면 끝에 `끝.` 표기가 있는가?
- [ ] 기린 승인 필요 사항인 경우, 발송 전 확인 요청했는가?
