## 1. 기획 업무 범위

공존공간과 행;그라운드의 기획 업무는 크게 3가지로 나뉨.

| 기획 유형 | 설명 | 주요 브랜드 |
|-----------|------|------------|
| 행사 기획 | 지역 문화행사, 프로그램, 이벤트 기획 및 운영 | 공존공간, 행;그라운드 |
| 콘텐츠 기획 | 온/오프라인 콘텐츠 방향 설정 및 소재 발굴 | 전 브랜드 |
| 브랜드 개발 | 브랜드 정체성, 스토리, 신규 서비스 기획 | 공존공간, 행;그라운드, 사이드 프로젝트 |

## 2. 업무 영역별 상세

### 2-1. 행사 기획

| 업무 | AI 역할 | 사람 역할 |
|------|---------|----------|
| 행사 기획안 초안 작성 | 기획서 양식 기반 초안 작성 | 기린 — 방향 확정, 담당자 — 실행 |
| 타임라인 / 진행 순서 구성 | 흐름 초안 제안 | 기린 — 검토 및 조정 |
| 행사 예산안 작성 | 항목별 예산 초안 구성 | 정직원 — 숫자 검증, 기린 — 최종 승인 |
| 행사 체크리스트 | 단계별 체크리스트 생성 | 담당자 — 실행 |
| 행사 결과 보고서 | 참가자 수, 피드백 등 데이터 기반 초안 작성 | 기린 — 검토 |

### 2-2. 콘텐츠 기획

| 업무 | AI 역할 | 사람 역할 |
|------|---------|----------|
| 월별 콘텐츠 캘린더 작성 | 소재 제안 + 날짜 배분 초안 | 기린 — 방향 승인 |
| 콘텐츠 소재 발굴 | 트렌드 리서치, 지역 이슈 수집 | 기린 — 소재 선택 |
| 콘텐츠 브리프 작성 | 소재별 작성 방향, 핵심 메시지, 타깃 정리 | 기린 — 확인 후 팀 전달 |
| 시즌별 기획 제안 | 명절, 계절, 지역 이슈 연계 기획안 초안 | 기린 — 채택 여부 결정 |

### 2-3. 브랜드 개발

| 업무 | AI 역할 | 사람 역할 |
|------|---------|----------|
| 브랜드 스토리 정리 | 기존 자료 기반 스토리 초안 정리 | 기린 — 방향 확인 및 최종 수정 |
| 신규 서비스 기획안 | 아이디어 구체화, 기획서 초안 작성 | 기린 — 방향성 결정, 코워커 — 논의 |
| 경쟁사/시장 리서치 | 유사 사례 수집 및 비교 분석 | 기린 — 인사이트 도출 |
| 브랜드 슬로건/네이밍 | 옵션 다수 제안 | 기린 — 최종 선택 |

## 3. 기획 문서 기본 양식

기획서 작성 시 아래 구조를 기본값으로 사용:

```
1. 개요
   - 행사/프로젝트명
   - 목적 (한 줄)
   - 일시 / 장소
   - 주관 / 담당

2. 배경 및 목표
   - 왜 하는가
   - 무엇을 얻고 싶은가

3. 대상
   - 주요 타깃

4. 내용 구성
   - 프로그램 흐름 / 콘텐츠 구성

5. 예산
   - 항목별 예산표

6. 일정
   - 준비 → 실행 → 마무리 타임라인

7. 담당자 배분
   - 역할별 담당자
```

## 4. 에스컬레이션 규칙

### 기획에서 AI가 단독 처리 불가 (초안 후 반드시 승인 요청)

| 항목 | 이유 | 질문 예시 |
|------|------|----------|
| 외부 파트너와의 협업 기획 | 대외 관계 영향 | "파트너 협업 기획안 초안입니다. 진행 방향 확인해 주세요" |
| 예산이 포함된 기획안 확정 | 재무 연동 사항 | "예산안 포함된 기획서입니다. 최종 승인해 주세요" |
| 신규 브랜드 방향 결정 | 브랜드 정체성 영향 | "브랜드 방향 옵션 3가지 제안드립니다. 어떤 방향으로 갈까요?" |
| 공식 발표 전 기획 확정 | 대외 발신 문서 | "발표용 기획안 초안입니다. 발표해도 될까요?" |

### AI가 자율 처리 가능한 것

| 항목 | 조건 |
|------|------|
| 기획서 초안 작성 | 기린이 방향 제시한 경우 |
| 유사 사례 리서치 | 정보 수집 및 정리만 |
| 체크리스트 / 타임라인 초안 | 기존 승인된 포맷 기반 |
| 콘텐츠 소재 제안 | 아이디어 나열만, 채택 결정은 기린 |

| 분기별 | 분기 행사/프로그램 기획 검토 | AI 초안 → 기린 + 코워커 논의 |
| 시즌 전 (4주 전) | 시즌 기획안 초안 작성 | AI → 기린 검토 |
