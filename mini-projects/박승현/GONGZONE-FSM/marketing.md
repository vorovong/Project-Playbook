## 3. 업무 영역별 상세

### 3-1. SNS 콘텐츠 제작

| 업무 | AI 역할 | 사람 역할 |
|------|---------|----------|
| 인스타그램 포스트 캡션 작성 | 브랜드 톤에 맞는 캡션 초안 작성 | 기린 — 확인 및 최종 수정 후 업로드 |
| 릴스/스토리 기획 | 소재 제안 + 구성 초안 | 기린 — 채택 여부 결정 |
| 해시태그 세트 구성 | 브랜드·지역·주제별 해시태그 조합 제안 | 기린 — 확정 |
| 콘텐츠 소재 발굴 | 지역 이슈, 트렌드, 시즌 소재 수집 | 기린 — 소재 선택 |
| 이미지 설명 문구 | 사진 설명 및 ALT 텍스트 초안 | 기린 — 확인 |

### 3-2. 홍보

| 업무 | AI 역할 | 사람 역할 |
|------|---------|----------|
| 행사 홍보 문구 작성 | 채널별 홍보 문구 초안 (SNS, 카카오, 문자) | 기린 — 확인 후 발송 |
| 홍보 일정 계획 | D-30/14/7/1 홍보 타임라인 초안 | 기린 — 승인 |
| 카드뉴스 텍스트 구성 | 슬라이드별 텍스트 초안 | 기린 — 확인, 디자인은 별도 |
| 지역 커뮤니티 홍보 문구 | 맘카페, 지역 단톡방용 문구 초안 | 기린 — 확인 후 직접 게시 |

### 3-3. 고객 소통

| 업무 | AI 역할 | 사람 역할 |
|------|---------|----------|
| 문의 응대 템플릿 작성 | 자주 묻는 질문별 답변 템플릿 작성 | 기린 — 확인 후 사용 |
| 리뷰/피드백 답글 초안 | 긍정/부정 리뷰별 답글 초안 | 기린 — 확인 후 게시 |
| 입점팀 소개 콘텐츠 | 인터뷰 기반 소개 콘텐츠 초안 | 기린 — 입점팀 확인 후 게시 |
| 고객 피드백 정리 | 문의/리뷰 데이터 취합 및 패턴 정리 | 기린 — 개선 방향 결정 |

## 4. 브랜드별 콘텐츠 톤 가이드

### 공존공간

| 항목 | 내용 |
|------|------|
| 핵심 메시지 | "가장 가까운 곳에서 만드는 일상" |
| 톤 | 진중하되 따뜻함. 지역과 사람 이야기 중심 |
| 피해야 할 것 | 과도한 홍보성 문구, 화려한 수식어 |
| 자주 쓰는 키워드 | 행궁동, 로컬, 수원, 공존, 지역경영 |

### 행;그라운드

| 항목 | 내용 |
|------|------|
| 핵심 메시지 | 함께 일하는 공간, 연결되는 사람들 |
| 톤 | 활기차고 실용적. 공간의 일상감 강조 |
| 피해야 할 것 | 딱딱한 기업 홍보 투, 과도한 이벤트성 표현 |
| 자주 쓰는 키워드 | 쉐어오피스, 코워킹, 행궁동, 입점, 커뮤니티 |

## 5. 에스컬레이션 규칙

### 마케팅에서 AI가 단독 처리 불가 (초안 후 반드시 승인 요청)

| 항목 | 이유 | 질문 예시 |
|------|------|----------|
| 공식 채널 게시물 업로드 | 대외 발신, 브랜드 이미지 영향 | "포스트 초안입니다. 업로드해도 될까요?" |
| 부정적 리뷰/클레임 답글 | 브랜드 리스크 | "답글 초안입니다. 이대로 게시할까요?" |
| 유료 광고 집행 | 예산 집행 | "광고 소재 초안입니다. 집행 전 확인해 주세요" |
| 입점팀 관련 콘텐츠 게시 | 제3자 관련, 사전 동의 필요 | "입점팀 소개 콘텐츠입니다. 게시 전 해당 팀 확인이 필요합니다" |

### AI가 자율 처리 가능한 것

| 항목 | 조건 |
|------|------|
| 캡션/문구 초안 작성 | 기린이 소재 또는 방향 제시한 경우 |
| 해시태그 세트 제안 | 옵션 제안만, 확정은 기린 |
| 콘텐츠 소재 수집 | 아이디어 나열만 |
| 문의 응대 템플릿 초안 | 기린 확인 후 사용 |
| 월별 콘텐츠 캘린더 초안 | 기린 승인 전까지는 초안 상태 유지 |
