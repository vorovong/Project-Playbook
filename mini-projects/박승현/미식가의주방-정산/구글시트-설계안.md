# 미식가의 주방 정산 — 구글시트 설계안

---
created: 2026-03-07
status: 초안
purpose: 파편화된 엑셀 136개 시트 → 구글시트 2개 시트로 단순화
---

## 변경 전 vs 변경 후

| | 변경 전 | 변경 후 |
|---|---|---|
| **정산 방식** | 공존공간이 매출에서 수수료만 빼고 지급 → 미식가가 임대료/관리비/공과금을 다시 입금 | 매출에서 **전부 차감** → 최종 금액만 지급 |
| **미수금 발생** | 미식가가 재입금 안 하면 미수금 (현재 4천만원) | 구조적으로 미수금 거의 없음 (현금매출분만) |
| **파일 구조** | 엑셀 3~4개 파일, 136개+ 시트 | 구글시트 1개 파일, 2개 시트 |
| **입력** | 매주 새 시트 생성 + 수동 계산 | 매주 1줄 입력 → 자동 계산 |

## 계약 기준 (현행, 변경 예정 부분은 ★ 표시)

| 항목 | 금액/비율 | 비고 |
|---|---|---|
| 고정 임대료 | 월 3,000,000 (VAT별도) | 매월 25일 |
| 카드 수수료 | 매출의 15% (VAT별도) | 카드수수료+기타관리비 포함 |
| 현금 수수료 | 매출의 13% (VAT별도) | |
| 수수료 상한 | 매출 1.2억 기준 | |
| 관리비 | 월 55,000 + 정수기 15,900 = 70,900 | |
| 공과금 | 수도광열비 사용량의 95% | 가스/전기/수도 |
| 캡스/인터넷 | 월 50,000 (VAT별도) | |
| 정화조 청소 | 연 2회, 25일 정산 | |
| 재생전골 연계 회식 | 해당 매출의 5% (VAT별도) | 발생 시만 |
| ★ 정산 흐름 | **매출에서 전부 차감 후 최종 금액만 지급** | 계약 변경 예정 |

## 시트 구조

### 시트1: 정산 입력

**한 줄 = 한 주. 1년 = 최대 60줄.**

#### 컬럼 구조

```
[사용자 입력 = 🟡 노란색 배경]  [자동 계산 = ⬜ 회색 배경]
```

| 컬럼 | 이름 | 입력/자동 | 설명 |
|---|---|---|---|
| A | **월** | 🟡 입력 | 1~12 |
| B | **주차** | 🟡 입력 | 1~5 |
| C | **시작일** | 🟡 입력 | 해당 주 월요일 |
| D | **종료일** | 🟡 입력 | 해당 주 일요일 |
| E | **카드매출** | 🟡 입력 | POS 기준 카드 매출액 |
| F | **현금매출** | 🟡 입력 | POS 기준 현금 매출액 |
| G | 총매출 | ⬜ 자동 | =E+F |
| H | 카드수수료(15%) | ⬜ 자동 | =E×0.15 |
| I | 현금수수료(13%) | ⬜ 자동 | =F×0.13 |
| J | 수수료 합계 | ⬜ 자동 | =H+I |
| K | 수수료 VAT(10%) | ⬜ 자동 | =J×0.1 |
| L | **고정임대료(주간배분)** | ⬜ 자동 | =3,000,000÷해당월주차수 |
| M | 임대료 VAT | ⬜ 자동 | =L×0.1 |
| N | **관리비(주간배분)** | ⬜ 자동 | =70,900÷해당월주차수 |
| O | **공과금** | 🟡 입력 | 해당 월 첫째 주에만 입력 (95% 적용된 금액) |
| P | **캡스/인터넷** | ⬜ 자동 | 첫째주만 50,000 (=IF(B="1주",50000,0)) |
| Q | **기타비용** | 🟡 입력 | 정화조, 회식 5% 등 발생 시 |
| R | **총 차감액** | ⬜ 자동 | =J+K+L+M+N+O+P+Q |
| S | **미식가 지급액** | ⬜ 자동 | =E-R (카드매출에서 전부 차감) |
| T | **현금매출 수령액** | ⬜ 자동 | =F-(I+I×0.1) (현금매출 - 현금수수료-VAT) |
| U | **비고** | 🟡 입력 | 특이사항 메모 |

#### 입력 예시 (2026년 3월 1주차)

| 월 | 주차 | 시작 | 종료 | 카드매출 | 현금매출 | 총매출 | 카드15% | 현금13% | 수수료합 | VAT | 임대료 | 임대VAT | 관리비 | 공과금 | 캡스 | 기타 | 총차감 | 지급액 | 현금수령 |
|---|---|---|---|---|---|---|---|---|---|---|---|---|---|---|---|---|---|---|---|
| 3 | 1 | 3/3 | 3/9 | 18,000,000 | 200,000 | 18,200,000 | 2,700,000 | 26,000 | 2,726,000 | 272,600 | 750,000 | 75,000 | 17,725 | 2,000,000 | 50,000 | 0 | 5,891,325 | 12,108,675 | 171,400 |

**사용자가 넣는 숫자: 카드매출, 현금매출, 공과금, 기타 = 최대 4개**
**나머지 15개 컬럼은 전부 자동 계산**

#### 월 구분 & 소계

- 매월 마지막 주 아래에 **월 소계 행** 자동 삽입
- 월 소계: 해당 월의 매출/수수료/차감/지급 합계
- 연간 합계: 맨 하단에 12개월 총합

#### 미수금 관리 섹션 (시트1 하단)

기존 미수금(기물양도비 등) 관리를 위한 별도 테이블:

| 컬럼 | 내용 |
|---|---|
| 미수금 항목 | 기물양도비, 기존 공과금 미수 등 |
| 원금 | 최초 미수금 |
| 차감 누적 | 정산에서 차감한 합계 |
| 잔액 | 원금 - 차감 누적 |
| 비고 | 차감 회차, 날짜 등 |

### 시트2: 대시보드

#### 상단 — KPI 카드 (4개)

```
┌─────────────┐ ┌─────────────┐ ┌─────────────┐ ┌─────────────┐
│  이번달 매출  │ │ 이번달 수익  │ │  미수금 잔액  │ │ 공과금 누적  │
│ 73,076,800  │ │ 12,450,000  │ │ 36,867,242  │ │  2,023,812  │
│   ▲ +5.2%   │ │   ▲ +3.1%   │ │   ▼ -15%    │ │   ─ 동일    │
└─────────────┘ └─────────────┘ └─────────────┘ └─────────────┘
```

#### 차트 1 — 월별 매출 추이 (막대 그래프)

- X축: 월 (1~12)
- Y축: 금액
- 막대: 카드매출 (파랑) + 현금매출 (초록) 누적 막대
- 선: 전년도 동월 매출 (회색 점선, 2025년 데이터 있으므로)

#### 차트 2 — 공존공간 수익 구성 (도넛 차트)

- 카드수수료 (15%)
- 현금수수료 (13%)
- 고정임대료
- 관리비
- 공과금

#### 차트 3 — 주간 매출 추이 (꺾은선 그래프)

- X축: 주차 (1월1주 ~ 12월5주)
- Y축: 주간 총매출
- 선: 올해 매출 (파랑), 전년도 (회색 점선)

#### 차트 4 — 미수금 잔액 추이 (영역 그래프)

- X축: 월
- Y축: 미수금 잔액
- 목표: 0원까지 감소하는 추이

#### 하단 — 월별 정산 요약 테이블

|---|---|---|---|---|---|---|---|---|---|---|

## 추후 과제 (자동화)

> 승현님 요청: "숫자, 영수증 사진만 입력하면 알아서 정산이 시트로 들어가는 것"

| 단계 | 내용 | 도구 |
|---|---|---|
| 1단계 (지금) | 구글시트 수동 입력 + 자동 계산 | 구글시트 |
| 2단계 | POS 데이터 자동 연동 → 시트 자동 입력 | Apps Script / Zapier |
| 3단계 | 영수증 사진 → OCR → 자동 입력 | Google Vision API / 슈파 |
| 4단계 | 정산 보고서 자동 생성 + 알림 | Apps Script + 카톡/이메일 |
