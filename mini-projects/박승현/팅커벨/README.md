## 팅커벨이란?

팅커벨은 박승현(기린)의 **개인 AI 인프라**입니다.
특정 AI(클로드, GPT 등)에 종속되지 않고, 마크다운 파일로 구조화된 "나"의 시스템.

### 기존 FSM과의 관계

```
팅커벨 (최상위 — 나를 아는 AI)
├── TELOS/              ← 나는 누구인가 (모든 판단의 기준)
├── MEMORY/             ← 기억 시스템 (세션을 넘어 학습)
├── ALGORITHM/          ← 의사결정 루프 (일관된 사고 회로)
├── GIRIN FSM/          ← 개인 자동화 (기존)
└── GONGZONE FSM/       ← 회사 운영 (기존)
```
