> 승현님이 던져주시는 글, 생각, 메모, 링크를 **원본 그대로 보관**하는 곳.
> 팅커벨이 사서로서 분류하고, 연결하고, 빈 곳을 찾아 질문한다.

## 운영 원칙

| 원칙 | 설명 |
|------|------|
| **원본 보존** | 승현님이 던진 글/생각을 최대한 그대로 저장. 요약은 별도 |
| **조각이 먼저, 구조는 나중에** | 카테고리는 조각이 10개 이상 쌓인 후에 만듦 |
| **승현님 맥락 기록** | 왜 이걸 공유했는지, 대화에서 나온 코멘트도 함께 |
| **에센스 연결** | 관련 에센스/TELOS가 있으면 태그로 연결 |
