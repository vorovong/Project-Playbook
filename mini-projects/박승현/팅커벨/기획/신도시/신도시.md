## 스핀오프 배경

순서가 중요하다. 처음부터 계획한 분리가 아니다:
1. 글로컬상권 2년간 현장에서 부딪힘
2. "상권 지원 = 간접 지원"이 아니라 "파트너로서 직접 효과"가 필요하다는 것을 깨달음
3. 이 깨달음이 공존공간의 사업 구조 자체를 바꾸게 만듦
4. 그래서 신도시라는 새 법인으로 재편·스핀오프

## 회사 개요

| 항목 | 내용 |
|------|------|
| 역할 | 인력 발굴·육성·투자 |
| 기능 1 | 로컬벤처스튜디오 — 창업가 발굴·보육·투자연계 (AC 등록 필요) |
| 기능 2 | 슈파(SUPA) — 소상공인 맞춤 AI 자동화 |
| 연계 자원 | LIPS I(5억) + LIPS II(2억), 새빛펀드 AC, TIPS |
| 타임라인 | 2026 상반기 설립 → 하반기 LIPS II + 슈파 베타 → 2027~ 새빛펀드 |

## 지분 구조 (2026.02.26 회의)

- 박승현 개인 60%+ / 공존공간 법인 40%-
- 이유: 수원 로컬 정체성 유지 + 향후 VC 투자 조건 충족
- CSP와의 연결 재무 구조도 고려 대상

## 자본금 로드맵 (4단계)

1. 설립 시 1,000만원 (최소 자본)
2. 유상증자 → 1.5억 (LIPS 연계)
3. 무상증자 (실적 기반)
4. 추가 증자 2~3억 (새빛펀드/투자 연계)

## 슈케이크하우스 1층 활용

실사 진행 중 — 신도시 또는 슈파 오피스/쇼룸 가능성
