## 본질

**가장 가까운 곳에서 상인의 프로덕트와 서비스를 콘텐츠로 만들고, 그것이 모여 상권이 되는 구조를 설계·운영한다.**

15년간 직접 해보면서 증명한 것:
- 카페, 게스트하우스, 양조장, 식당을 직접 운영 → 프로덕트가 콘텐츠가 된다
- 콘텐츠가 되면 산 꼭대기까지 찾아온다 → 상권이 된다
- 민간이 주도하면 정부 사업이 끝나도 남는다

---

## 증명된 것 (글로컬상권창출팀 2년)

2024.06 ~ 2025.12, 총 사업비 최대 155억 원.

| 지표 | Before | After | 의미 |
|------|--------|-------|------|
| 외국인 소비액 | 14억 | 29억 (+107%) | 양보다 질 — 방문객 +40%인데 소비액은 +107% |
| 소상공인 협업 | 20회 | 142회 (+710%) | 상인이 스스로 참여하는 구조가 작동 |
| 창업가 발굴 | 0명 | 44명 | 발굴 역량 입증 |
| 1인당 소비액 | 9,333원 | 13,810원 (+48%) | 고부가가치 전환 |
| 대기업 협업 | 없음 | 롯데, 오뚜기, 경희대, 아주대 | 비예산 자생적 협업 발생 |

핵심: 행궁동을 **고부가가치 관광·로컬경제 거점**으로 전환. "양보다 질" 성장 모델.

## 현재 구조

```
박승현 (기획자, 개인)
├── (주)공존공간 ─── 공간·문화·상권기획 운영
│                    커뮤니티 REITs·BTO (미래)
│
├── (주)신도시 ───── 인력 발굴·육성·투자 (스핀오프)
│   ├── 로컬벤처스튜디오 (AC)
│   └── 슈파(SUPA) — 소상공인 AI SaaS
│
└── (사)행궁동행 ─── 지역 자율 운영 구조 (개인 자격 기획자로 참여)
```

| 항목 | 내용 |
|------|------|

| 항목 | 내용 |

| 타임라인 | 2026 상반기 발족 |

## 성장 경로

### 공간 확장

```
행궁동 (증명 완료)
  → 남문시장 (백년시장 40억, 2025.12 선정)
    → 수원 전역
      → 타 지역 (화성 등)
```

### 사업 확장

```
상권기획 (공존공간 기존 역량)
  → 창업가 발굴·투자 (신도시/로컬벤처스튜디오)
    → AI SaaS (슈파)
      → 부동산 (REITs/BTO)
```

### 슈파(SUPA) 확장

```
수원 소상공인 시범 운영
  → 수원 전역
    → 전국 소상공인
```

## 슈파(SUPA) — 별도 심화 필요

현재 파악된 것:
- SNS·리뷰·매출리포트·예약 자동화
- 회원 증가 → 운영비 분산 → 구독료 인하
- 수익 일부 지역 상권 기금 환원

아직 정리 안 된 것:
- [ ] 구체적 AI 기능 범위와 우선순위
- [ ] 가격 모델 (월 구독? 건당?)
- [ ] 경쟁 제품 대비 차별점
- [ ] MVP 범위 정의
- [ ] 공존공간 2년 현장 데이터를 슈파에 어떻게 활용할지

→ 슈파는 별도 전략 문서로 분리해서 깊이 파야 할 수도 있음.
