## 본질

**가장 가까운 곳에서 상인의 프로덕트와 서비스를 콘텐츠로 만들고, 그것이 모여 상권이 되는 구조를 설계·운영한다.**

15년간 직접 해보면서 증명한 것:
- 카페, 게스트하우스, 양조장, 식당을 직접 운영 → 프로덕트가 콘텐츠가 된다
- 콘텐츠가 되면 산 꼭대기까지 찾아온다 → 상권이 된다
- 민간이 주도하면 정부 사업이 끝나도 남는다

## 에센스 — 각 기획의 근본

각 기획서와 사업의 근본 목적. 청중별 프레이밍이 달라져도 이 에센스는 변하지 않는다.

### 1. 글로컬상권창출팀 — "콘텐츠의 총합이 상권이다"의 증명

> **에센스**: 콘텐츠의 총합이 상권이다. 그리고 상권 지원의 패러다임이 바뀌어야 한다.

두 겹의 에센스가 있다:

**첫째, 명제의 증명** — "콘텐츠의 총합이 상권이다"
- 방문객 +40%인데 소비액 +107% → 콘텐츠가 있는 상권은 **질적 성장**을 한다
- 소상공인 협업 20회 → 142회 → 상인이 스스로 콘텐츠를 만들기 시작했다
- 대기업이 먼저 찾아옴 → 콘텐츠가 모이면 생태계가 자생한다

**둘째, 패러다임 전환** — 이 증명이 가리키는 방향
- 기존 상권 지원 = 교육, 컨설팅 등 **간접 지원** → 효과 제한적
- 앞으로의 상권 지원 = 슈파처럼 **파트너가 돼서 일을 같이 하거나 줄여주는 직접적 효과**
- 글로컬상권에서 변화한 상권의 본질이 이 방향을 가리킴

### 2. 신도시 스핀오프 — 현장에서 깨달은 것이 사업 구조를 바꿨다

> **에센스**: 글로컬상권 사업을 통해 변화한 현실을 깨닫고, 공존공간의 사업을 재편하여 스핀오프한 것.

순서가 중요하다. 처음부터 계획한 분리가 아니다:
1. 글로컬상권 2년간 현장에서 부딪힘
2. "상권 지원 = 간접 지원"이 아니라 "파트너로서 직접 효과"가 필요하다는 것을 깨달음
3. 이 깨달음이 공존공간의 사업 구조 자체를 바꾸게 만듦
4. 그래서 신도시라는 새 법인으로 재편·스핀오프

- 로컬벤처스튜디오 = 44명 발굴 경험에서 나온 "사람을 키우는 사업"
- 슈파 = 142회 협업 경험에서 나온 "파트너로서 직접 효과를 주는 기술"
- **회의에서 드러난 것**: 수원 로컬 정체성을 지키면서 VC 투자를 받을 수 있는 구조 설계 (→ 지분·자본금 섹션 참고)

### 3. 수원형 모두의창업 — 소상공인을 보는 시선의 전환

> **에센스**: 소상공인은 '보호 대상'이 아니라 '도전하고 성장하는 주체'다. 그런 사람들이 만드는 콘텐츠가 상권이 된다.

기존 정부의 시선:
- 소상공인 = **보호 대상** → 교육, 컨설팅, 지원금

승현님이 보는 시선:
- 로컬크리에이터, 창업가 = **도전과 성장의 주체** → 이 사람들이 상권을 만들고 콘텐츠를 만든다
- 창의가 담긴 도전과 시도가 담긴 곳이 상권이어야 한다
- 이건 글로컬상권의 "콘텐츠의 총합이 상권이다"와 직접 연결됨

정책 프레임으로서:
- 4대 축: 지역밀착 AC + 중기부 연계 + 시 자원 활용 + 축제(락성연)
- 민간 CMO 사단법인 = 행정이 바뀌어도 남는 민간 주도 구조
- 테크(스타트업) + 로컬(소상공인) 균형 = 전국에 없는 모델
- **주의**: 이건 정책 제안 프레임이지, 공존공간의 사업 자체는 아님. 정책이 바뀌어도 신도시·슈파는 독립적으로 작동해야 함.

### 4. 슈파(SUPA) — 각 가게의 고유한 문제를 AI로 해결

> **에센스**: "모든 소상공인에게 같은 도구"가 아니라, **각 소상공인의 고유한 문제를 맞춤 자동화**한다.

- 시장 보고에서는 "AI SaaS"로 표현했지만, 회의에서 드러난 실체는 다름
- PT 사인 시스템처럼 가게마다 다른 문제를 50~100만원 단위로 해결
- **차별점의 근본**: 공존공간이 2년간 142회 협업하며 쌓은 "각 가게가 뭘 필요로 하는지 아는 현장 지식"
- 이 현장 지식이 없으면 그냥 또 하나의 generic SaaS → 경쟁 불가
- (에센스 심화는 하단 슈파 섹션 참고)

### 에센스 간의 관계

```
글로컬상권 — "콘텐츠의 총합이 상권이다" 증명 + 패러다임 전환 깨달음
  ↓ 이 깨달음이
신도시 — 공존공간 사업 재편의 동력. 현장에서 배운 것이 구조를 바꿈
  ↓ 신도시 안에서
슈파 — "파트너로서 직접 효과"를 기술로 구현. 현장 지식이 무기
  ↓ 이 전체의 철학이
모두의창업 — "보호 → 도전과 성장"이라는 시선 전환의 정책 프레임
```

**핵심**: 네 가지는 하나의 이야기다. "콘텐츠가 상권이 되고, 그 상권을 만드는 건 도전하는 사람이고, 그 사람의 파트너가 되는 것이 슈파이고, 이 전체를 가능하게 하는 사업 구조가 신도시다."

## 증명된 것 (글로컬상권창출팀 2년)

2024.06 ~ 2025.12, 총 사업비 최대 155억 원.

| 지표 | Before | After | 의미 |
|------|--------|-------|------|
| 외국인 소비액 | 14억 | 29억 (+107%) | 양보다 질 — 방문객 +40%인데 소비액은 +107% |
| 소상공인 협업 | 20회 | 142회 (+710%) | 상인이 스스로 참여하는 구조가 작동 |
| 창업가 발굴 | 0명 | 44명 | 발굴 역량 입증 |
| 1인당 소비액 | 9,333원 | 13,810원 (+48%) | 고부가가치 전환 |
| 대기업 협업 | 없음 | 롯데, 오뚜기, 경희대, 아주대 | 비예산 자생적 협업 발생 |

핵심: 행궁동을 **고부가가치 관광·로컬경제 거점**으로 전환. "콘텐츠의 총합이 상권이다"를 숫자로 증명.
그리고 이 과정에서 깨달은 것: 교육·컨설팅 같은 간접 지원이 아니라, **파트너로서 직접 효과를 주는 형태**로 상권 지원이 바뀌어야 한다.

## 슈파(SUPA) — 에센스 심화

### 시장 보고 vs 회의에서 드러난 실체

| 구분 | 시장 보고 (대외용) | 회의 (내부 실체) |
|------|-------------------|-----------------|
| 포지셔닝 | 소상공인 AI SaaS | 소상공인 맞춤 자동화 |
| 서비스 형태 | SNS·리뷰·매출리포트·예약 자동화 | 가게별 고유 문제 해결 (PT 사인 시스템 등) |
| 가격대 | 구독료 (미정) | 50~100만원 단위 커스텀 자동화 |
| 차별점 | 회원 증가 → 운영비 분산 | 공존공간의 현장 지식 = 진입 장벽 |

### 왜 "맞춤 자동화"가 더 정확한가

- generic SaaS는 이미 시장에 많음 (배민, 네이버, 캐시노트 등)
- 공존공간의 진짜 자산 = **"이 가게가 뭘 필요로 하는지 아는 것"**
- 142회 협업에서 쌓인 현장 지식 → 각 가게에 맞는 자동화 설계 가능
- 예: PT 사인 시스템 = 특정 가게의 특정 문제를 AI로 해결한 사례

### 두 가지 모델의 공존 가능성

```
Layer 1: 공통 플랫폼 (SNS·리뷰·매출 — 구독형, 낮은 가격)
  ↑ 이것으로 소상공인 접점 확보

Layer 2: 맞춤 자동화 (가게별 커스텀 — 프로젝트형, 50~100만원)
  ↑ 이것이 진짜 차별점이자 수익원
```

### "파트너"의 두 가지 의미 (2026-03-02 승현님 직접 정의)

1. **브랜딩** — 사장님들에게 "도구"가 아니라 "파트너"라는 친근한 이미지. 위에서 내려다보는 지원이 아니라 옆에서 같이 하는 관계.
2. **실질적 효과** — AI를 활용해서 사장님들의 잡무를 확 줄여주는 것. 세무, 마케팅 같은 것부터 시작. 구체적인 잡무 목록은 슈파 프로젝트 하면서 발굴.

### 가격 전략 — "무료/소액 → 유저수 → 데이터/피지컬 AI" (2026-03-02 승현님 직접 정의)

> **핵심**: 서비스 이용료는 **무료거나 소액**. AI 생태계에서는 **유저수 확보가 더 중요**하다.

수익 모델의 본질:
- 서비스 자체로 돈을 버는 게 아님
- 유저수 확보 → **데이터** 축적 → 데이터 기반 수익
- 유저수 확보 → **피지컬 AI** (로봇 등) 연결 → 하드웨어 수익
- 이건 기존 SaaS "구독료"와 근본적으로 다른 접근

- 슈파: 에센스 자체는 유지, 슈파 프로젝트에서 계속 심화 예정

**배운 것**: 승현님이 직접 교정해주면 에센스의 정확도가 확 올라간다. 이 과정을 반복할수록 팅커벨의 추론 신뢰도도 올라감.
