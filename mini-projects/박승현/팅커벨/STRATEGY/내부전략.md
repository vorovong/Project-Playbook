## 본질

**가장 가까운 곳에서 상인의 프로덕트와 서비스를 콘텐츠로 만들고, 그것이 모여 상권이 되는 구조를 설계·운영한다.**

---

## 에센스 — 각 기획의 근본

### 1. 글로컬상권창출팀 — "콘텐츠의 총합이 상권이다"의 증명

> 콘텐츠의 총합이 상권이다. 그리고 상권 지원의 패러다임이 바뀌어야 한다.
> 간접 지원(교육, 컨설팅) → **파트너로서 직접 효과**

→ 상세: `기획/공존공간.md`

### 2. 신도시 스핀오프 — 현장에서 깨달은 것이 사업 구조를 바꿨다

> 글로컬상권 2년 → "파트너로서 직접 효과"가 필요하다는 깨달음 → 공존공간 사업 재편 → 신도시 스핀오프

→ 상세: `기획/신도시/신도시.md`

### 3. 수원형 모두의창업 — 소상공인을 보는 시선의 전환

> 소상공인은 '보호 대상'이 아니라 '도전하고 성장하는 주체'다.
> 정책 제안 프레임이지, 공존공간의 사업 자체는 아님.

→ 상세: `기획/공존공간.md`

### 4. 슈파(SUPA) — 각 가게의 고유한 문제를 AI로 해결

> "모든 소상공인에게 같은 도구"가 아니라, 각 소상공인의 고유한 문제를 맞춤 자동화.
> 가격: 무료/소액 → 유저수 확보 → 데이터/피지컬 AI 수익

→ 상세: `기획/신도시/슈파.md`

### 에센스 간의 관계

```
글로컬상권 — "콘텐츠의 총합이 상권이다" 증명 + 패러다임 전환 깨달음
  ↓ 이 깨달음이
신도시 — 공존공간 사업 재편의 동력. 현장에서 배운 것이 구조를 바꿈
  ↓ 신도시 안에서
슈파 — "파트너로서 직접 효과"를 기술로 구현. 현장 지식이 무기
  ↓ 이 전체의 철학이
모두의창업 — "보호 → 도전과 성장"이라는 시선 전환의 정책 프레임
```

**핵심**: 네 가지는 하나의 이야기다.

## 현재 구조

```
박승현 (기획자, 개인)
├── (주)공존공간 ─── 공간·문화·상권기획 운영       → 기획/공존공간.md
│                    커뮤니티 REITs·BTO (미래)
│
├── (주)신도시 ───── 인력 발굴·육성·투자 (스핀오프)  → 기획/신도시/
│   ├── 로컬벤처스튜디오 (AC)                       → 기획/신도시/벤처스튜디오.md
│   └── 슈파(SUPA) — 소상공인 맞춤 AI 자동화        → 기획/신도시/슈파.md
│
└── (사)행궁동행 ─── 지역 자율 운영 구조             → 기획/행궁동행.md
```

## 성장 경로

**공간**: 행궁동 → 남문시장 → 수원 전역 → 타 지역
**사업**: 상권기획 → 창업가 발굴·투자 → AI 자동화 → 부동산(REITs)
**슈파**: 수원 시범 → 수원 전역 → 전국
