## 패스트무버 비교 분석

> 중기부 보고서를 만드는 주요 기관·조직이 어떤 형식을 쓰는지 비교한 결과.

| 유형 | 대표 사례 | 형식 특징 | 강점 | 약점 |
|------|----------|----------|------|------|
| **중기부 공식 보고서** | 업무보고, 정책 브리핑 | HWP 기반, 블루 그라데이션, KRDS 색상, 표 중심 | 관료 친화적, 신뢰감 | 디자인 단조로움 |
| **전문 제작사 (DesignWith 등)** | 결과보고서 전문 용역 | A4 인쇄물, 인포그래픽 삽입, 고급 종이·코팅 | 시각적 완성도 최고 | 비용 높음, 수정 어려움 |
| **미리캔버스/Canva** | 공공기관 템플릿 활용 | 템플릿 기반, 아이콘+차트 조합, 카드뉴스형 | 빠른 제작, 한국어 최적화 | 커스터마이징 한계 |
| **Gamma AI** | IR 피치덱, 빠른 프로토타입 | AI 자동 생성 → PPTX, 슬라이드 중심 | 속도 최고 | 워드 문서 부적합 |
| **Antigravity (채택)** | 브라우저 에이전트 → Google Docs 자동 생성 | MD 원고 기반 → 에이전트가 직접 Google Docs 포매팅 | 구조화된 원고 그대로 반영, 수정 용이, 무료 | 수동 폴리싱 필요 |

**채택 이유: Antigravity**
- MD 원고의 구조(제목, 표, 불릿)를 Google Docs에 충실하게 반영
- 공문서 서식 가이드를 프롬프트에 포함하면 정부 보고서 톤 재현 가능
- 생성 후 Google Docs에서 직접 미세 조정 → .docx 내보내기 가능
- IR 덱(슬라이드) 제작 때 이미 검증된 워크플로우

## 4. '온라인 간판' 만들기 — 글로벌 디지털 접근성 구축

### 가. 문제 인식

국내 소상공인은 네이버 스마트플레이스에는 정보가 비교적 잘 등록되어 있으나, 외국인이 주로 이용하는 구글맵, 트립어드바이저, 샤오홍슈 등 글로벌 플랫폼에는 상호명만 등록되어 있고 상세 정보(메뉴, 영업시간, 사진, 리뷰 등)가 부실한 상황이었음. 이로 인해 외국인 관광객의 신뢰도가 떨어지고, 오프라인 상권의 매력이 온라인에서 발견되지 못하는 구조적 단절이 존재하였음.

[서식 노트]

## Antigravity 활용 팁 (워드 문서 생성 시)

1. **Planning Mode에서 문서 구조를 먼저 검토** — "이 문서 구조를 보여줘" 로 목차 확인 후 시작
2. **한 번에 전체를 만들지 말고 섹션별로** — "1~4장까지 먼저 만들어줘" 식으로 나눠서 생성
3. **표 서식은 수동 폴리싱 필요** — 홀짝행 배경, 헤더 색상은 Antigravity 생성 후 Google Docs에서 조정
4. **KPI 박스는 텍스트 박스+배경색으로** — Google Docs의 "그리기 삽입" 또는 1셀 표로 구현 가능
5. **폰트 대체** — Pretendard 미지원 시 Noto Sans KR 또는 맑은 고딕으로
6. **생성 후 .docx 내보내기** — Google Docs > 파일 > 다운로드 > Microsoft Word (.docx)
7. **페이지 나눔** — 각 H2 섹션 전에 "페이지 나눔을 넣어줘"라고 명시적으로 요청
