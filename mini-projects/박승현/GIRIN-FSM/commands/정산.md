# "정산" — 미식가의주방 정산 주문

> 정산 작업 시 필요한 맥락을 자동 로드하고 워크플로우를 안내.

---

## 실행 순서

| 순서 | 행동 | 누가 | 설명 |
|------|------|------|------|
| 1 | **정산 맥락 로드** | 메인 | 아래 파일들 읽기 |
| 2 | **현재 상태 파악** | 서브에이전트 | 구글시트 설계안 + 최근 세션 + progress에서 어디까지 했는지 |
| 3 | **정산 브리핑** | 메인 | 현재 상태 + 다음 할 일 출력 |
| 4 | **작업 시작** | 메인 | 승현님 확인 후 진행 |

### 로드할 파일

| 파일 | 내용 |
|------|------|
| `mini-projects/박승현/미식가의주방-정산/구글시트-설계안.md` | 시트 구조, 계약 기준, 정산 타이밍 규칙 |
| `mini-projects/박승현/미식가의주방-정산/apps_script.gs` | Apps Script 코드 (현재 상태) |

### 서브에이전트 프롬프트 (상태 파악)

```
수집 항목:
1. progress/박승현/ 에서 "미식가" 키워드 포함된 최신 기록 → 어디까지 했는지
2. mini-projects/박승현/미식가의주방-정산/sessions/ 최신 파일 → 마지막 세션 내용
3. git log -5 --oneline -- "mini-projects/박승현/미식가의주방-정산/" → 최근 변경

반환 형식:
---진행 상태--- (1~3줄)
---마지막 세션--- (1~3줄)
---최근 변경--- (커밋 목록)
```

---

## 정산 타이밍 퀵 레퍼런스

| 주차 | 차감 항목 |
|------|----------|
| 1~2주차 | 수수료만 (카드15% + 현금13% + VAT) |
| 3주차 | 수수료 + **지난달 공과금** |
| 4주차 | 수수료 + **임대료 300만+VAT + 관리비 + 캡스** |

## 출력 형식

```
> 🧚 정산 — 미식가의주방

--- 진행 상태 ---
어디까지 했는지 요약

--- 🔵 다음 할 것 ---
이번에 해야 할 작업

--- 📋 퀵 레퍼런스 ---
정산 타이밍 테이블 (위 표)

이어서 진행할까요?
```

---

## 규칙

- 정산 숫자는 절대 추측하지 않음. 시트/문서에 있는 값만 사용
- 계약 조건 변경은 승현님 확인 필수
- 구글시트 작업 시 → 설계안.md의 셀 색상 규칙 준수
- 정산 외 미식가 관련 작업(계약, 미수금 협상 등)은 이 주문 범위 밖
