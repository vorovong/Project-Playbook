# 클로드 코드, 처음이시죠?

## 이게 뭔가요?

클로드 코드는 **대화로 컴퓨터에게 일을 시키는 도구**입니다.

우리가 이미 쓰고 있는 ChatGPT나 Claude 앱은 "질문하면 답을 주는" 도구죠.
클로드 코드는 한 단계 더 나아갑니다. **답을 주는 게 아니라, 직접 일을 해줍니다.**

- 매주 똑같은 형식으로 작성하는 보고서? → 자동으로 만들어줍니다
- 이메일에서 매번 같은 정보를 복사-붙여넣기? → 알아서 정리해줍니다
- 엑셀 데이터를 정리하고 요약하는 반복 작업? → 한 번 말하면 끝입니다

코딩을 몰라도 됩니다. 한국어로 대화하면 됩니다.

---

## 불안하신 거, 다 압니다

**"코딩 몰라도 진짜 되나요?"**
→ 됩니다. 이 플레이북이 그걸 위해 만들어졌습니다. 클로드 코드가 한 단계씩 설명하면서 같이 해줍니다.

**"뭔가 잘못 건드리면 컴퓨터 망가지지 않나요?"**
→ 아닙니다. 파일을 만들거나 지우는 정도이고, 중요한 건 항상 먼저 물어봅니다. 실수해도 되돌릴 수 있습니다.

**"그 까만 화면(터미널)... 무섭거든요"**
→ 터미널은 그냥 글자로 대화하는 창입니다. 거기에 한국어로 말을 걸면 클로드가 답하는 거예요. 처음에 낯설 뿐, 어렵지 않습니다.

---

## 시작하려면 뭐가 필요한가요?

**Claude 구독** (Pro 또는 Max) — 이것만 있으면 됩니다.
이미 Claude를 유료로 쓰고 계시다면, **추가 비용 없이** 바로 시작할 수 있습니다.
API 키 같은 건 필요 없습니다.

## 설치 가이드

혼자 하기 어려우면 이 문서를 설치를 도와줄 사람에게 보여주세요.
아래 단계를 하나씩 천천히 따라가면 됩니다.

**먼저 본인 컴퓨터가 Mac인지 Windows인지 확인하세요.** 각 단계에서 자기 컴퓨터에 맞는 것만 따라하면 됩니다.

---

### 1단계: 터미널 열기

까만 창이 하나 뜹니다. 여기에 글자를 입력하면 컴퓨터가 그대로 실행하는 창이에요.
무서울 것 없습니다. 그냥 텍스트 대화창이라고 생각하세요.

**Mac의 경우:**
`Cmd + 스페이스바` → "터미널" 검색 → 열기

**Windows의 경우:**
`Win 키` → "PowerShell" 검색 → **"Windows PowerShell"** 열기
> "명령 프롬프트"나 "CMD"가 아닙니다. 반드시 **PowerShell**을 열어주세요.

---

### 2단계: 클로드 코드 설치

아래를 그대로 복사해서 터미널(또는 PowerShell)에 붙여넣고 `Enter`를 누르세요:

**Mac의 경우:**
```
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows의 경우:**
```
irm https://claude.ai/install.ps1 | iex
```

- 영어로 뭔가 주르륵 나옵니다. **정상입니다.** 설치가 진행되는 거예요.
- 글자가 멈추고 더 이상 아무 일도 안 일어나면 설치가 끝난 겁니다.
- 이제 **창을 닫아주세요.**
  - Mac: `Cmd + Q` 또는 창 닫기
  - Windows: 그냥 창 닫기

---

### 3단계: 터미널 다시 열기 + 작업 폴더 만들기

1단계와 같은 방법으로 터미널(또는 PowerShell)을 다시 열어주세요.

> 왜 닫았다 다시 여냐면: 방금 설치한 클로드 코드를 컴퓨터가 인식하려면
> 터미널을 새로 열어야 합니다. 한 번만 하면 됩니다.

터미널이 열리면, 아래를 복사해서 붙여넣고 `Enter`:

**Mac의 경우:**
```
mkdir -p ~/projects/Project_Playbook
```

**Windows의 경우:**
```
mkdir -Force "$HOME\projects\Project_Playbook"
```

- 아무 반응이 없으면 **정상입니다.** 폴더가 조용히 만들어진 거예요.

---

### 4단계: 클로드 코드 실행

아래를 복사해서 붙여넣고 `Enter`:

**Mac의 경우:**
```
cd ~/projects/Project_Playbook && claude
```

**Windows의 경우:**
```
cd "$HOME\projects\Project_Playbook"; claude
```

---

### 5단계: 처음 실행 — 로그인하기

처음 실행하면 몇 가지 질문이 나옵니다. 당황하지 마세요, 하나씩 안내합니다.

**5-1. 텍스트 스타일 선택**

터미널에 글자가 어떻게 보일지 고르는 화면이 나옵니다.
`↑` `↓` 화살표 키로 골라서 `Enter`. 아무거나 골라도 됩니다.

**5-2. 로그인 방법 선택**

로그인 방법을 묻는 화면이 나옵니다.
**"Claude account with subscription"** 을 선택하세요 (화살표로 이동 후 `Enter`).

> "Anthropic Console"이 아닙니다. 구독자는 위쪽 옵션을 선택하면 됩니다.

**5-3. 브라우저에서 로그인**

`Enter`를 누르면 **웹 브라우저가 자동으로 열립니다.**
평소 Claude 웹사이트(claude.ai)에 로그인하는 것과 똑같이 로그인하세요.

로그인이 완료되면 브라우저에 "성공" 메시지가 뜹니다.
이제 **터미널로 돌아가세요.**

**5-4. 보안 안내 + 폴더 신뢰 확인**

터미널에 영어로 보안 관련 안내가 나옵니다. `Enter`를 누르세요.
그다음 "이 폴더를 신뢰하시나요?"라는 질문이 나옵니다.
**"Yes, proceed"** 를 선택하고 `Enter`.

---

### 6단계: 완료!

`>` 표시가 나오면 **설치 끝, 대화 준비 완료**입니다.

클로드가 먼저 말을 겁니다.
"무엇을 자동화하고 싶으세요?" 같은 질문부터 시작해서,
**30분 안에 하나의 결과물을 같이 만들게 됩니다.**

어렵게 느껴지면 언제든 "잘 모르겠어요"라고 말해도 됩니다.
클로드가 다시 쉽게 설명해줍니다.
